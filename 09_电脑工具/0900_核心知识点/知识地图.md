# 0900_电脑工具知识地图

## 模块定位

| 项 | 内容 |
|---|---|
| 所属目录 | 09_电脑工具 |
| 解决的问题 | 工具如何进入用户本机开发、自动化、文档和知识生产工作流 |
| 典型输入 | 工具文章、CLI 教程、插件教程、工作流实践、自动化案例 |
| 典型产出 | 工具选型边界、配置路径、自动化动作、验证方式、排障准则 |

## 流程节点

| 节点 | 节点问题 | 当前目录 |
|---|---|---|
| T00 使用场景 | 工具解决什么任务，是否只是资讯或清单 | 0901、0902 |
| T01 安装配置 | 本机依赖、权限、路径、认证、配置文件在哪里 | 0901、0902、uv、OpenCLI、npm、Git、Obsidian |
| T02 操作接口 | CLI、GUI、插件、API、文件协议如何触发动作 | 0901、0902、uv、OpenCLI、npm、Git、Obsidian |
| T03 自动化链路 | 能否和脚本、Agent、浏览器、知识库、定时任务串起来 | 0901、uv、OpenCLI、npm、Git、Obsidian |
| T04 输出验证 | 工具输出如何检查、回放、恢复、迁移 | 0901、0902 |
| T05 维护边界 | 版本、同步、锁定、平台依赖和替代方案是什么 | 具体工具目录 |

# 一、用AI快速“榨干”吴士宏《越过山丘》

核心工具：腾讯IMA+微信读书

（也可以用其他工具，这是我自用的组合）

第一步：关联IMA和微信读书，快速找到《越过山丘》

1. 打开IMA（我用的手机端，电脑端操作一样）；
2. 在对话框输入微信读书网址：weread.qq.com，回车；
3. 自动用我登录IMA的微信账号登录；
4. 找到我的微信读书书架，搜索“吴士宏《越过山丘》”，一键打开，不用重新找书、重新登录。

重点：打开后，直接点击右上角“小灯泡”，把这本书添加到IMA个人知识库，这样后面就能随时和这本书“对话”，问难点、要总结，不用反复翻书。

第二步：用AI筛重点，梳理全书框架（解决“读不完”）

这一步就是帮大家避开“逐字逐句读全书”的坑，用AI快速摸清整本书的框架，找准重点章节，给大家节省时间。

1.在IMA里打开《越过山丘》，点击“问问IMA”

3.再用微信读书自带的AI导读，快速浏览每一章的小标题和核心摘要，进一步细化框架，锁定需要重点拆解章节。

这些章节要精准契合AI结构化答案的核心逻辑，也是书籍中最具实操价值和核心内涵的部分，重点拆解这些内容即可。

这一步做完，不仅解决了“读不完”的痛点，还找准了拆解方向，把时间聚焦在核心章节上，效率直接翻倍。

第三步：解构提炼书中三大篇章核心章节（解决“记不住”）

二、通用方法：套用AI逻辑，拆解任何一本书

上面是我用《越过山丘》做的演示，下面我为大家整理出通用方法，不管读什么书，直接套用就可以了。

1.前期准备

按前面的步骤，IMA+微信读书，把目标书籍添加到IMA知识库，确保AI能基于原文回应。

2.四步拆解流程（直接复制我给的指令就行）

第一步：定向拆解

万能指令：“请作为我的深度阅读伙伴，帮我梳理《XXX》的核心与框架：①用100字左右总结本书核心思想与主旨；②说明这本书主要解决了什么问题、最适合哪类读者；③结合书籍目录结构和三大支撑维度（认知重构、策略破局、系统融合），用金字塔原理梳理全书整体逻辑框架，标明核心论点与支撑论据的层级关系，语言简洁精准，贴合书籍原文。”

第二步：精加工拆解

万能指令：“请对《XXX》三大支撑维度对应的核心章节进行深度拆解：①分析各维度对应章节的论证结构、核心观点及原文论据（标注页码）；②阐释各维度核心概念的定义、内涵及可能的误解，结合书籍对应章节及页码佐证；③提取书中与三大维度及对应章节相关的1-2句金句（标注页码）和1个反常识观点，用清晰列表呈现，贴合书籍原文。”

第三步：批判迁移

万能指令（这里要记得改场景）：“请对《XXX》三大支撑维度的核心结论，①模拟杠精角色提出2-3个质疑及反驳思路（贴合书籍原文）；②说明理解三大维度及对应章节内容需要了解的作者个人经历或背景；③将三大维度核心逻辑迁移应用到[你的场景，如读书分享、职场]中，给出1个具体应用场景和操作步骤。”

第四步：内化输出

万能指令：“请将《XXX》三大支撑维度及对应核心章节的拆解内容，①整理成结构化读书笔记（包含核心观点、关键论据、个人启发、待行动项）；②列出3-5条可立即付诸实践的行动建议；③基于三大维度主题，推荐2-3本相关书籍及推荐理由。”

# 三、AI深度阅读流程的思路梳理

这里再给大家把思路讲清楚了，你明白了思路，就可以根据自己个性化需求来生成指令了：

1. 联动工具：IMA+微信读书，同步书架，把目标书籍添加到IMA知识库；
2. 定向拆解：用IMA生成书籍框架，锁定重点章节，解决“读不完”；
3. 精加工拆解：用IMA拆解核心，微信读书划线答疑，解决“记不住”；
4. 批判思考：用IMA模拟质疑，深化理解；
5. 迁移应用：用IMA生成场景方案，解决“用不上”；
6. 内化输出：用IMA生成笔记、清单和推荐，积累分享素材；
7. 复盘优化：每周用IMA复盘，形成闭环，彻底摆脱读书痛点。

## 跨域路由准则

- 工具如何安装、配置和工作流整合 -> 本目录。
- Agent 自动维护知识库、RAG 架构、向量检索 -> `02_Agent与AI工程/0203_RAG与知识库`。
- Mermaid/图表在前端组件中的渲染实现 -> `07_工程与架构/0702_前端工程`。
- PDF/Markdown/HTML 的解析归 `090207_文档解析与OCR`；生成与排版归 `090210_文档生成与排版`。

## 排重指纹

```text
工具名 + 使用场景（创建/检索/自动化/AI 接入） + 核心机制 + 解决问题 + 工作流整合/迁移成本边界 + 对用户的认知增量
```

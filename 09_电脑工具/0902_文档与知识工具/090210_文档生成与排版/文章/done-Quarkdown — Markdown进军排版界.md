# 如果你是Markdown的忠实用户，是否也曾为它在排版上的“朴素”而感到遗憾？ 今天给大家介绍一个令人眼前一亮的开源项目 —— **Quarkdown**，它或许正在开启 Markdown 在排版设计领域的一次小小革命。

##

## 什么是 Quarkdown？

**Quarkdown** 是由开发者 iamgio 打造的一个开源项目，简单来说，它是一个让你**用 Markdown 写出漂亮页面排版的工具**。

项目地址：
👉 https://github.com/iamgio/quarkdown

它的核心理念是：

> 用极简的 Markdown + 简洁的语法糖，实现富排版效果，**生成美观的 PDF 或网页文档**。

Quarkdown 不是另一个笔记应用，也不是简单的 Markdown 转 PDF 工具，它有点像「排版界的 LaTeX + Notion + Markdown」。

## 特点亮点

### ✅ 极简语法

Quarkdown 基于 Markdown，扩展了一些非常易懂的语法糖，例如：

```
@title: 我的作品集@theme: clean
```

```

```

加上一些特别的块语法，就可以轻松实现网格排版、图文布局等，零 CSS、零 JS。

###

### 🎨 多种主题+可自定义

项目内置了多个主题，如 `clean`、`soft`、`vintage` 等，甚至可以自定义主题样式 —— 对设计控非常友好。

###

### 📄 自动生成美观 PDF

无论是**作品集、简历、文档还是展示页面**，Quarkdown 都可以一键导出 PDF，不再需要手动排版。

###

### 🌐 跨平台、开源、免费

* 开源（MIT License），可放心使用、二次开发
* 支持本地使用或部署为网页版编辑器
* 无需联网，也没有账户绑定，纯粹

##

## 它适合谁？

* 想用 Markdown 写出“高级感”的写作者
* 需要制作简历、项目展示页的程序员 / 设计师
* 对 Notion 样式感兴趣，但又希望更自由的用户
* 喜欢写作、写教程、做电子书的内容创作者

## 未来趋势：Markdown 的第二春？

从 Typora、Obsidian 到 Quarkdown，Markdown 正在以不同形式重构我们的写作方式。

Quarkdown 展示了 Markdown 不只是写笔记的工具，它有能力进入**排版、设计、出版、内容展示**等更多场景。它的轻便、结构化和可扩展性，正是新时代内容创作需要的基础设施。

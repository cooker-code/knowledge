## 排重准则

问题指纹：

```text
工具名 + 操作接口 + 工作流节点 + 自动化方式 + 验证/排障边界 + 对用户的认知增量
```

| 判断项 | 排重规则 |
|---|---|
| 都是 CLI 工具 | 按解决的任务拆分，不按命令形式判重 |
| 都是 AI CLI | 区分浏览器控制、网站接口化、本机工作台、开发代理、消息平台接入 |
| 都是包管理/构建工具 | 区分依赖解析、发布、环境隔离、锁文件、仓库配置 |
| 只有命令清单 | 没有失败模式、验证动作或迁移边界时只保留原文锚点 |
| 涉及 Obsidian | 移到 `0902_文档与知识工具/090201_Obsidian` |

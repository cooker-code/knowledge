# 090100_开发工具与CLI知识地图

## 模块定位

| 项 | 内容 |
|---|---|
| 所属目录 | 09_电脑工具 / 0901_开发工具与CLI |
| 解决的问题 | 开发工具和 CLI 如何进入本机工作流，并形成可验证、可排障、可迁移的使用准则 |
| 典型输入 | CLI 教程、工具发布、自动化实践、包管理和构建工具文章 |
| 典型产出 | 安装配置路径、常用动作、验证方式、失败模式、抽离具体工具的依据 |

## 宏观流程

| 节点 | 节点问题 | 文章吸收规则 |
|---|---|---|
| C00 场景识别 | 工具解决的是开发、发布、浏览器自动化、消息平台、终端工作台还是包管理问题 | 工具清单只保留锚点，能明确场景才进入候选 |
| C01 安装配置 | 依赖、路径、权限、认证、配置文件如何落到本机 | 有可复现配置和失败信号才沉淀 |
| C02 命令接口 | 命令参数、输入输出、退出码、JSON/文本契约是什么 | 优先沉淀可被 Agent 或脚本稳定调用的接口 |
| C03 自动化组合 | 能否接入浏览器、网站、微信、飞书、Git、发布流程、定时任务 | 只吸收有链路、触发条件和验收结果的内容 |
| C04 输出验证 | 工具执行结果如何验证、回放、恢复、对比 | 没有验证动作的教程不写核心知识点 |
| C05 维护替代 | 版本变化、平台限制、依赖风险、替代方案是什么 | 对用户长期工具链有影响才正式沉淀 |

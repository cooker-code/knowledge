Python 以“简洁而优雅”著称——有时候，一行代码就能完成你以为需要十行才能实现的事。

今天整理了 **10 个经典的 Python 一行技巧**，每一个都体现了 Python 的“优雅之道”。

### 1️⃣ 反转字符串

一句话颠倒乾坤，`[::-1]` 简直是魔法。

### 2️⃣ 从列表中找出偶数

列表推导式让筛选与转化优雅到极致。

### 3️⃣ 判断两个单词是否是字母异位词（Anagram）

只要排序后相同，它们就是“变位兄弟”。

### 4️⃣ 统计元素出现频率

一行就能生成“词频表”，非常实用。

### 5️⃣ 交换两个变量

不用中间变量，Python 自带“魔术交换术”。

### 6️⃣ 扁平化嵌套列表

用嵌套推导式轻松“拍扁”多层结构。

### 7️⃣ 一行读取文件内容

写脚本时非常方便（正式代码请用 `with open()`）。

### 8️⃣ 获取列表唯一值

用 `set()` 去重，就像一把数字梳子。

### 9️⃣ 反转列表

和字符串一样，列表也能直接反转。

### 🔟 一行 if-else

表达清晰又自然，是 Pythonic 写法代表之一。

这些 Python 一行技巧，既是优雅的写法，也是理解 Python 思维方式的窗口。建议你亲手敲一遍，会更直观地感受到 Python 的魅力。

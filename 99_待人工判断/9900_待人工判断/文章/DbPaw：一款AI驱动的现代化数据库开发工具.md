DbPaw 是一款基于现代化技术栈构建、免费开源的数据库开发工具，支持可选的 AI 助手。

DbPaw 基于 Rust + TypeScript + Tauri 框架构建，遵循 Apache 2.0 开源协议，代码托管在 GitHub：

https://github.com/codeErrorSleep/dbpaw 

## 功能特性

* • **跨平台**：DbPaw 支持 macOS、Windows、Linux 操作系统，多个平台统一体验。
* • **数据库**：目前支持 MySQL、MariaDB、PostgreSQL、ClickHouse、TiDB、DuckDB、SQL Server 和 SQLite，可以通过 SSL或者 SSH 隧道访问远程数据库。

* • **SQL 编辑器**：提供语法高亮、自动补全、一键格式化等辅助功能，支持查询保存并且复用常用 SQL 脚本。

* • **数据网格**：在数据网格中浏览结果，支持过滤、排序、分页以及新增、修改、删除数据；同时可以显示表结构信息和 DDL 语句。

* • **数据导出**：可以将表或者查询结果中的数据导出为 CSV、JSON、SQL 文件。
* • **AI 助手**：提供可选的 AI 语言大模型配置（OpenAI、Gemini、Anthropic、DeepSeek、Qwen 等），支持 SQL 辅助编写、查询归纳、表结构解释和慢查询优化等功能。

* • **轻量级**：程序安装包大小 10 MB 左右，磁盘占用大约 80 MB，启动速度极快，内存占用极低。
* • **定制化**：支持不同语言、主题、字体、快捷键等自定义设置。

作为一个现代化、轻量级的数据库客户端，DbPaw 专注于开发者最常用的功能，为用户带来了一种简洁流畅的体验。

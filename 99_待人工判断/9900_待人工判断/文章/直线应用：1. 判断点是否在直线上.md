接下来探讨相关几何问题：

* 1. 判断点是否在直线上 （叉积）

点和直线有两种位置关系，不在直线上（左图）或者 在直线上（右图），可以利用叉积去计算。

由前文知道，叉积是 a × b = |a| |b| sin(θ) n。当PA 与 PB的夹角为0°或者180°，sin(θ) = 0，PA×PB=0；当角度为π的整数倍时，sin(θ) = 0；

```
// 假设 Point 是一个包含 x 和 y 坐标的结构体或类// dcmp 函数用于比较浮点数，返回 -1, 0, 或 1int dcmp(double x) {   // EPS 是一个很小的正数，表示允许的误差范围    if (fabs(x) < EPS) return 0;     return x > 0 ? 1 : -1;}  
// Cross 函数计算两个二维向量的叉积double Cross(Point P1, Point P2) {    return P1.x * P2.y - P1.y * P2.x;}  
// Dot 函数计算两个向量的点积double Dot(Point P1, Point P2) {    return P1.x * P2.x + P1.y * P2.y;}  
// 判断点 P 是否在直线 AB 上bool OnLine(Point P, Point A, Point B) {    return dcmp(Cross(A - P, B - P)) == 0;}
```

* 2. 判断点是否在线段上（叉积 & 点积）

判断点是否在线段上，就只需要，多判断一个点积（a · b = |a| |b| cosθ），如果PA和PB的夹角为180，则计算结果<0（因为cos π = -1）；

在计算上，满足叉积=0，则在直线上，再加上计算点积是否<0, 则可以判断是否在AB之间。如果点积=0，则点P在在点A或者点B上。

```
// 判断点 P 是否在线段 AB 上（包括端点）bool OnSegment(Point P, Point A, Point B) {    return dcmp(Cross(A - P, B - P)) == 0           && dcmp(Dot(P - B, P - A)) <= 0;}
```

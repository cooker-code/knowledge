你好呀，欢迎来到公众号：AI财友。

我是吴俊驰，一名AI产品经理，和你一起用AI生财的朋友。

最近Skills火得一塌糊涂，很多人问我：“俊驰，我是不是得去学学怎么写Python脚本，才能做个Skill？”

**我的回答是：大可不必。**在这个时代，**“整合”比“创造”更赚钱，也更省力。**

你想想，GitHub上有几千万个开源项目，从下载视频到分析股票，什么神仙工具没有？以前我们用不了，是因为那是给程序员用的，要配环境、装依赖，稍微错一个标点就跑不通。

**但今天，我发现了一个极其“鸡贼”的玩法：**用一个Skill去“吃掉”另一个GitHub项目，把它瞬间变成我可以调用的插件。**这不就是AI时代的“套壳”吗？但真香。**今天我们来撩一撩。

一、既然要偷懒，那就贯彻到底

我没有自己去写代码，而是先请出了一个“工头”**Skill-Creator**。这本身就是Anthropic官方出的一个Skill，它的作用只有一个：**帮我写Skill。**

我直接在Trae里输入：*“安装这个skill，项目地址是*:https://github.com/anthropics/skills/tree/main/skills/skill-creator

实战3分钟“白嫖”一个视频下载神器，为了验证这个逻辑，我找了个硬骨头：**yt-dlp**。这是GitHub上最强的视频下载工具，支持几千个网站。以前想用它，你得在命令行里敲代码，对小白极度不友好。

我对AI说了一句话，这句也是**核心Prompt**：

> *“帮我把开源工具**https://github.com/yt-dlp/yt-dlp**打包成一个Skill。只要我后续给出视频链接，就可以帮我下载视频。”*

**见证奇迹的时刻，**AI自动分析了那个复杂的开源项目，自动提取了核心逻辑，自动生成了Skill的配置文件。**它没有让我写一行代码，就帮我把这个“复杂的****命令行工具****”封装成了一个“听懂人话的插件”。**

我试着扔进去一个视频链接，10秒钟后，它告诉我：下载好了。

这个玩法的商业价值在哪里？

这个测试虽然简单，但背后的逻辑让我**细思极恐**。

以前我们开发一个功能，路径是：需求、找程序员、写代码、部署服务器、调试、上线。

现在这套玩法的路径是：需求、去GitHub找轮子、让AI封装成skill、直接调用。

**这意味着什么？**意味着GitHub不再是程序员的代码仓库，它变成了我们普通人的“半成品超市”。只要你会这招“移花接木”，**你瞬间就拥有了全世界最顶尖的工程师为你写好的几十万种能力**，而且**即插即用****，不用维护**。

二、写在最后

别总想着自己从零去造一个Skill。**最高级的猎手，通常是以“猎物”的形式出现的。**去GitHub上看看吧，那里全是黄金，现在你有铲子了。

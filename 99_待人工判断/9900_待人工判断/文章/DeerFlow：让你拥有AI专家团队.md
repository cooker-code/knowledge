## 🛠️ 如何使用？

|  |  |
| --- | --- |
| 1 | 安装准备  bash  # 克隆项目  git clone https://github.com/bytedance/DeerFlow.git  # 进入目录  cd DeerFlow  # 安装依赖  pip install -r requirements.txt |

|  |  |
| --- | --- |
| 2 | 配置你的 AI 团队  python  from deerflow import DeerFlow  deerflow = DeerFlow()  deerflow.configure(    models=["doubao", "claude", "gpt-4"],    roles=["coordinator", "planner",          "researcher", "analyst", "reporter"]  ) |

|  |  |
| --- | --- |
| 3 | 下达任务  python  task = "分析2024年新能源汽车市场趋势"  result = deerflow.execute(task)  report  = result["report"]   # 图文报告  podcast = result["podcast"]  # 播客音频  slides  = result["slides"]   # PPT 文件  charts  = result["charts"]   # 数据图表 |

|  |  |
| --- | --- |
| 4 | 坐等成果           ✅ 完整的市场分析报告（PDF/Word）          ✅ 播客音频文件（MP3）          ✅ PPT 演示文稿          ✅ 数据可视化图表 |

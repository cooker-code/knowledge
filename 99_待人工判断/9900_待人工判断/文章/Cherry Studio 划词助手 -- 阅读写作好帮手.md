# ⏮️ 前言

在Cherry Studio的早期版本中，官方推出了「划词助手」功能，最初仅支持Windows平台且功能尚不完善。自1.5.7版本起，该功能已全面兼容Mac系统，经实际测试表现优异。将「划词助手」与Cherry Studio搭配使用，能显著提升阅读文章的效率，让学习工作事半功倍。今天就来为大家详细介绍这款实用的阅读写作助手——「划词助手」。

## ⚙️ 基础设置

进入如下图所示的设置界面，打开右侧的「启用」按钮，就开启划词助手的功能。取词方式选择「划词」，其他保持不变。

在设置界面的下方「功能」选项，官方已经内置了几个功能，例如翻译、解释、引用、总结、优化等，你可以将常用的功能拖到功能区启用该功能。

官方内置的功能都非常有用，下面结合实际场景给大家逐一介绍。根据上面步骤设置完毕后，并且让 Cherry Studio常驻后台就可以使用划词助手。

## ❇️ 功能介绍

### 🌍 英文翻译

浏览网页时难免会遇到一些英文材料，此时有一个翻译助手可以让我们阅读英文材料时事半功倍。鼠标选中需要翻译的内容后，点击之前设置好的「翻译」按钮，Cherry Studio 就会调用翻译模型的 API 对文本进行翻译。

### 💡 概念解释

当我们接触不熟悉领域的资料的时候，常常会对某些**概念**或者**名词**感到迷惑，此时可以选中这个概念，一键让AI解释，再也不用复制粘贴问百度了。

### 🧩 内容引用

当我们看到某些内容需要结合 AI 再深入讨论的时候，可以选中内容后点击「引用」按钮，然后在 Cherry Studio 的对话框中系统就会以 Markdown 格式引用该内容。

### 📝 总结归纳

选中长篇大论，让 AI 给你一键总结。

### ✍️ 优化改写

当你创作完部分内容后，但是自己不是太满意，又不知道怎么修改，不如试试 AI 帮你改写。

### 🔧 自定义功能

如果以上功能你都不满意，Cherry Studio 还开放了自定义功能，打造专属于你的划词功能（视频中演示了将文本中的人名统一替换为 XXX 的功能）。

## 🎟️ 使用建议与避坑指南

### 📄 设置白名单

设置白名单后，可以让划词助手在特定应用中选中文本时才会触发，不设置的话所有应用都会触发，例如这里我设置了只有在使用 Chrome 浏览器时才触发划词助手。

### 🪟 设置窗口样式

我们可以通过是否开启紧凑模型和弹窗透明度调节窗口样式，开启紧凑模式后页面弹窗就只有 logo 没有文字按钮，更加简洁，缺点是功能多的时候可能不清楚 logo 对应的功能是什么。

### ⚠️ 可能遇到的问题

#### 模型设置

使用划词助手时，一定要先设置默认助手模型和翻译模型，否则可能调用失败。同时不要使用推理模型，否则模型中间推理过程的内容也会显示在划词助手的弹窗中。

#### 短文本不翻译

这个问题只能等后续版本迭代优化。

#### 超出模型上下文窗口

这个功能实际是把文本都发给大模型处理，如果文本过长超出模型上下文窗口，有可能报错，要注意模型的上下窗口有多大。

#### 推理速度慢

选择推理速度快的厂商或者小模型，使用划词助手时体验更加。

## 🔚 总结

划词助手显著提升了阅读写作效率——只需在后台运行Cherry Studio，即可轻松替代所有翻译软件或浏览器插件，体验极为流畅。其强大的自定义功能更增添了使用乐趣与灵活性。快来感受这份便捷与高效吧！

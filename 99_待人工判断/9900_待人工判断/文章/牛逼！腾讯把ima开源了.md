最近，腾讯开源了一个项目，很值得关注，想本地部署知识库的，可以尝试下这个开源。

项目名称叫WeKnora，是腾讯开源的一个文档检索框架。大白话就是一个开源知识库了。

大家都知道腾讯有一个ima知识库应用，这个项目也可以认为是ima的开源版。

ima使用起来有一点不方便的是，ima不开放API，所以每次更新资料，都得手动加入。完全没法通过工作流或者自动化的方式来自动更新。

有了这个开源项目，你就可以进行本地部署或者云端部署，并且通过API的方式来调用，维护这个知识库，使用场景相比ima大大的拓展了。

周末在家把这个开源在本地部署跑了下，感觉还不错。并且有完善的API调用指南，后续可以和智能体，工作流很好的结合起来。

**01**

**本地部署**

git clone https://github.com/Tencent/WeKnora.git

cd WeKnora

# 复制示例配置文件

cp .env.example .env

# 编辑 .env，填入对应配置信息

# 所有变量说明详见 .env.example 注释

将项目下载下来后，需要编辑下docker-compose.yml文件。将下面几个端口修改下

这两个端口默认分别为8080和80。一般情况下，电脑上这2个端口都是被占用的。提前修改，避免在启动的时候报错。

然后执行 docker compose up -d进行安装。

**02**

**模型配置**

浏览器中输入：http://localhost:9972  进入网页配置界面

第一个是LLM 模型配置，可以选择Ollama本地部署以及Remote API的方式。

第二个是Embedding 嵌入模型配置。Embedding嵌入模型是将文本，图像等数据转换成向量，这个向量能捕捉到对象的核心特征和语义信息，是知识库中的关键因素。

这里我选择了阿里的嵌入式模型。模型名称：text-embedding-v1。 API-key在云百炼官网申请即可。

在配置导航中，还有多模态配置，文档分隔，Rerank配置等参数可以进行设置

**03**

**开始使用**

配置完成后，进入主界面。主界面非常的清爽简单。分成知识库和对话2个功能

知识库支持主流的文档格式，也支持图片和视频格式的输入。

导入完成后，就可以基于知识库进行提问了。

好了，下面进行到重要部分。API调用。前面说了，因为缺少API，使得知识库的更新没法自动化。但WeKnora解决了这个问题，它提供了丰富的API可调用。

API分成了9个功能，每个功能都有详细的操作使用说明。

认证机制以及，通信方式都有非常全面的介绍。

比如知识库管理API，就包含了创建，获取，更新，删除，拷贝等动作

把这些API和智能体工作流结合起来，简直是打造本地知识库的一个绝佳组合。

API文档地址：

https://github.com/Tencent/WeKnora/blob/main/docs/API.md

另外，我还发现WeKnora也是微信对话平台的核心框架。

也就是说腾讯还搞了在线版的对话平台，这个平台可以无缝对接到公众号，小程序，企业微信，微信客服等接口。

如果不想本地部署的，可以尝试下在线版。

地址  https://weknora.weixin.qq.com/

**写在最后**

WeKnora其实蛮适合企业端使用。在本地部署大模型后，知识库可以不用上网，就在本地进行向量化，查询。

并且API的方式可以轻松的接入工作流，和日常办公，工作场景等无缝对接。

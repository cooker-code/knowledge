我在github上找了一些上千star的科研写作skills进行参考.

好的skill凝练了作者的智慧。

阅读这些skill的功能描述，我能从中学习作者的思考方式，并借此改进我的skill。

[ blader/humanizer ] .

消除文本中人工智能生成的痕迹，使其听起来更自然、更像人类的写作方式。

https://github.com/blader/humanizer/blob/main/SKILL.md

[ Master-cai/Research-Paper-Writing-Skills ] .

* 撰写或重写 Abstract / Introduction / Method / Experiments / Conclusion
* 改善段落衔接与章节逻辑
* 做 claim-evidence 对齐检查
* 提交前从 reviewer 视角进行自审

https://github.com/Master-cai/Research-Paper-Writing-Skills/blob/main/research-paper-writing/SKILL.md

[ Leey21/awesome-ai-research-writing] .

* 翻译、润色、分析等场景的实战 prompt
* 论文写作相关的agent-skills.

https://github.com/Leey21/awesome-ai-research-writing

[ K-Dense-AI/claude-scientific-skills] .

一个大的科研技巧库。

我从中摘取了科研写作部分的skill

https://github.com/K-Dense-AI/claude-scientific-skills/blob/main/scientific-skills/scientific-writing/SKILL.md

还有很多优质的开源的科研写作库，但是太多了,我就没有放进来.

感兴趣的可以自行检索。

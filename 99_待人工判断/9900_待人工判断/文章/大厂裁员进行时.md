大厂裁员进行时

大逃杀正式开始~

从2022年到今天，全球科技行业裁掉了多少人？

近100万人。

你没看错。不是10万，是100万。

这个数字什么概念？相当于整个深圳南山区的全部就业人口，一夜之间清零。

 

他们给了你什么理由？

AI变革、提效、组织优化、去肥增瘦。

听着多体面。

但你想过没有——

|  |
| --- |
| 上一次大规模裁员，用的理由是“后疫情修正”。再上一次，是“反垂断监管”。再再上一次，是“经济寒冬”。理由在变，动作一直没变。 |

 

裁员从来不是目的。裁员是手段。

今天这篇文章，我拆给你看：裁的是谁，用的什么借口，真正的目的是什么。

——————————

一、全球大厂裁员全景：中美科技公司的“大逃杀”

先说美国。

中国科技公司大抵落后美国5-10年。这几年差距在缩小，近期看就落后个1-2年。落后的不光是技术，公司文化、管理思想、政策制定，其实都是有样学样。

美国大厂已经裁了四年了。中国大厂，正在进行中。逃不掉的。

再说中国。

上一次互联网批量裁员是什么时候？2021-2022年。用的理由是什么？反垂断、双减、游戏限制、数据安全审查。

阿里巴巴四年账面减少了13万人。但别被这个数字吓到——其中大头是卖掉银泰、大润发等业务带来的"纸面出表"，那些人换了个老板继续上班。真正从核心业务裁掉的，保守估计也有数万人。数字没那么夸张，但逻辑是一样的：用"资产剥离"四个字把裁员包装成战略调整，财报一出，股价照涨。

 

注意一个细节：这些公司大规模裁员的同时，股价大多在涨。

Meta裁员后股价翻了三倍。Google裁员后市值创新高。Amazon裁员后利润创纪录。

裁的是你的饭碗，涨的是股东的财富。这笔账，你算明白了吗？

二、利益链条拆解：裁员从来不是因为“你不行”

我们拆开这条利益链看看。

 

第一步：谁赚钱？

股东。裁员的本质是降低人力成本，提升利润率，抬高股价。华尔街喜欢听到“裁员”这个词。每次裁员公告一发，股价应声而涨。

 

第二步：谁买单？

你。中年员工、高薪员工、管理层。你以为你是资产，其实在财报里你叫“人力成本”。

 

第三步：信息差在哪？

公司告诉你“AI提效了，你的岗位不需要了”。但真相是：你的岗位还在，只是换了个便宜的人来干。

 

|  |
| --- |
| “媒体报道：某产品经理被裁，月薪2.6万。同一个岗位立刻重新发布招聘，薪资不到1.5万。” —— 新浪财经报道 |

 

第四步：荒谬性暴露。

你以为裁员是因为AI替代了你？

但Business Insider 2026年4月的调查显示：大厂裁员后，总员工数并没有减少多少。它们一边裁员，一边招人。裁的是贵的，招的是便宜的。

 

|  |
| --- |
| 这不叫“AI替代”，这叫“汰换血液”。用新理由，做老动作。 |

 

你回忆一下：

• 2019年，裁员理由是“经济寒冬”

• 2021年，裁员理由是“反垂断监管”

• 2023年，裁员理由是“后疫情修正”

• 2025年，裁员理由是“AI提效”

 

理由在轮换，周期是2-3年一止。动作一直是同一个动作：把贵的人裁掉，换便宜的人进来。

 

被裁的人是什么画像？

 

| 维度 | 高风险画像 |
| --- | --- |
| 年龄 | 中国35+，美国40+ |
| 岗位 | HR、招聘、市场、中层管理 |
| 薪资 | 高于岗位中位数 |
| 司龄 | 5-15年（薪资累积高，但没有高管保护伞） |
| 性别 | 女性被裁概率高出男性65% |
| 特征 | 非收入产出岗、“成本中心”岗位 |

 

|  |
| --- |
| 你以为被裁是因为能力不足？不，是因为你太贵了。你的能力没变，但你的价格超过了公司愿意支付的上限。 |

讲讲底层逻辑：

你有没有想过一个问题——公司为什么要定期裁员？不是一次，是周期性地裁？

打个比方。

一辆车跑了5万公里，你会换刹车片。跑了10万公里，你会换正时皮带。不是因为这些配件坏了，是因为它们到了"性价比拐点"——继续用，风险在累积，成本在上升，而换一个新的，便宜、耐用、省心。

公司看员工，就是这个逻辑。

你刚入职的时候，月薪1万5，干劲十足，加班不计较，什么活都愿意接。三年后你月薪2万5，五年后3万5。你的能力确实涨了，但在公司眼里——你的"性价比曲线"已经过了顶点。同样的活，一个新人1万5就能干，你要3万5。

这不是你的问题。这是系统的逻辑。

再打个更直白的比方：你去租房，房东每年涨租金。涨到某一天你觉得不值了——不是房子变差了，是价格超过了你的承受阈值。你搬走了。公司裁你，本质上就是"房东觉得你太贵了，换个便宜租客"。

所以裁员的底层逻辑只有一句话：员工的成本是随时间自然膨胀的，但公司愿意为同一个岗位支付的价格是有天花板的。

工资会涨，年终奖会涨，社保基数会涨，期权会vest。而你的产出——说实话，干了三年和干了五年，差距没有薪资涨幅那么大。公司每隔几年就要做一次"重置"：把贵的换成便宜的，把老的换成新的。

至于用什么理由？无所谓。经济寒冬也行，AI革命也行，组织优化也行。理由是给外界听的，给媒体写的，给你的离职证明上写的。真正的动作只有一个——控制人力成本的增长曲线，让它不要跑得比利润快。

这就是为什么裁员每2-3年来一轮。不是因为公司真的活不下去了，是因为成本到了该"保养"的时候了。跟你换机油一个道理。

三、AI替代码农？这个逻辑根本讲不通

我知道很多码农朋友非常焦虑。每天别人告诉你，AI要抹杀你的工作了。

我给你摆几个事实。

 

事实一：

2025年METR研究所做了一个严格的随机对照实验：让有经验的开源开发者使用AI编码工具。

结果？使用AI工具的开发者反而慢了19%。

开发者自己觉得快了20%，实际上慢了19%。这是目前最严谨的实验数据。

 

事实二：

Klarna的CEO用AI替代了700个客服。结果？服务质量暴跌，客户满意度崩盘。2025年年中就认怡了，重新招人。CEO公开承认：“客户更喜欢跟人说话。”

 

事实三：

MIT斜坦研究：AI更可能是补充人类工作，而不是替代。它能影响11.7%的工作内容，但这是“岗位内的部分任务”，不是“整个岗位”。

 

事实四：

2026年4月，杭州中院判决：企业不能以“AI可以替代”为由裁员。法院认定这不属于法定解雇情形。

 

|  |
| --- |
| 写代码只占码农工作的20-30%。剩下的是需求分析、系统设计、复杂调试、架构决策、跨团队沟通。至少现在这个基建情况下，AI完全替代还为之尚早 |

 

而且你要知道，互联网的牛马，是这个社会上最擅长学习新知识的人群。

以前技术迭代周期是三五年。现在一年半载就大迭代。夸张的时候一个月一个新技术。但人家一直在学。

让最会学习的人因为“学习工具变强了”而失业？这个逻辑本身就是荒谬的。

 

|  |
| --- |
| 打个比方：你告诉一个全国最会开车的人——“车升级了，你被淘汰了”。可新车还不是得人来开？车再牛，没有司机能跑吗？ |

四、别觉得天塌了。市场比你想象的大得多

说完数据和逻辑，我想跟你说点心里话。

 

我身边有不少朋友被裁员了。有的在阿里干了6年，有的在字节干了4年，有的在百度干了8年。没有一个是因为能力不行被赶走的。

他们刚被裁的时候什么状态？

自我怀疑。觉得是不是自己哪里不够好。然后开始狂投简历、狂刷题、疯狂焦虑。

 

我想跟你说三件事：

 

第一，裁员不是你的错。

你没做错任何事。这是公司的财务策略，不是你的能力评价。你不是被淀汰的，你是被“优化成本”的。千万不要妄自菲薄。

 

第二，世界比你想象的大得多。

你知道这个世界上有多少公司连数字化都没做好吗？更别提AI化了。市里的服装店还在用手写账本。工厂的产线还在用Excel管理。小区的物业还在贴纸质通知。

这些地方，都是你的机会。大厂不是世界的全部。眼界放宽，天地很大。

 

第三，每个进大厂的人，都有他的不可替代性。

你能进大厂，本身就证明了你的能力。这份能力不会因为一纸裁员通知就消失。你的经验、你的视野、你解决问题的能力，这些都是真实的。

 

你只是从一个不再需要你价格的平台离开了。不是你不行，是他们买不起了。

——————————

五、保护好自己，比什么都重要

最后再说点真心话。

 

我见过太多人被裁员后把自己逐步PUA进死胡同的。觉得“我应该越来越强”，觉得“年纪大了不能走下坡路”，觉得“别人都在卖力我怎么能停”。

 

这个逻辑本身就是错的。

 

人生不是一条单行线。不是非得越来越强、越来越快、越来越贵。这个社会给你的叙事是“永远上升”，但真实的人生是波浪型的。有高峰有低谷，都正常。

 

工作很重要吗？重要。但它不是人生不可或缺的一部分。它只是你生活的一个维度而已。不要把全部身份认同都绑在工作上。

 

我自己写公众号，其实也是一种尝试。以前觉得自己只能在互联网公司干活，后来发现，大千世界很多东西值得去尝试。跟你想象的不一样。

 

我最想说的一句话：

 

|  |
| --- |
| 保护好你的身体。这是你最大的资产。 |

 

失业没有那么可怕。真正可怕的是你为了追一个不属于你的东西，把身体搅坏了。

科技在进步。万一过几年，医疗突破了，人类寿命大幅延长了，你那时候身体已经块了——等不到这个美好时代的降临，那多可惜。

 

少加班。多睡觉。多运动。多晒太阳。

别焦虑。你比你以为的值钱。只是有人不愿意付这个价而已。

 

世界很大。不只是互联网。不只是大厂。

大千世界，等你发现更多精彩。

喵喵喜欢早上起床后刷一下热点新闻，但是新闻热点都是由机构媒体选出来的，他们来决定热点是什么，所以要多个平台都要刷一下。

所以可能要切换十几个个新闻源。最近喵喵发现了一个很棒的开源项目，让你在一个平台去快速浏览各大平台的热点新闻！——**NewsNow[1]**

---

## 📰 介绍

NewsNow 是一个界面清爽聚合多个平台的新闻聚合工具，可以鼠标按住版块的右上角拖动修改显示位置。通过拖动新闻卡片的方式自定义页面布局，让页面更符合你的阅读习惯。

## 功能特性

* • **极简界面**：采用‘少即是多’的设计原则，去除冗余信息，提升信息密度。
* • **多端适配**：响应式设计支持从 PC 到移动端的无缝切换，暗黑模式保护夜间用眼健康。
* • **分层缓存机制**：默认 30 分钟缓存降低重复请求，登录用户可强制刷新获取最新数据。
* • **一键部署**：支持 Docker ，5 分钟内完成搭建。
* • **数据同步**：支持跨设备云端同步，确保办公与居家场景的无缝切换

## 🚀 三分钟极速部署指南

### 方式一：Docker 一键部署

```
# 复制粘贴三连  
git clone https://github.com/ourongxing/newsnow.git  
cd newsnow  
docker compose up -d  # 记得把-compose拼写正确！
```

### 方式二：本地开发（可以添加自定义信息源）

```
corepack enable     
pnpm i          # 依赖安装（建议搭配国内镜像食用）  
pnpm dev      
```

> 叮咚咚，今天来个比较班味的文章，通过各种组合插件完成一周的工作计划安排。

**划重点**

* Calendar组件：显示日历，点击某日生成日计划。点击周，生成周计划。

* Timeline组件：日待办事项，插入临时性工作

* 模板组件：快捷生成会议纪要

* CarBoard组件：汇总周待办清单，形成当日清单

# 模板功能

> 事先创建好各种模板，在使用时快捷插入，还可以设定基本的时间参数

* 模板是核心插件，不需要安装，直接启用就好。只需要在管理面板配置要保存的目录即可。
* 新建模板：例会模板。在模板目录中创建一个新文档，如下内容可以直接用。

  ### 

  ```
  ### 会议信息  
  会议主题：业务部门{{date:YYYY}}W{{date:ww}}周例会  
  会议计划：{{date:YYYY}}年{{date:MM}}月{{date:DD}}日 {{date:dddd}}  10:00 ~ 11:00  
  会议时间：{{time:HH:mm}} ~ {{time:HH:mm}}  
  会议地点：某地某会议室  
  会议方式：线下会议/线上会议  
  与会人员：  
  会议发起人：  
  会议记要人：  
  ### 会议议程：  
  1、部门总结与计划  
  2、专项事务讨论  
  3、会议结论与行动计划  
  ### 会议纪要：  
  ##### A部门主管：  
  ##### B部门主管：  
  ### 专项任务  
  * 专项一：  
  * 专项二：  
  * 专项三：  
  ### 待办事项  
  * [ ] 待办事项一  
  * [ ] 待办事项二  
  * [ ] 待办事项三
  ```

  ###

然后在开会的时候，新创建一个文档，快捷插入模板（例会模板），就可以直接记录部门事宜。

# 周工作计划+Calendar插件

> 利用创建好的周计划模板文件，在Calendar插件中配置选择好模板，和周计划文件存放目录。（我使用了98-日记管理，和下面的日计划放在一起）

* 启用插件后，在OB的右边（可以调整至右下方，右上方用于放Timeline插件界面）。这样在插件界面点，点击日期上的任意天，就可以进入某一天的工作日记（点击最左边的周，进入周计划日记）。创建的笔记在之前设定的目录中。

# 日记（日计划）+Timeline

> 日记插件也是OB的核心插件，启用日记插件就好。在管理界面输入日记目录（根目录下新建一个）和模板文件（输入选择之前创建的日计划模板文件）。

* 配置Timeline插件，右边显示当前工作时间，可以在任意时间点击进入（创建）日计划笔记，创建待办事项。

# 待办清单CardBoard插件

> 安装CardBoard插件，启用之后，在左侧快捷按钮打开面板。

* 点击左上角设置按钮，进行如下配置：

* 重点配置：Filters的Definitions中。输入之前的日计划目录（也是周计划目录），那么这个目录中所有文件中 - [ ] 标记事项都会收录进来，按下面Columns中日期要求，就会分类出来。

* 对应的待办事项，点击事项旁边的编辑按钮，会直接进入事项所在的日记进行编辑。

* 无论是在面板列表中，还是原始计划的笔记中，都可以打勾完成事项。看板中会自动更新到已完成列表中的。

# 总结：

# OB本身的模板功能非常强大，日记配合模板可以满足很多工作待办事项。

# 使用了Calendar和Timeline，可以更好的在面板上操作日计划，周计划，当日待办

# 使用CardBoard汇总上述创建的- [ ] 待办事项，按日期（周）形成待办/已办看板。

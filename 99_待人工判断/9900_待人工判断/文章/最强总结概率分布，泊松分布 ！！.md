## 1. 泊松分布

泊松分布是一般用来描述在单位时间、单位面积或其他固定区域内独立随机事件发生的次数。它的提出源于法国数学家西蒙·泊松。

主要适用于以下场景：

* 某一时间段内电话呼入次数
* 单位时间内客户到达服务中心的次数
* 某地区内稀有事件（如交通事故、故障等）的发生次数

**泊松分布适用的三个基本假设：**

1. **独立性假设：** 各个时间段或区域内事件的发生彼此独立；
2. **常数速率假设：** 在单位时间或单位区域内，事件发生的平均速率（记为 λ）是固定不变的；
3. **稀疏性假设：** 在足够短的时间间隔内，发生两个及以上事件的概率可以忽略不计。

在满足以上假设的前提下，泊松分布成为描述随机计数数据的一种非常有用的工具。

## 3. 案例分析 1

基本泊松分布的概率质量函数与数据模拟。

在第一个案例中，我们将以固定的  值为例（例如 ），生成泊松分布的概率质量函数（PMF）的图形，并模拟一组随机数据。

* 利用公式计算不同  值（例如  到 ）下的概率；
* 利用 Python 中的 `scipy.stats.poisson` 模块生成理论概率数据；
* 利用 Matplotlib 绘制色彩鲜艳且标签为英文的概率分布图；
* 分析图中各数据点的意义，讨论随着  值增大概率下降的趋势。

### 理论意义

该图形直观展示了当  为常数时，随机事件发生次数的概率分布情况。你可以观察到随着事件数  的增加，其概率先上升后迅速下降，这体现了泊松分布的“峰值”特性，且均值与方差均为 。

```
import numpy as np  
import matplotlib.pyplot as plt  
import seaborn as sns  
from scipy.stats import poisson  
  
# 设置绘图风格和颜色  
sns.set(style="whitegrid")  
plt.rcParams["figure.figsize"] = (10, 6)  
  
# 定义泊松分布参数 λ  
lambda_val = 4  
# 设定 k 的范围（0 到 15）  
k_values = np.arange(0, 16)  
# 计算每个 k 对应的概率 P(X=k)  
pmf_values = poisson.pmf(k_values, lambda_val)  
  
# 绘制泊松分布的概率质量函数（PMF）  
markerline, stemlines, baseline = plt.stem(k_values, pmf_values, linefmt='purple', markerfmt='o', basefmt=" ")  
  
# 设置颜色（matplotlib 3.4+ 版本支持）  
plt.setp(markerline, color="purple")  
plt.setp(stemlines, color="purple")  
  
plt.xlabel("Event Count (k)", fontsize=14)  
plt.ylabel("Probability P(X=k)", fontsize=14)  
plt.title("Poisson Distribution PMF (λ = 4)", fontsize=16)  
plt.xticks(k_values)  
plt.ylim(0, max(pmf_values)*1.2)  
plt.show()
```

* **横轴（Event Count）** 表示单位时间内可能发生的事件数 ；
* **纵轴（Probability P(X=k)）** 表示对应的概率值。
* 该图展示了当  时，最可能的事件数大约在 3 到 4 之间，而超过这一范围后概率急剧下降。

此外，我们还可以生成一组服从泊松分布的随机样本，并将其直方图与理论分布进行对比，验证理论分布的正确性：

```
# 生成 1000 个服从泊松分布的随机样本  
sample_size = 1000  
samples = np.random.poisson(lambda_val, sample_size)  
  
# 绘制随机样本的直方图，并叠加理论 PMF 曲线  
plt.figure(figsize=(10, 6))  
# 直方图：使用鲜艳的蓝色，透明度调节为 0.7  
plt.hist(samples, bins=np.arange(0, 17)-0.5, density=True, color='dodgerblue', alpha=0.7, label="Sample Histogram")  
# 叠加理论 PMF：用鲜艳的红色标记点和线条  
plt.plot(k_values, pmf_values, 'ro-', lw=2, label="Theoretical PMF")  
plt.xlabel("Event Count (k)", fontsize=14)  
plt.ylabel("Probability Density", fontsize=14)  
plt.title("Sample Histogram vs Theoretical Poisson PMF (λ = 4)", fontsize=16)  
plt.xticks(k_values)  
plt.legend()  
plt.show()
```

* **直方图（蓝色）** 展示了利用 NumPy 随机生成的 1000 个泊松分布样本的分布情况；
* **红色曲线** 表示理论上由公式计算得到的泊松概率质量函数。
* 二者基本吻合，验证了泊松分布在描述随机事件计数上的适用性。

### 模型训练步骤

1. **生成虚拟数据集：**

* 随机生成自变量 （如服从均匀分布或正态分布）；
* 根据泊松分布，利用设定的  和  计算每个样本对应的  值；
* 根据  值生成因变量 （服从泊松分布）。

2. **模型拟合：**

* 使用 Python 中的 `statsmodels` 模块，构建泊松回归模型；
* 通过极大似然估计（MLE）求解回归系数。

3. **结果分析：**

* 评估模型拟合优度；
* 通过预测值与实际观测值的对比，验证模型的准确性；
* 绘制预测结果与实际值的对比图，直观展示模型的拟合效果。

下面给出详细的 Python 实现代码：

```
import pandas as pd  
import statsmodels.api as sm  
  
# 设置随机数种子，保证结果可复现  
np.random.seed(42)  
  
# 生成虚拟数据集  
n_samples = 200  
# 自变量 X：在区间 [0, 10] 上均匀分布  
X = np.random.uniform(0, 10, n_samples)  
# 设定真实回归系数  
beta_0 = 0.5  
beta_1 = 0.8  
# 根据泊松回归模型，计算每个样本的 λ 值（确保 λ > 0）  
lambda_true = np.exp(beta_0 + beta_1 * X)  
# 根据 λ 值生成服从泊松分布的因变量 Y  
Y = np.random.poisson(lambda_true)  
  
# 将数据整理为 DataFrame 以便后续分析  
data = pd.DataFrame({"X": X, "Y": Y})  
  
# 绘制 X 与 Y 的散点图，观察 X 与计数数据 Y 的关系  
plt.figure(figsize=(10, 6))  
plt.scatter(data["X"], data["Y"], color="magenta", alpha=0.6, edgecolors="black", label="Observed Data")  
plt.xlabel("Predictor X", fontsize=14)  
plt.ylabel("Count Y", fontsize=14)  
plt.title("Scatter Plot of Predictor X vs Count Y", fontsize=16)  
plt.legend()  
plt.show()  
  
# 构建泊松回归模型  
# 在 statsmodels 中需要手动添加常数项  
X_with_const = sm.add_constant(data["X"])  
poisson_model = sm.GLM(data["Y"], X_with_const, family=sm.families.Poisson())  
poisson_results = poisson_model.fit()  
  
# 输出模型拟合结果  
print(poisson_results.summary())  
  
# 根据模型进行预测  
data["Y_pred"] = poisson_results.predict(X_with_const)  
  
# 绘制实际 Y 与预测 Y 的对比图  
plt.figure(figsize=(10, 6))  
plt.scatter(data["X"], data["Y"], color="darkorange", alpha=0.6, edgecolors="black", label="Observed Y")  
# 绘制预测值：为了更平滑地显示趋势，我们按照 X 排序后绘制拟合曲线  
sorted_data = data.sort_values("X")  
plt.plot(sorted_data["X"], sorted_data["Y_pred"], color="green", lw=3, label="Predicted Y")  
plt.xlabel("Predictor X", fontsize=14)  
plt.ylabel("Count Y", fontsize=14)  
plt.title("Observed vs Predicted Count Y (Poisson Regression)", fontsize=16)  
plt.legend()  
plt.show()
```

1. **数据生成与散点图：**

* 我们随机生成了 200 个样本，其中自变量  均匀分布在 [0,10] 内；
* 根据设定的回归模型  生成对应的泊松分布因变量 ；
* 散点图展示了  与  之间的关系，通常可以看到  随  增大而呈现上升趋势，但因泊松分布的离散性和方差与均值相等的特性，数据具有一定的离散波动性。

2. **泊松回归模型拟合结果：**

* 利用 `statsmodels` 中的 GLM（广义线性模型）构建泊松回归模型，通过极大似然估计方法估计模型参数；
* 输出的回归结果包括系数估计值、标准误、z 值和 p 值等，若 p 值较小，则说明对应自变量对计数数据有显著影响。

* 图中橙色散点表示实际观测到的  值；
* 绿色曲线为模型根据  值预测的  均值；
* 该对比图可以直观地展示模型在捕捉  与  之间关系方面的效果。如果绿色曲线能够较好地穿过散点数据云团，则表明模型拟合效果较好。

## 5. 数据分析

* **泊松分布的 PMF 图：** 第一个图形（使用 stem 图）展示了不同事件数  下的理论概率分布。图中可见，当  远离均值（此处均值为 ）时，其发生的概率急剧下降，这符合泊松分布“峰值集中、尾部迅速衰减”的特点。直观理解为，在大多数情况下，随机事件的发生次数会聚集在均值附近，而出现极端情况的概率非常低。
* **样本直方图与理论分布对比图：** 通过将随机生成的样本数据直方图与理论 PMF 曲线进行叠加，我们验证了模拟数据与理论分布之间的吻合程度，进一步证明了泊松分布在模拟现实随机事件计数时的有效性。
* **泊松回归模型图：** 利用泊松回归模型，我们能够探讨自变量  对计数型因变量  的影响。散点图展示了原始数据的离散分布，而拟合曲线则显示了在模型假定下，随着  的变化， 的平均值如何变化。通过对比，可以检验模型是否合理捕捉到了数据的趋势。

### 需要注意的问题

1. **模型假设检验：**泊松回归要求因变量服从泊松分布，其均值与方差相等。但在实际应用中，经常会遇到过度离散（Overdispersion）问题，即数据的方差远大于均值。此时，可能需要采用负二项回归（Negative Binomial Regression）等方法来更好地拟合数据。
2. **自变量与因变量关系的非线性：**泊松回归中的对数连接函数虽然保证了预测值为正，但若数据中存在非线性关系，可能需要引入非线性项或进行变量变换。
3. **数据量与离散性：**由于泊松分布的离散特性，样本量较小时数据可能较为稀疏，模型估计可能不够稳定。建议在样本量充足的情况下再使用泊松回归进行建模。
4. **实际应用场景：**在实际问题中，诸如交通事故、疾病发生、网络请求等事件通常可以近似服从泊松分布，但在模型建立前应仔细验证数据是否符合基本假设。

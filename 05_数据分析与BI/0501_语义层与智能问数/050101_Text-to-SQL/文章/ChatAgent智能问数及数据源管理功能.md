1. 数据源管理

用户可以对数据源进行管理，例如添加Mysql、Doris等类型的数据源，这样用户后续的数据查询、分析报告中来使用。

在数据源列表中，用户可以看到数据源状态，还可以对数据源进行编辑、测试连通性、预览等操作。例如用户在预览时，可以看到当前数据源中包含的数据表清单

如下图中所示，用户可以查看订单表中的字段，用户启用所关心的字段，还可以预览表中的数据，帮助用户快速探查表中的数据样例。

2. 数据查询

在数据查询模块用户可以选择某一个数据源，开始智能问数功能，例如输入7月份每种产品的订单总金额，可以看到执行的sql语句，结果数据以表格形式来展示

结果数据支持多种不同的展现形式，折线图、柱状图、饼图等，用户在右侧选择不同的图标类型即可。

3. 分析报告

用户通过描述需求，系统会自动生成对应的分析报告，例如用户选择订单数据源，在需求分析文本框中输入：分析2023年上半年订单收入数据

，即可一键生成业务分析报告，用户还可以将生成的报告以文档的格式导出。

报告内容示例如下

```
一、概述本报告基于2023年上半年的销售数据，对各产品的收入情况进行了全面分析，并对下半年的收入趋势进行了预测。二、产品收入分析2023年上半年各产品的收入情况如下：智能手机A：收入3,250,000元，占总收入的42.3%笔记本电脑B：收入2,890,000元，占总收入的37.6%平板电脑C：收入1,250,000元，占总收入的16.3%智能手表D：收入280,000元，占总收入的3.6%其中，智能手机A的收入贡献最大，占比超过四成。三、产品贡献度排名智能手机A - 42.3%笔记本电脑B - 37.6%平板电脑C - 16.3%智能手表D - 3.6%四、下半年预测基于历史数据和市场趋势分析，预计2023年下半年各产品的收入情况如下：智能手机A：预计收入3,800,000元（增长17%）笔记本电脑B：预计收入3,100,000元（增长7%）平板电脑C：预计收入1,450,000元（增长16%）智能手表D：预计收入400,000元（增长43%）总体预计下半年总收入将达到8,750,000元，比上半年增长约13%。五、建议继续加大智能手机A的市场推广力度，巩固市场份额关注智能手表D的快速增长趋势，适当增加产品线投入考虑在下半年推出笔记本电脑B的升级版本，以提升竞争力
```

## 简述SDK

SDK是“软件开发工具包”（Software Development Kit）的缩写。是一组适合于开发人员的平台特定构建工具集。您需要调试器、编译器和库等组件来创建在特定平台、操作系统或编程语言上运行的代码。

*有码友可能分不清SDK和API（应用程序编程接口）有啥区别：*

* *SDK包含API：SDK通常会包含所需的API，以便开发者能够使用特定的功能或服务。*
* *API独立于SDK：API可以存在独立于SDK之外，作为单独的服务或库来提供特定功能的访问方式。*
* *SDK更全面：SDK提供了比单独的API更全面的工具和资源，例如编译器、调试器、示例代码等，帮助开发者进行全面的软件开发。*

*SDK是一个更大范围的工具集，而API是一种通信协议和接口规范，用于不同软件组件之间的交互和集成。*

SDK按照应用场景分为以下类别：

* 广告和变现
* 分析
* 数据管理
* 位置
* 营销和互动
* 付款
* 社交
* 用户身份验证
* 用户支持

## 广告SDK

广告SDK（Advertising SDK）是用于在移动应用或者PC程序中集成广告服务的软件开发工具包。它们通常由广告平台或第三方广告提供商提供，广告SDK提供了开发者需要集成的工具、库和API，以便于应用程序向用户展示各种类型的广告内容。

广告SDK的主要目的是为了帮助移动应用开发者实现通过广告获得收入的目标，同时确保良好的用户体验和广告效果的最大化。通过提供技术支持和工具，广告SDK在移动应用生态系统中发挥着重要的作用，连接了开发者、广告平台和广告主之间的利益关系。

## 随着技术的发展，媒体转型加快以及广告主需求提升，相关广告售卖技术及服务逐渐趋于成熟，小媒体开发者需要把自己的流量变现，大媒体需要把中长尾流量售卖出去，程序化广告成了必不可少的一环，SDK也是程序化广告的流量池SSP的重要组成部分，下面是程序化广告产品链。

广告SDK一般支持以下几种广告类型

* **开屏广告**

开屏广告以App启动作为曝光时机，提供5s左右的可感知广告展示。用户可以点击广告跳转到目标页面，或点击右上角的“跳过”按钮，跳转到app内容首页。

* **激励广告**

指将短视频融入到app场景当中，成为app任务之一，用户观看短视频广告后可以得到一些应用内奖励。  常见于游戏内的广告样式，多出现在游戏的复活、登录等位置，或者网服类app的一些增值服务场景。

* **插屏广告**

插屏半屏是移动广告的一种常见形式，在应用流程中弹出，当应用展示插页式广告时，用户可以选择点按广告，访问其目标网址，也可以将其关闭，返回应用。插屏全屏广告是移动广告的一种常见形式，在应用自然流畅结束时全屏弹出，当应用展示插页式广告时，用户可以选择点按广告，访问其目标网址，也可以将其关闭，返回应用。

* **横幅Banner广告**

Banner广告位于app顶部、中部、底部任意一处，横向贯穿整个app页面；当用户与app互动时，Banner广告会停留在屏幕上，并可在一段时间后自动刷新。  常见的在文章页末尾，详情页面底部，信息流顶部等。

* **信息流广告**

适用于App 推荐列表信息流里，是一种广告形式，设计和呈现方式与其所在的平台或应用的内容风格和用户体验高度一致，使得广告与正常内容无缝融合，用户往往在不自觉中接触到这些广告。

* **详情页插入广告**

在App 详情页内随内容呈现插入的广告。

* **沉浸式视频流广告**

指的是利用沉浸式技术（如增强现实（AR）或虚拟现实（VR））创建的广告形式。这种广告形式旨在通过提供更加沉浸和交互式的体验，吸引用户的注意力并增强品牌与消费者之间的互动。

* **视频贴片**

在视频片头、片尾或片中随片播放等位置播放的广告，有5s、15s、30s等多种广告形式。

* **开发者自渲染广告**

开发者自行定义广告布局样式和展示场景。用于在APP信息流，任何希望展示广告的地方，APP中视频内容的框内插入。

## 广告SDK的类型

**Android SDK**

Android Software Development Kit是针对开发基于Android操作系统的移动应用程序的工具集合。它包含了开发Android应用程序所需的各种工具、库文件、文档和示例代码

下面是Google Play上的广告SDK的列表，这些广告SDK可用于集成到Android应用程序中。

**IOS SDK**

IOS Software Development Kit是用于开发iOS应用程序的工具集合，由苹果公司提供。它包含了开发者所需的工具、框架、API、文档和示例代码，帮助开发者利用Objective-C或Swift编程语言开发运行在iPhone、iPad和iPod Touch设备上的应用程序。

以下是一些常见的IOS广告SDK：

* AdMob SDK: Google的移动广告平台，为IOS开发者提供了广告显示和收入优化的解决方案，支持横幅广告、插页广告、原生广告和激励广告等多种广告格式。
* Facebook Audience Network SDK: Facebook提供的广告SDK，允许开发者在iOS应用中显示Facebook广告，支持横幅广告、插页广告、原生广告和激励视频广告。
* Unity Ads SDK: Unity为使用Unity游戏引擎的开发者提供的广告SDK，支持iOS平台上的视频广告和插页广告，帮助开发者通过游戏中的广告实现收入。
* MoPub SDK: MoPub是一种广告调解平台，支持iOS应用程序集成多个广告网络的SDK，帮助开发者通过管理多个广告源来最大化广告收益。
* AppLovin SDK: AppLovin提供的SDK支持在IOS应用中显示插页广告、激励视频广告和原生广告，同时提供分析和优化工具来增加广告收益。

**Unity SDK**

Unity SDK（Unity Software Development Kit）是为开发者提供的工具集合，用于创建基于Unity游戏引擎的应用程序和游戏。Unity是一个跨平台的游戏引擎，支持多种平台如PC、移动设备、主机游戏机等。Unity SDK包含了开发游戏所需的各种组件、工具和资源，使开发者能够在统一的开发环境中进行游戏开发和测试。

**JS SDK**

通常指的是JavaScript Software Development Kit，即JavaScript软件开发工具包。这是为了帮助开发者使用JavaScript语言开发应用程序时所提供的工具集合。JavaScript是一种广泛应用于Web开发的编程语言，具有跨平台的特性，因此JS SDK主要用于在Web环境下开发各种应用。

**小程序&小游戏 SDK**

常见小程序平台、快应用、小游戏平台的提供了一套标准的SDK以实现数据上报功能。常见厂商有微信，华为，字节，QQ，支付宝等厂商提供的小程序或快应用，小游戏平台。

## 核心问题

广告归因要把曝光、点击、设备标识、渠道回传、转化窗口、优先级和作弊风险连起来，判断一次转化应该归给哪个触点。

## 判断准则

| 模块 | 准则 |
|---|---|
| 标识匹配 | 先确认设备、账号、Cookie、IDFA/GAID 等标识能否稳定关联 |
| 归因窗口 | 必须明确点击、曝光和转化的时间窗口 |
| 优先级 | MMP、渠道、自归因平台之间需要清楚优先级和覆盖规则 |
| 回传链路 | 转化事件、金额、去重、延迟和失败重试必须可追踪 |
| 作弊识别 | 异常点击、设备农场、归因劫持和素材同质化要单独处理 |
| 素材与曝光异常 | 素材标签、曝光回传和异常诊断要能接入归因链路，不能只看平台侧报表 |

## 认知校准点

| 偏差 | 处理 |
|---|---|
| 广告归因只是模型问题 | 更常见瓶颈是标识、回传、窗口、优先级和作弊治理 |
| 平台报表就是事实 | 需要核对回传、去重和归因窗口 |
| 素材或渠道优化等同因果证明 | 仍需实验或准实验验证 |

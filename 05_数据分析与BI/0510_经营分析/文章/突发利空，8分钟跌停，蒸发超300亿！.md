今天，有一个坏消息和好消息。

坏消息是A股跌了。

好消息是，和全球其他市场相比，A股跌的是最少的。

（图片来源：东方财富APP，截至2022/6/13 18:45，不作投资推荐）

**上周五晚，美国公布了5月份的CPI数据，同比增长8.6%，创1981年12月以来新高，导致美股大跌**，道琼斯指数跌2.73%，纳斯达克指数跌3.52%，标普500指数下跌2.91%。

**恐慌情绪影响了全球股市**，今天A股开盘惯性下跌，三大指数一度跌超1%，不过午后，出现了些转机，**最终跌幅收窄，两市成交额再度突破万亿，个股涨多跌少。**

而隔壁港股就没这么幸运了，**恒生指数下跌3.39%%，恒生科技指数下跌4.72%。**

|  |  |
| --- | --- |
|  |  |

（图片来源：东方财富APP，截至2022/6/13，不作投资推荐）

仿佛刚看到了点回本的希望，就又被打回了原形。

A股今天的下跌，主要还是两方面原因。一**个是外围股市下跌影响，一个是北向资金净流出超135亿元**，内外资流出的压力较大。

而受到利空消息影响，**千亿龙头药明康德开盘8分钟便触及跌停，市值蒸发超300亿元**，随后股价维持在跌停附近，对CRO板块和指数都造成了一定影响。

|  |  |
| --- | --- |
|  |  |

（图片来源：东方财富APP，截至2022/6/13，不作投资推荐）

此前，在大幅下跌之后，药明康德发布了二季度公告，股价一度反弹重回100元，不过，好景不长。

6月10月晚，公司公告称，因自身资金需求，公司实际控制人控制的股东及与实际控制人签署一致行动协议的股东计划自公告之日起15个交易日后的不超过90日期间，**减持合计不超过公司总股本3%**。

如果最终实现顶格减持，以6月10日收盘价102.83元/股计算，本次减持将套现91.20亿元。

**这不仅使得药明康德A、H股大跌，也使得CRO板块跌超1%。**

药明康德也是不少明星基金经理的心头好，包括葛兰、赵蓓等。

（数据来源：东方财富Choice数据、基金一季报，截至2022/3/31，不作投资推荐）

**另外，白酒、大金融、地产、煤炭等板块也出现了调整，对沪指有一定承压。**

**而逆市上涨的，多为新能源相关板块。其中，能源金属****上涨3.19%****，锂电池****上涨2.30%****，汽车整车****上涨2.26%****。**

上周五，中汽协发布数据显示，5月份新能源汽车销量44.7万辆，同比增长105.2%。销售大幅回升。

**而市值升级至万亿的比亚迪今天也不负众望，盘中一度涨近3%续创新高**，不过，随后便出现跳水。

|  |  |
| --- | --- |
|  |  |

（图片来源：东方财富APP，截至2022/6/13，不作投资推荐）

**有消息称，比亚迪的融资折算率由60%调低至了0%**，对此，券商相关人士表示这是因为根据《融资融券交易实施细则》，需要将静态市盈率在300倍以上或者为负数的股票折算率下调为0%。

不过，机构对比亚迪的态度依然乐观，**摩根大通还指出，比亚迪正成为像科技股多于汽车股**，虽然集团现阶段的外部电池业务盈利贡献仍小，但预估至2022年的贡献可提升到10%至15%。

午后，比亚迪跌幅有所收窄，最终收于平盘。

**而宁德时代跌幅超过2%，“宁迪”之战要分出胜负了？**

要说今天最靓的崽，还得是新东方了，**港股新东方在线半天涨幅就超过了100%，不过午后出现回落，最终涨幅接近40%**，波动较大。

（图片来源：东方财富APP，截至2022/6/13，不作投资推荐）

不知道大家最近有没有刷到过新东方的直播，因为边教英语边带货的形式火了。

**近日，新东方旗下的直播带货平台“东方甄选”因为主播双语直播带货人气大涨**，3天涨粉超过150万，销售额增加1777万元。

**这也让A股教育板块逆市上涨3.89%**，仿佛给沉寂许久的教育股点了一把火。

|  |  |
| --- | --- |
|  |  |

（图片来源：东方财富APP，截至2022/6/13，不作投资推荐）

不过，这把火能持续多久，还不得而知。

近期，A股重回震荡模式，市场面临的考验仍在继续，**本周，美联储将举行为期两天的议息会议，鲍威尔将在北京时间周四凌晨2点公布利率决议**，而因为通胀高涨，本次加息幅度会否高于市场普遍预计的50基点，成为关注的焦点。

昨天，是5178点7周年的日子，**今天，是科创板开板3周年的日子。**

2019年6月13日，万众瞩目的科创板正式开板，7月22日，科创板首批公司上市。

4月27日以来，科创板强势的反弹也让不少投资者重新开始关注科创主题基金。**最早成立的科创主题基金三年的封闭期也在近期届满，转为开放，并且对基金名称进行了修改。**

科创主题基金也交出了一份不错的成绩单，**其中，涨幅最高的基金已经实现翻倍。**

（数据来源：东方财富APP、银河证券，截至2022/6/10，不作投资推荐）

不过近期，科创板也迎来了回调，**7月将迎来科创板解禁潮**，解禁市值为2711.4亿元，是今年解禁市值最高的一个月，短期而言对市场会有一定压力。

近期，基金圈还有个大事刷屏，**6月10日晚，中国证券投资基金业协会发布《基金管理公司绩效考核与薪酬管理指引》**，提出绩效考核与薪酬管理应遵循的四项原则，同时对薪酬结构、薪酬支付、绩效考核、薪酬内控管理等方面提出了具体要求。

（图片来源：东方财富APP，不作投资推荐）

其中，要求基金经理至少用30%绩效薪酬买本公司基金，并且要求绩效薪酬递延支付期限不少于3年。

**这能避免基金经理过分追逐短期利益，也是为基民的长期利益考虑**，让基金经理和基民同甘共苦的举措。

短期的成功，可能靠的是运气，但长期的成功，靠的一定是好的决策，我们投资基金同样如此。

**就像播种之后不可能第二天就能有所收获，很多成功的投资是需要时间的力量来浇灌的。**  
看淡短期的涨跌，时刻保持理性、耐心，相信“以时间换空间”，**经历时间的洗礼后，终会迎来空间上的馈赠。**

**最后一次巴菲特午餐开拍。**

受疫情影响被迫中断两年后，2022年的巴菲特午餐于北京时间今天7时30分在eBay开拍了，将持续到北京时间6月18日10:30结束，起拍价为2.5万美元。据格莱德基金会介绍，2022年将是最后一届巴菲特慈善午餐拍卖活动。在往届已知的12位中标人中，来自中国的有4位，分别为段永平（2006年）、“私募教父”赵丹阳（2008年）、天神娱乐董事长朱晔（2015年）、波场TRON创始人孙宇晨（2019年）。而从过去的中标金额看，自2008年以来，每次中标的金额都超过100万美元。虽然这不是月薪5000该考虑的事，但股神的投资精神还是值得我们学习的。

# 0510_经营分析知识地图

## 模块定位

| 项 | 内容 |
|---|---|
| 所属目录 | `05_数据分析与BI/0510_经营分析` |
| 解决的问题 | 业务数据如何连接目标、路径、用户、财务、动作和复盘 |
| 典型输入 | 经营目标、用户行为、收入成本、渠道、留存、LTV、ROI |
| 典型产出 | 经营分析框架、用户分层、增长杠杆、财务经营结论、行动闭环 |

## 降权与纠偏

| 类型 | 处理 |
|---|---|
| 单纯金融市场新闻、投资机会评论 | 不进入核心知识点，只作为低价值错放样例 |
| GEO/SEO 方法清单 | 可作为营销经营了解，但缺数据验证时不沉淀为核心 |
| AI Excel/自动会计/多表分析工具 | 已迁入 `050102_智能数据分析助手` |
| 连续登录 SQL 题 | 已迁入 `030204_SQL书写` |

## 待补缺口

| 缺口 | 后续判断 |
|---|---|
| 经营分析会制度与复盘节奏 | 是否说明责任人、指标、动作和复盘周期 |
| LTV 预测和预算联动 | 是否给出回本周期、边际 ROI 和不确定性 |
| 财务分析到业务动作 | 是否穿透收入、成本、效率和组织动作 |

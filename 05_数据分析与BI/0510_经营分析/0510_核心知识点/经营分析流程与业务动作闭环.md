## 核心问题

经营分析要把目标、路径、组合、指标、动作和复盘连成闭环。只做同环比、画图或套分析模型，不能直接产生经营价值。

## 判断准则

| 环节 | 准则 |
|---|---|
| 定目标 | 明确增长、利润、效率、留存或风险目标 |
| 看组合 | 横向比较产品、渠道、地区、人群、品类的结构 |
| 看路径 | 纵向拆解从触达到转化、复购、收入、成本的链路 |
| 找杠杆 | 识别能被团队动作影响的关键节点 |
| 做动作 | 给出负责人、动作、成本、预期指标和验证方式 |
| 复盘 | 回看动作效果，更新指标和分析框架 |
| 数据理解 | 先确认数据来源、口径、粒度、缺失和业务含义，再进入模型或 AI 分析 |

## 认知校准点

| 偏差 | 处理 |
|---|---|
| 数据分析就是描述现状 | 必须落到可执行动作和验证计划 |
| AI 自动分析能省掉问题定义 | AI 只适合辅助处理，不能替代业务假设和责任归属 |
| 同环比就是深入分析 | 同环比只是入口，主因和动作才是重点 |

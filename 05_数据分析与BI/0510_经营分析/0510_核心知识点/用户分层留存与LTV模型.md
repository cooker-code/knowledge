## 核心问题

用户分层、留存、复购和 LTV 的价值在于把用户群体拆成可触达、可预算、可验证的运营策略，而不是只算一组指标。

## 判断准则

| 方法 | 应回答的问题 |
|---|---|
| 留存分析 | 哪批用户在什么时间窗口留下或流失 |
| 流失分析 | 流失前有哪些行为信号和业务触点 |
| 复购分析 | 复购间隔、最佳触达时机和复购成本是什么 |
| RFM | 价值分层是否对应差异化触达策略 |
| LTV | 用户长期价值是否支持获客成本和补贴预算 |
| 路径分析 | 用户关键行为路径是否能定位流失、转化和复购触点 |

## 认知校准点

| 偏差 | 处理 |
|---|---|
| LTV 是一个固定数 | LTV 依赖人群、渠道、时间窗口、折现和成本 |
| 用户标签越多越好 | 标签必须服务触达、预算或风险控制 |
| 留存题解就是经营分析 | SQL 解法只是实现，真正要沉淀的是人群、窗口和动作 |

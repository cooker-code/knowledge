6 月 29 日，Kyligence 宣布推出**智能指标驱动的管理和决策平台 Kyligence Zen**，该产品基于 Kyligence 核心 OLAP 能力打造，利用 AI 增强引擎和预计算技术优势，提供指标目录、指标自动化、API 集成等功能，以便捷、易用、直观的产品，为用户带来**高效协同管理、业务敏捷提升、数据口径一致、开发成本降低**等价值，助力领先企业构建指标体系，驱动业务和管理目标。

指标作为量化的衡量标准，能够直接反映企业经营状况，为企业决策提供数据支撑。在实际场景中，企业为了用指标反映业务和管理情况，往往选用了不同的数据系统和平台，然而 ETL 任务、报表、仪表盘中的海量数据仍缺乏治理，企业面临着**指标口径不一致、数据信任缺失、指标无法复用、数据开发投入大、价值转化效率低**等痛点。

Kyligence 联合创始人兼 CEO 韩卿提到：“从分布在不同数据系统中的数千份报告中找到正确的指标非常困难，所以数据团队更多关注的是数据管道，但对数据的真正价值关注较少，我们相信指标驱动的管理和决策平台将帮助企业建立业务和指标之间的一致性，Kyligence Zen 希望为企业带来一种改变指标使用与管理的方法。”

在推出 Kyligence Zen 之前，Kyligence 已经凭借集业务模型、指标管理、指标加工、数据服务等于一体的企业级指标中台解决方案服务多家金融、零售、制造企业，帮助客户以指标为中心，构建数字化管理体系。因此，Kyligence Zen 依托智能指标中台建设的成功实践与丰富的行业服务经验，力求通过标准化的产品能力解决企业面临的指标管理、分析和应用痛点，以四大核心能力助力实现指标驱动的管理与决策。

*Kyligence Zen 产品架构图*

**01 目标管理**

**组织上下目标对齐，实现管理的可观测性**

Kyligence Zen 自主研发与设计的目标管理与指标对齐功能，使得组织能够从管理者视角，将企业管理分解为相关目标，设置合理的结果或过程指标，并持续追踪、对比，进一步驱动组织的数字化转型。

**02 指标目录**

**快速定义和管理指标，打造统一的指标口径**

Kyligence Zen 的指标目录功能，帮助组织轻松定义和管理指标，形成统一的指标口径，并进一步通过治理，增强数据的可信度。同时，Kyligence Zen 的指标批量导入能力，能够便捷从现有 BI 系统、数据仓库等快速导入并自动生成相关指标定义等。

**03 指标自动化**

**高效生成和推荐指标，智能捕获最有价值的信息**

Kyligence Zen 自主研发与设计的指标自动化能力，依托 Kyligence AI 增强引擎，自动从源数据库、数据仓库、数据湖等的 SQL 历史和使用日志中，动态提取数据集市中的相关指标、维度等信息，快速生成和持续维护指标目录，自动捕获最有价值的指标和数据点。同时，该产品能够通过查询历史中的更多信息，向业务用户智能推荐各种模式下的指标，例如最热指标、最重要指标等。

**04 API 集成**

**轻松构建各种数据应用，释放数据潜力**

Kyligence Zen 提供的开放 API 接口，能够轻松完成和对接各类管理、协同、BI 及可视化工具，赋能企业实现自动化管理、分析和工作流等。指标体系实现一处定义、多处复用，极大降低开发和维护成本，释放企业数据资产的潜力。

数据源上，Kyligence Zen 支持接入各种主流数据源，如 Amazon S3、CSV 等，并且提供指标可视化能力，用户可以在该产品中创建指标看板快速查看指标数据，也可以组织内发布共享、协同维护。

对于企业而言，指标驱动的管理与决策正成为其产品和服务创新、运营效率提升、盈利能力增强、企业数据文化构建的有力抓手。Kyligence 希望通过高效、便捷、易用的产品，释放数据平民化带来的“数据红利”，让每个人都能通过数据进行决策、开展业务、对齐管理和业务目标。Kyligence 坚持以客户成功为成功，全新推出的 Kyligence Zen 将助力企业以数据驱动业务发展和效率提升，支持企业创新转型，在竞争日益激烈的市场中建立发展新优势。

**关于 Kyligence**

上海跬智信息技术有限公司 (Kyligence) 由 Apache Kylin 创始团队于 2016 年创办，致力于打造下一代企业级智能多维数据库，为企业简化数据湖上的多维数据分析（OLAP）。通过 AI 增强的高性能分析引擎、统一 SQL 服务接口、业务语义层等功能，Kyligence 提供成本最优的多维数据分析能力，支撑企业商务智能（BI）分析、灵活查询和互联网级数据服务等多类应用场景，助力企业构建更可靠的指标体系，释放业务自助分析潜力。

Kyligence 已服务中国、美国、欧洲及亚太的多个银行、证券、保险、制造、零售等行业客户，包括建设银行、浦发银行、招商银行、平安银行、宁波银行、太平洋保险、中国银联、上汽、Costa、UBS、MetLife 等全球知名企业，并和微软、亚马逊、华为、Tableau 等技术领导者达成全球合作伙伴关系。目前公司已经在上海、北京、深圳、厦门、武汉及美国的硅谷、纽约、西雅图等开设分公司或办事机构。

## 用户认知重点

| 项 | 内容 |
|---|---|
| 已知基础 | 了解均值、方差、P 值等基础概念，知道 AB 实验需要统计显著性 |
| 待补边界 | 统计方法服务分析判断 vs 服务建模训练的区别；何时用频率派 vs 贝叶斯 |
| 易偏差点 | P<0.05 就等于"有效"；忽视样本量和多重检验问题 |
| 优先抽取 | 置信区间解释、效应量、样本量计算、多重检验校正 |

## 排重指纹

```text
统计方法 + 应用场景（实验/分析/决策）+ 核心概念 + 解决问题 + 适用假设/样本量/解释边界 + 对用户的认知增量
```

| 判断项 | 排重规则 |
|---|---|
| 都是 AB 实验统计 | 区分样本量估算、显著性检验、效应量和多指标问题 |
| 都是贝叶斯 | 区分先验选择、后验解释、与频率派对比和实际场景适用性 |
| 因果推断服务建模训练 | 归入 `06_机器学习`；服务业务分析决策归本目录 |
| 只有公式推导 | 没有业务判断准则时降权 |

## 已覆盖问题（按三级节点）

| 节点 | 已覆盖 | 还缺 |
|---|---|---|
| 假设检验与置信区间 | P 值、置信区间、I/II 型错误、标准误 | 样本量和功效分析 |
| 统计检验方法选择 | t 检验、方差分析、卡方检验、自由度、多重比较 | 变量类型到方法的选型树 |
| 数据标准化与描述统计 | Z-score、标准化、标准差/标准误、协方差、相关系数 | 异常值与稳健统计 |
| 相关回归与因果推断 | 回归、DID、合成控制、贝叶斯网络 | 真实业务因果识别案例 |
| 贝叶斯估计与分布认知 | 贝叶斯、最大似然、常见分布 | 贝叶斯 AB 测试 |

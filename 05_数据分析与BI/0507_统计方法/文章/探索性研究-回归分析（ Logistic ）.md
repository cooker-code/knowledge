在医学、社会学、经济学等众多领域，我们常常需要研究某一事件发生的概率与多个影响因素之间的关系。然而，传统的线性回归分析要求因变量是连续型变量，当因变量为分类变量时，线性回归就不再适用。Logistic 回归分析作为一种强大的统计方法，专门用于处理因变量为分类数据的情况，能够有效地揭示自变量与分类因变量之间的关联，帮助我们做出预测和决策。

**02**

Logistic 回归分析的流程

（一）问题提出与研究设计

明确研究问题，确定因变量（分类变量）和可能的自变量。同时，要规划好数据收集的方法、样本量的大小[样本量：科学研究与数据分析的基石](https://mp.weixin.qq.com/s?__biz=Mzk4ODM1NDQ0MA==&mid=2247483951&idx=1&sn=14ed1278bc6cd3289d91e9e05aff28aa&scene=21#wechat_redirect)以及数据收集的对象等。[抽样方法详解](https://mp.weixin.qq.com/s?__biz=Mzk4ODM1NDQ0MA==&mid=2247484051&idx=1&sn=ca71a6456fb4bd87f568a72d85c5b548&scene=21#wechat_redirect)

（二）数据收集与整理

根据研究设计，收集相关数据。在收集过程中，要确保数据的准确性和完整性。收集完成后，对数据进行整理，检查数据的质量，处理缺失值、异常值等问题。例如，可以通过删除缺失值过多的记录、插补缺失值（如均值插补、中位数插补、多重填补等方法）来处理缺失值；对于异常值，可以通过统计检验（如 Z - score 检验）或可视化方法（如箱线图）[顶刊都在用的「数据描述三件套」](https://mp.weixin.qq.com/s?__biz=Mzk4ODM1NDQ0MA==&mid=2247483666&idx=1&sn=b0b5d70e87fac788d9a94e66a7f765fb&scene=21#wechat_redirect)识别并决定是否修正或删除。

（三）模型选择

根据因变量的分类情况选择合适的 Logistic 回归模型。

* 二分类 Logistic 回归：当因变量只有两个类别，如疾病的发生（是 / 否）、患者的生存（生存 / 死亡）等，使用二分类 Logistic 回归模型。
* 多分类 Logistic 回归：若因变量有三个或三个以上无序类别，如不同类型的疾病诊断，采用多分类 Logistic 回归模型。
* 有序 Logistic 回归：当因变量是三个或三个以上有序类别，比如疾病严重程度分为轻度、中度、重度，适合使用有序 Logistic 回归模型。

（四）模型估计与拟合

使用收集和整理好的数据，通过最大似然估计\*等方法来估计 Logistic 回归模型的参数（β0，β1......βp）。模型拟合的过程就是找到一组参数值，使得模型预测的概率与实际观测到的类别尽可能接近。

注：最大似然估计（Maximum Likelihood Estimation，MLE）是一种在统计学中广泛应用的参数估计方法，其核心目标是利用已知的样本数据，找出使得样本出现的概率最大的模型参数值。

（五）模型检验与评估

* 显著性检验：对回归系数进行显著性检验，常用的方法是 Wald 检验、似然比检验等。原假设为 (H0: βi = 0)，即自变量 (Xi) 对因变量没有影响。
* 拟合优度检验：评估模型对数据的拟合程度，常用的指标有 Hosmer - Lemeshow 检验、伪 (R^2) 等。Hosmer - Lemeshow 检验原假设为模型拟合良好；伪 (R^2) 类似于线性回归中的 (R^2)，用于衡量模型的解释能力，值越接近 1，说明模型对数据的解释能力越强（一般不用）。
* 预测性能评估：通过计算灵敏度、特异度、准确率、受试者工作特征曲线下面积（AUC）等指标来评估模型的预测性能。灵敏度反映了模型正确识别阳性结果的能力，特异度反映了模型正确识别阴性结果的能力，准确率是正确预测的比例，AUC 则综合衡量了模型的整体预测准确性，AUC 越大，模型预测性能越好，一般认为 AUC 大于 0.7 时模型具有较好的预测价值。

（六）结果解释与应用

根据模型估计和检验的结果，解释回归系数的意义，分析每个自变量对因变量的影响方向和程度。例如，正的回归系数表示该自变量增加时，事件发生的概率增加；负的回归系数则表示自变量增加时，事件发生的概率降低。利用建立好的模型进行预测，为决策提供依据，如在医学中预测疾病的发生风险，帮助医生制定预防和治疗策略。

**03**

SPSS指南

（一）案例背景

研究某种疾病的发生与年龄、性别、吸烟史之间的关系。因变量为疾病发生情况（是 = 1，否 = 0），自变量分别为年龄（连续变量）、性别（男 = 1，女 = 0）、吸烟史（有 = 1，无 = 0）。

（二）数据录入

打开 SPSS 软件，进入数据视图。将收集到的数据按照行（每个观测对象为一行）和列（每个变量为一列）的形式录入，分别命名变量为 “疾病发生”“年龄”“性别”“吸烟史”。

（三）二分类 Logistic 回归操作

依次点击 “分析” - “回归” - “二元 Logistic”。设置变量：将 “疾病发生” 变量选入 “因变量” 框，将 “年龄”“性别”“吸烟史” 变量选入 “协变量” 框。定义分类变量（如果有）：由于 “性别” 和 “吸烟史” 是分类变量，点击 “分类” 按钮，将 “性别” 和 “吸烟史” 变量移至 “分类协变量” 框，SPSS 会自动对其进行编码处理。模型选项设置：点击 “选项” 按钮，可以设置一些模型估计的参数，如置信区间、迭代次数等。通常保持默认设置即可。也可以勾选 “ Hosmer - Lemeshow 拟合优度” 用于检验模型拟合优度。输出结果：点击 “确定” 按钮，SPSS 将输出一系列结果，包括模型系数的估计值、标准误、Wald 检验值、显著性水平（P 值）、优势比（OR 值）等。同时会输出 Hosmer - Lemeshow 检验结果用于评估拟合优度。

（四）结果解读

* 回归系数与显著性：查看 “方程中的变量” 表格，关注 “B” 列（回归系数估计值）、“S.E.” 列（标准误）、“Ward” 列（Wald 检验值）和 “Sig.” 列（显著性水平 P 值）。若某自变量的 (P < 0.05)，说明该自变量对疾病发生有显著影响。例如，若年龄的 (P < 0.05)，且回归系数为正，表明年龄越大，疾病发生的风险越高。
* 优势比（OR 值）：同样在 “方程中的变量” 表格中，“Exp (B)” 列即为优势比。例如，若吸烟史的 (Exp(B) = 2.5)，表示有吸烟史的人发生该疾病的优势是无吸烟史人的 (2.5) 倍。
* 拟合优度：查看 “Hosmer - Lemeshow 检验” 表格，若 (P > 0.05)，说明模型对数据的拟合效果较好，即模型能够较好地解释疾病发生与这些自变量之间的关系。

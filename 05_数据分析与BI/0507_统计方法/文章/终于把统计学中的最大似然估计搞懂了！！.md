最大似然估计（MLE）是一种在统计学中广泛使用的参数估计方法。

它的基本思想是：在已知样本数据的情况下，选择一组参数值，使得在这些参数下观测到当前数据的概率（即似然函数）最大。

### 基本原理

假设我们有一组独立同分布的观测数据

这些数据服从某个已知分布族，其分布由参数  控制。

我们的目标是估计参数 。

#### 1.似然函数

首先，构建似然函数 ，它表示在参数  下，观测数据  出现的联合概率（对于连续分布，是联合概率密度）。

其中， 是单个观测值  在参数  下的概率密度函数（对于连续分布）或概率质量函数（对于离散分布）。

#### 3.最大化对数似然函数

通过对参数  求导并令导数等于零，找到使对数似然函数最大的参数值 ，即最大似然估计。

### 最大似然估计的性质

* 一致性：随着样本量的增加，MLE 估计量趋近于真实参数值。
* 有效性：在一定条件下，MLE 是渐近有效的，即在所有一致估计量中，具有最小的方差。
* 无偏性：在大样本下，MLE 的偏差趋于零。

#### 示例代码

下面是使用 Python 进行最大似然估计的示例代码。

我们将使用 scipy.optimize 模块来最大化对数似然函数。

```
import numpy as np  
from scipy.optimize import minimize  
import matplotlib.pyplot as plt  
  
# 生成样本数据  
np.random.seed(0)  
true_mu = 5  
true_sigma = 2  
sample_size = 100  
data = np.random.normal(loc=true_mu, scale=true_sigma, size=sample_size)  
  
# 定义负对数似然函数  
def neg_log_likelihood(params, data):  
    mu, sigma = params  
    if sigma <= 0:  
        return np.inf  # sigma必须大于0  
    n = len(data)  
    return 0.5 * n * np.log(2 * np.pi * sigma**2) + np.sum((data - mu)**2) / (2 * sigma**2)  
  
# 初始参数猜测  
initial_params = [0, 1]  
  
# 最小化负对数似然函数  
result = minimize(neg_log_likelihood, initial_params, args=(data,), method='L-BFGS-B',  
                  bounds=[(None, None), (1e-6, None)])  
  
if result.success:  
    mle_mu, mle_sigma = result.x  
    print(f"最大似然估计结果:\n均值 μ = {mle_mu:.4f}\n标准差 σ = {mle_sigma:.4f}")  
else:  
    raise ValueError("最大似然估计未收敛")  
  
# 对比理论值和估计值  
plt.hist(data, bins=15, density=True, alpha=0.6, color='g', label='样本数据')  
  
# 绘制真实分布  
x = np.linspace(min(data), max(data), 100)  
plt.plot(x, (1/(true_sigma * np.sqrt(2 * np.pi))) * np.exp(- (x - true_mu)**2 / (2 * true_sigma**2)),  
         'k', linewidth=2, label='真实分布')  
  
# 绘制MLE估计的分布  
plt.plot(x, (1/(mle_sigma * np.sqrt(2 * np.pi))) * np.exp(- (x - mle_mu)**2 / (2 * mle_sigma**2)),  
         'r--', linewidth=2, label='MLE估计分布')  
  
plt.legend()  
plt.xlabel('x')  
plt.ylabel('密度')  
plt.title('最大似然估计示例')  
plt.show()
```

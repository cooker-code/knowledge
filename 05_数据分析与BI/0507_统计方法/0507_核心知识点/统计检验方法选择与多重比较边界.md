## 核心问题

统计检验先问“变量是什么类型、比较几个组、是否配对、是否满足假设、业务要什么决策”，再选择方法。不能因为工具里有一个检验按钮，就把 P 值当结论。

## 判断准则

| 问题形态 | 优先方法 | 前置检查 |
|---|---|---|
| 同一个对象前后比较 | 配对 t 检验 | 配对关系、差值分布、异常值 |
| 两个独立组均值比较 | 独立样本 t 检验 | 正态性、方差齐性、样本量 |
| 三个及以上组均值比较 | 方差分析 | 方差齐性、组间独立、事后多重比较 |
| 两个类别变量是否独立 | 卡方检验 | 频数表、期望频数、样本独立 |
| 多次两两比较 | Bonferroni、Tukey、Scheffe 等校正 | 多重检验导致 I 型错误膨胀 |
| 多因子试验 | 全因子/部分因子设计 | 主效应、交互效应、中心点、失拟 |

## 认知偏差

| 常见错误认知 | 正确理解 |
|---|---|
| 两组用 t 检验，多组用方差分析就够了 | 还要看是否配对、变量类型、分布假设和方差齐性 |
| 方差分析显著就知道哪组不同 | 显著后还需要事后多重比较 |
| 多做几次检验总能找到显著 | 多重比较会提高误报风险，必须校正 |
| 自由度只是公式参数 | 自由度表达估计中还剩多少独立信息，会影响方差估计和检验分布 |
| P 值显著就是业务有效 | 还要看效应量、置信区间、成本和业务阈值 |

## 待验证缺口

- 需要补一张“变量类型 -> 组数 -> 是否配对 -> 方法”的选型树。
- 需要把 AB 实验中的多指标、多分层、多实验并行问题接入多重检验校正。

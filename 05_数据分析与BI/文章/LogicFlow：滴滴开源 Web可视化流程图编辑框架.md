LogicFlow 是一个功能强大、灵活易用的流程图编辑框架，适用于需要在 Web 应用中实现流程图功能的场景。无论是业务流程管理还是技术架构展示，LogicFlow 都能提供相应的支持

支持7种基础节点：矩形、圆形、椭圆、多边形、菱形、文本、HTM

支持自定义节点

支持3种基础边：直线、直角折线、贝塞尔曲线

支持自定义边

支持扩展插件，包含菜单、拖拽面板、控制面板、小地图、框线功能等

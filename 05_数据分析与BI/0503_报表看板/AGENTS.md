| 项 | 内容 |
|---|---|

## 排重指纹

```text
报表工具 + 设计维度（信息架构/图表/筛选/权限/治理）+ 核心机制 + 解决问题 + 可维护性/用户体验边界 + 对用户的认知增量
```

| 判断项 | 排重规则 |
|---|---|
| 都是 Superset | 区分数据集配置、图表类型、仪表盘布局、权限和 API 使用 |
| 都是 Grafana | 区分 PromQL/LogQL 面板、告警规则、变量和用于可观测性 vs BI 的场景差异 |
| 可视化技术本体 | 归入 `0504_可视化`；工具使用归本目录 |

# 可视化

## 类目定位

| 项 | 内容 |
|---|---|
| 一级类目 | 数据分析与 BI |
| 二级类目 | 可视化 |
| 核心问题 | 如何用图表、图谱、交互和视觉编码表达数据关系、趋势、结构和洞察 |
| 不解决什么 | 不承接纯 BI 平台建设、指标口径治理、通用文档排版或无数据分析目标的设计工具 |
| 用户当前认知假设 | 以 [用户画像.md](../../用户画像.md) 为准；优先关注可视化表达是否服务分析判断，而不是工具清单 |

## 排重准则

问题指纹：

```text
可视化对象 + 图表/图谱/交互/视觉编码模块 + 核心机制 + 解决问题 + 分析场景边界 + 对用户的认知增量
```

## 待补技术和问题

| 技术/问题 | 为什么要补 | 优先级 |
|---|---|---|
| 图数据可视化 | 开源生态、血缘、关系分析常需要图谱表达 | 中 |
| 可视化误导与图表选择 | 需要形成分析判断准则 | 中 |

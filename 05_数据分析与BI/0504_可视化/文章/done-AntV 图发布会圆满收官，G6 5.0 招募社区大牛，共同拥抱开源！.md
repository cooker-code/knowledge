**01**

**G6 5.0：优雅 & 专业的图可视化引擎**

演讲嘉宾：十吾 ( AntV G6 负责人 ) 沧东 ( AntV G 负责人 )

G6 5.0 alpha 带你玩转图可视化！借助规范和自定义元素功能、多渲染器支持、高性能布局和交互、插件化架构，G6 5.0 能够大幅提升图形渲染性能和交互体验。此外，还支持 JSON 描述方式自定义主题和 3D 大图展示。快来尝试吧！

Github: *https://github.com/a**ntvis/**G6* 

在线求个🌟 ~

**02**

**G6VP 2.0 热浪来袭**

演讲嘉宾：山果 ( AntV G6VP 负责人 )

连接上下游产品，G6VP 2.0 满足你的所有需求！

G6VP 2.0 已经从专注于提高生产效率的工具产品，转型成为**提高流转效率的平台产品**，致力于解决上下游链接问题。山果老师放话，“无论你的角色是产品经理，后端开发，前端开发，还是分析师，在 G6VP 上都能找到你想要的功能”！

Github: *https://github.com/a**ntvis/**G6VP*

在线求个🌟 ~

**G6VP 始终保持初心，帮助更多的用户用上图，用好图！**

**03**

**聊聊图计算**

演讲嘉宾：泰初 ( GeaFlow 负责人) 友风 ( TuGraph-DB 负责人 )

第三趴邀请了泰初老师和友风老师，他们分享了图数据库和流式图计算产品，并介绍了它们的特点和应用案例。此外，两位嘉宾还通过视频演示了产品的使用和操作方法，直观展现了产品的功能和优势。

**04**

**图分析在安全领域的应用实践**

演讲嘉宾：夏漱 ( 蚂蚁集团大安全 产品经理) 志飏 ( 蚂蚁集团大安全 前端工程师)

夏漱老师和志飏老师的对话主要讨论了基于 G6VP 平台仅仅开发 3 周就上线的**反洗钱分析定性工具**，重点包括使用图技术解决资金链路追踪定位和犯罪行为关联的问题，利用关系数据的算法和异常检测实现关联关系的分析，实现基于交易、画布和关键信息的动态交互，增加了点、边打标的能力，以及推进了算法策略迭代的持续反馈能力。

相比传统的图分析工具，优势在于算法策略和前端的融合、多种交互联动方式、以及沉淀能力的提升。他们还强调了图技术的重要性，并呼吁大家共同构建更加正向的循环。

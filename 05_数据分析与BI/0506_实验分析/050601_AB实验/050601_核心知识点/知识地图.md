# 050601_AB实验知识地图

## 模块定位

| 项 | 内容 |
|---|---|
| 所属目录 | 05_数据分析与BI / 0506_实验分析 / AB实验 |
| 解决的问题 | AB 实验如何从假设、分流、指标、显著性，走到因果解释和业务决策 |
| 典型输入 | 实验方案、用户行为数据、指标口径、分流日志、实验结果 |
| 典型产出 | 实验结论、效应估计、风险边界、复盘动作 |

| 节点 | 节点问题 | 当前原文锚点 | 待补缺口 |

| A01 分流设计 | 随机化单位、观察单位、实验范围是否一致 | [A_B测试进阶：当随机化与观察单位不一致，你该怎么办？](../文章/A_B测试进阶：当随机化与观察单位不一致，你该怎么办？.md) | 需要沉淀单位不一致的偏差和修正动作 |
| A02 指标体系 | 主指标、护栏指标、过程指标如何选择 | 暂无正式知识点 | 需要和指标体系目录交叉引用 |
| A03 效应估计 | 显著性、置信区间、效应量如何解释 | [AB实验的进阶打开方式：从统计显著到因果推断](../文章/AB实验的进阶打开方式：从统计显著到因果推断.md) | 需要补样本量和多重检验 |

| A05 决策复盘 | 上线、回滚、继续实验或分层策略如何决定 | 暂无正式知识点 | 需要补决策阈值和业务成本 |

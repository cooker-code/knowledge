## 核心问题

AB 实验要证明的是某个干预在特定样本、时间窗口和指标体系下的效果。统计显著只是效应估计的一部分，不能自动推出业务因果解释。

## 判断准则

| 环节 | 准则 |
|---|---|
| 假设 | 干预、目标人群、影响路径和预期指标必须先写清 |
| 分流 | 随机化单位、观察单位和分析单位要一致；不一致时要显式修正 |
| 指标 | 主指标、护栏指标、过程指标和反向指标要分层 |
| 估计 | 同时看效应量、置信区间、样本量和业务成本 |
| 解释 | POM 负责效应估计，SCM 或业务机制负责解释路径 |
| 决策 | 上线、回滚、继续实验或分层放量要有阈值 |
| 平台化 | 实验系统要沉淀分流、指标、自动化分析、权限和实验资产复用 |

## 认知校准点

| 偏差 | 正确处理 |
|---|---|
| P 值显著就说明业务有效 | 还要看效应量、成本、护栏指标和长期影响 |
| 随机化后就没有偏差 | 单位不一致、污染、网络效应和样本流失仍会制造偏差 |
| 实验结论天然可外推 | 只能外推到相似人群、时间窗口和业务机制 |

## 待验证缺口

- 网络效应、长期效应和实验污染的处理策略。
- 多指标、多实验并行时的决策规则。

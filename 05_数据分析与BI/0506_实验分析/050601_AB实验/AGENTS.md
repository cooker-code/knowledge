## 技术定位

| 项 | 内容 |
|---|---|
| 技术名 | AB 实验 |
| 一级类目 | 数据分析与 BI |
| 二级类目 | 实验分析 |
| 技术本体 | 通过随机分流、对照组、实验指标和统计检验评估干预效果，并支撑业务决策 |
| 全局位置 | 位于假设提出、实验设计、指标评估、因果解释和决策复盘之间 |
| 主要使用者 | 数据分析师、产品、运营、增长团队 |
| 主要产出 | 实验方案、样本量、显著性判断、效应估计、异质性分析、决策建议 |

## 排重准则

问题指纹：

```text
实验对象 + 分流/观察单位 + 指标体系 + 统计机制 + 因果解释边界 + 决策增量
```

| 判断项 | 排重规则 |
|---|---|
| 都是 AB 实验入门 | 基础概念不重复沉淀 |
| 补充实验失败模式 | 合并到对应设计或评估知识点 |
| 补充因果解释 | 记录 POM、SCM、混杂、中介、异质性等边界 |
| 只给业务案例 | 案例不改变准则时只保留原文锚点 |

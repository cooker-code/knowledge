# 030302_Flink CDC 版本记录

> 更新日期：2026-06-18。版本事实以官方 Release、官方公告和 Apache JIRA Release Notes 为准。

## 官方锚点

- GitHub Releases: https://github.com/apache/flink-cdc/releases
- Flink CDC 3.6.0 公告: https://flink.apache.org/2026/03/30/apache-flink-cdc-3.6.0-release-announcement/
- Flink CDC 3.5.0 公告: https://flink.apache.org/2025/09/26/apache-flink-cdc-3.5.0-release-announcement/
- Flink CDC 3.4.0 公告: https://flink.apache.org/2025/05/16/apache-flink-cdc-3.4.0-release-announcement/

## 版本敏感结论

| 版本 | 官方页确认点 | 对知识点的影响 |
|---|---|---|
| 3.6.0 | GitHub Release 标记为 Latest；下载区明确仅支持 JDK 11 及以上；发布包区分 Flink 1.20.x 和 Flink 2.2.x | 任何 3.6 结论必须标注 JDK 和 Flink 版本；旧 JDK 环境不能直接套用 |
| 3.5.0 | Release 列出 Pipeline Connector，包括 MySQL、Doris、StarRocks、Kafka、Paimon、Iceberg、PostgreSQL、Fluss 等 | Pipeline 选型要按具体 Connector 和目标端语义验证 |
| 3.4.0 | 官方公告和 JIRA Release Notes 存在 | 本地 `3.4 pipeline` 文章只作为 Pipeline 行为线索，需回查官方说明 |
| 3.0 | 本地多篇文章指向框架化转折 | 长期结论落在 Pipeline Framework，不再把 Flink CDC 当单个 MySQL Source |

## 待补证

- 3.6.0 Oracle Source、PostgreSQL Schema Evolution、Transform 增强等具体行为需要逐项回官方公告和 JIRA。
- 每条 Pipeline Connector 的目标端幂等、DDL 和恢复语义需要单独实验。

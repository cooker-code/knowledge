# 030506_湖仓架构与选型 版本记录

> 更新日期：2026-06-18。本目录不是单一软件版本线，版本记录只维护选型依赖的官方版本锚点。

## 依赖版本锚点

| 对象 | 官方锚点 | 版本敏感点 |
|---|---|---|
| Iceberg | https://github.com/apache/iceberg/releases | v3、删除向量、行级血缘、Catalog 和引擎支持 |
| Hudi | https://github.com/apache/hudi/releases | Timeline、索引、表服务、Spark/Flink/Trino 兼容 |
| Paimon | https://paimon.apache.org/download/ | Flink/Spark 集成、主键表、Changelog、系统表 |
| Delta Lake | https://github.com/delta-io/delta/releases | Spark 版本、Delta Kernel、UniForm、Catalog Managed Tables |
| Flink | https://flink.apache.org/downloads/ | 读写湖表 runtime、Flink SQL、Connector 兼容 |
| Spark | https://spark.apache.org/releases/ | Spark 版本决定 Delta/Iceberg/Hudi/Paimon runtime 坐标和行为 |

## 版本敏感结论

- “某表格式支持 Schema Evolution”必须同时标明表格式版本和引擎版本。
- “多引擎访问”必须同时核对 Catalog、runtime 包、存储权限和读写语义。
- “替代 Hive/数仓”这类结论不进入版本记录，只进入选型边界知识点并标风险。

# 030503_Iceberg 版本记录

> 更新日期：2026-06-18。Iceberg 版本事实以 Apache Iceberg GitHub Release 和官方文档为准。

## 官方锚点

- GitHub Releases: https://github.com/apache/iceberg/releases
- 官方文档: https://iceberg.apache.org/docs/latest/

## 版本敏感结论

| 版本 | 官方页确认点 | 对知识点的影响 |
|---|---|---|
| 1.10.x | GitHub Release 列出 1.10.0、1.10.1、1.10.2 等维护线，包含 deletion vector、Flink、Spark、Hive、Open API、安全依赖修复等条目 | v3/删除向量/行级能力相关结论必须标版本，并核对引擎支持 |
| 1.9.x | Gravitino 1.0 Release 明确其 Iceberg 支持升级到 1.9.0 | 元数据平台集成 Iceberg 时要看平台内置版本 |
| Hive 4.x + Iceberg | 本地文章涉及 Hive 与 Branch/Tag | Hive 接入结论要标 Hive 和 Iceberg 版本 |

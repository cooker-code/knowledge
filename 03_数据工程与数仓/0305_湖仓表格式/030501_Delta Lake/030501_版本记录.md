# 030501_Delta Lake 版本记录

> 更新日期：2026-06-18。Delta Lake 当前目录暂无本地正文文章，版本记录先作为官方锚点和后续补证入口。

## 官方锚点

- GitHub Releases: https://github.com/delta-io/delta/releases
- Delta 文档: https://docs.delta.io/

## 版本敏感结论

| 版本 | 官方页确认点 | 对知识点的影响 |
|---|---|---|
| 4.1.0 | Release 说明默认构建基于 Spark 4.1.0，同时保留 Spark 4.0.1 兼容；Maven artifact 增加 Spark 版本后缀 | Delta 4.1 结论必须标注 Spark 版本和 artifact 坐标 |
| 4.0.1 | Release 说明修复 4.0.0，并包含 Catalog managed table 命名变化、UC OAuth、Spark 4.0.1 兼容修复 | Catalog managed/UC 相关结论要看 4.0.1 之后语义 |
| 4.0.0 | Release 说明 Delta Standalone、Delta Flink、Delta Hive 不再随 4.x 发布，转向 Delta Kernel | 与 Flink/Hive 直接集成的旧经验要降权，重点追 Delta Kernel 路线 |

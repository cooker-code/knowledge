# 030701_Debezium 版本记录

> 更新日期：2026-06-18。版本事实以 Debezium 官方 Releases Overview 和文档为准。

## 官方锚点

- Releases Overview: https://debezium.io/releases/
- Documentation: https://debezium.io/documentation/

## 版本敏感结论

| 版本线 | 官方页确认点 | 对知识点的影响 |
|---|---|---|
| 3.6 development | 官方 Releases Overview 标注为 development，日期 2026-06-12 | 不作为生产稳定默认线 |
| 3.5 latest stable | 官方标注 latest stable，日期 2026-06-02 | 新链路优先查 3.5 文档和测试版本矩阵 |
| 3.4 stable | 官方标注 stable，日期 2026-03-30 | 作为稳定线候选 |
| 2.7 stable | 官方仍列为 stable，Java 11+ | 老链路迁移要看 Java/Kafka Connect/数据库版本差异 |

## 版本敏感点

- Java 版本、Kafka Connect 版本、源数据库版本、connector 插件和 snapshot/offset 行为都必须标版本。
- Debezium -> Kafka -> Hudi/Paimon/Iceberg 的下游正确性不由 Debezium 单独保证。

## 待补证

- MySQL/PostgreSQL/Oracle connector 在 3.5 稳定线下的 snapshot、schema history、delete tombstone 和 offset 恢复行为。

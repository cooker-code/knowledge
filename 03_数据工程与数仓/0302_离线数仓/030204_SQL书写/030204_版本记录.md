# 030204_SQL书写 版本记录

SQL书写不是单一软件产品，但结论高度依赖 Hive SQL、Spark SQL、Trino、Doris 等方言和执行引擎版本。本目录只维护离线数仓 SQL 写法的版本敏感点，具体产品版本详见对应技术目录。

## 官方锚点

- Hive 版本记录：../030202_Hive/030202_版本记录.md
- Spark 版本记录：../030205_Spark/030205_版本记录.md

## 版本敏感点

| 主题                         | 依赖对象                 | 处理规则                                     |
| -------------------------- | -------------------- | ---------------------------------------- |
| 窗口函数                       | Hive SQL / Spark SQL | 写法要标注执行引擎；函数语义和执行顺序不能跨引擎直接复用。            |
| `lateral view` / `explode` | Hive SQL / Spark SQL | 展开后粒度变化是通用风险，但语法细节和优化行为要看引擎。             |
| `COUNT DISTINCT`           | Hive / Spark         | 膨胀和 Shuffle 是通用风险；执行计划、AQE 和局部聚合能力看引擎版本。 |
| `GROUPING SETS` / `CUBE`   | Hive / Spark         | 口径和中间数据膨胀都要验证，不按单个语法清单沉淀。                |
| Join Hint / Skew Hint      | Spark SQL / Hive     | Hint 是否生效必须看引擎版本、配置、统计信息和 Explain。       |

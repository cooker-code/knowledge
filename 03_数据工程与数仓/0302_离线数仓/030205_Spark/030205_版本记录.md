# 030205_Spark 版本记录

Spark 版本结论必须区分 Apache Spark 上游版本、企业发行版/Databricks Runtime、Scala/JDK 版本、Hive Catalog 兼容和第三方 Native Engine。涉及迁移、SQL 语义、连接器、AQE 或 Spark Connect 时必须查官方 Release Notes。

## 官方锚点

- Apache Spark Downloads：https://spark.apache.org/downloads.html
- Apache Spark Releases：https://spark.apache.org/releases/
- Spark 4.0.0 Release Notes：https://spark.apache.org/releases/spark-release-4-0-0.html
- Spark 3.3.0 Release Notes：https://spark.apache.org/releases/spark-release-3-3-0.html

## Spark 4.0 关键边界

| 能力 | 官方依据 | 判断边界 |
|---|---|---|
| Scala/JDK | Spark 4.0.0 Release Notes | 官方列出 Scala 2.13 默认、丢弃 Scala 2.12，丢弃 JDK 8/11 并以 JDK 17 为默认。 |
| Spark SQL | Spark 4.0.0 Release Notes | VARIANT、SQL UDF、Session Variables、Pipe syntax、String Collation、ANSI 默认等都影响 SQL 迁移。 |
| AQE/SQL 执行 | Spark 4.0.0 Release Notes | AQE 仍有正确性和输出列顺序等修复线索，不能只看参数。 |
| Hive Catalog | Spark 4.0.0 Release Notes | Spark 4.0 支持 Hive 4.0 metastore，同时移除 Hive 2.0 之前支持；Hive 互通必须双查版本。 |
| Spark Connect | Spark 4.0.0 Release Notes | Connect API 覆盖和客户端增强是接入形态变化，不等同于执行引擎性能提升。 |

## 使用规则

- 写 Spark 4 结论时必须同时标注 Scala/JDK/Hive Catalog 兼容影响。
- Databricks Runtime、Gluten、Velox、Sail 等第三方能力不能直接等同于 Apache Spark 上游能力。
- 下载页、Release Notes 和 Maven 坐标出现差异时，以目标部署包和 release notes 复核为准。

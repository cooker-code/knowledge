# 030203_Kyuubi 版本记录

Kyuubi 的版本判断必须同时看 Kyuubi 自身版本、Spark/Flink/Hive 等引擎版本、Scala/JDK 兼容性和企业发行版依赖。发布文章只作为线索，正式结论以官方下载页和 release notes 为准。

## 官方锚点

- Apache Kyuubi Releases：https://kyuubi.apache.org/releases.html
- Kyuubi 1.11.1 Release Notes：https://kyuubi.apache.org/release/1.11.1.html
- Kyuubi 1.8.0 Release Notes：https://kyuubi.apache.org/release/1.8.0.html

## Apache Kyuubi 上游版本

| 版本 | 官方发布日期 | 对本目录的意义 |
|---|---:|---|
| 1.11.1 | 2026-03-26 | 当前官方下载页标注的 latest；release notes 中包含 Spark 4.1.1 相关依赖更新和 Spark 4.2 兼容线索。 |
| 1.10.3 | 2026-01-02 | 1.10 维护线；企业升级时需和 1.11 控制面能力分开看。 |

| 1.9.4 | 2025-05-27 | 1.9 维护线；与 1.8 企业文章相比，不能混用能力边界。 |
| 1.8.3 | 2024-11-27 | 1.8 维护线；本地多篇文章围绕 1.8，应核对到 1.8.0/1.8.x 文档。 |
| 1.8.0 | 2023-11-07 | 文章中“1.8 特性”的主要版本锚点。 |
| 1.6.1-incubating | 2022-11-16 | 1.6 incubating 线；用于理解早期社区文章，不代表当前能力。 |

## 关键版本边界

| 能力 | 官方依据 | 判断边界 |
|---|---|---|
| Kyuubi 1.8 | 1.8.0 Release Notes | 官方标注 Java 8/11/17、Scala 2.12、初步 Scala 2.13；Spark 3.1-3.4 fully verified、初步支持 Spark 3.5；Flink 1.16-1.18 fully verified。 |
| Flink Engine | 1.8.0 Release Notes | Flink YARN Application Mode 是 1.8 文章重点，但 Flink 状态、Checkpoint、作业恢复仍归 Flink 节点。 |
| AuthZ / Lineage | 1.8.0 Release Notes | Spark AuthZ、Iceberg/DSv2、Atlas hook 等是控制面增强，不能等同于权限治理全链路完成。 |
| Kyuubi 1.11.1 | 1.11.1 Release Notes | 1.11.1 是维护发布，包含 Spark 4.1.1 依赖更新、Spark 4.2 兼容等线索；具体生产兼容要继续查目标引擎版本。 |

## 使用规则

- 涉及引擎兼容时，同时写 Kyuubi、Spark/Flink/Hive、Scala/JDK 版本。
- 发布新闻只能进入本文件；只有改变架构判断时才进入知识点。
- 企业实践中的版本行为必须回查官方 release notes 或目标发行版文档。

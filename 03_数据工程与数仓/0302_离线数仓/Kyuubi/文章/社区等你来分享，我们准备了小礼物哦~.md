去年，社区发起了 "Who's using Apache Kyuubi？" —— 面向所有社区成员的征集活动。旨在希望作为用户或者开发者的成员留下实际使用场景，以便我们后续基于此探索更好的解决方案，制定版本计划，解决实际问题。

在去年的征集活动中，我们收获了众多社区成员提交的使用场景，涉及多个行业，如云基础设施、互联网、金融、医疗、打车服务、物流、游戏和 AICG，等等。大家的留言促使了 Kyuubi 的不断发展，也让越来越多的开发者和用户加入了 Kyuubi 社区。

\*图片仅展示部分企业用户

**用户征集**

**2023年，活动仍在继续。**欢迎前往 GitHub Discussions 补充您的信息：

https://github.com/apache/incubator-kyuubi/discussions/925

我们期待您的留言，内容包括但不限于以下部分：

* **Organization / 组织****:** e.g. NetEase
* **Corp.Region / 地区:**e.g. Hangzhou, China
* **Contact / 联系方式:** e.g. yao@apache.org
* **Usage Scenario / 使用场景:** e.g. we use Apache Kyuubi as a multi-tenant Spark server providing Spark as a service,which facilitates our users to build a lot of big data workloads, including BI reporting, ETL, etc. It's a perfect tool to move Hive QL jobs smoothly to Spark SQL.

为感谢您对 Apache Kyuubi 的长期支持，完成用户信息上传的同学可以在社区交流群内 **@Kyuubi 小助手** 领取专属的 Kyuubi 定制周边。

**投稿征文**

今年我们新增了投稿征文活动，**征集与 Kyuubi 相关的稿件内容，不论是炫技、经验、还是试用报告、学习体会等内容，**欢迎社区的小伙伴们分享关于你和 Kyuubi 的故事，促进社区成员们相互启发、学习和技术交流。

在活动时间内将您想要投稿的文章发送至 **Kyuubi 邮箱 apachekyuubi@163.com**，邮件写明作者姓名、作者信息，标题格式：【Kyuubi 投稿】+自拟标题。

**Apache Kyuubi PMC 成员将根据文章质量进行评选是否发布。**

****\*\*\*投稿截止日期：2023年12月17日\*\*\*****

参与征文活动的社区小伙伴，我们也准备了相应的礼品，获取规则：

* 经 PMC 成员审核，公众号采纳发布的用户，可获得 Kyuubi 社区周边一份（鼠标垫、棒球帽、文化衫等任选其一）。
* 文章经公众号后一周内阅读量达到 **600+** 的文章作者可获得 Kyuubi 定制双肩包。

* 文章经公众号后一周内阅读量达到 **1000+** 的文章作者可获得 Kyuubi 定制茶具一套。

**以上就是本次分享有礼活动的主要内容，期待看到大家的分享，快快行动起来吧~**

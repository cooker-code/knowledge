## 排重指纹

```text
数仓对象 + 所属层次/模块 + 核心机制 + 解决问题 + 口径/性能/治理边界 + 对用户的认知增量
```

| 判断项 | 排重规则 |
|---|---|
| 都是 Hive SQL 函数 | 只有函数清单不新建；涉及口径、版本差异或可复用模型才沉淀 |
| 都是性能优化 | 按 Join、倾斜、小文件、分区裁剪、资源参数拆分 |
| 都是建模 | 按事实表、维表、拉链、快照、累计指标拆分 |
| 都是面试题 | 大多跳过，除非补充真实工程边界 |
| 同一口径问题 | 优先合并到已有主题，追加原文锚点 |

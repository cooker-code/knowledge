# 030202_Hive 版本记录

Hive 文章中的版本结论必须区分 Apache 上游版本、企业发行版组件版本和执行引擎版本。涉及语法、函数、表结构演进、Metastore 兼容性和 Spark/Hive Catalog 互通时，先查官方来源。

## 官方锚点

- Apache Hive 官网：https://hive.apache.org/
- Apache Hive GitHub Tags：https://github.com/apache/hive/tags
- Apache Hive Wiki：https://cwiki.apache.org/confluence/display/Hive
- Spark 4.0.0 Release Notes：https://spark.apache.org/releases/spark-release-4-0-0.html

## Apache Hive 上游版本

| 版本 | 官方标签日期 | 对本目录的意义 |
|---|---:|---|
| 4.2.0 | 2025-11-21 | 当前 GitHub tags 中较新的 4.x release 标签；后续 Hive 4.x 语法和 Metastore 行为必须单独核对。 |
| 4.1.0 | 2025-07-30 | 4.x 维护线；公众号或环境若写 Hive 4.1，要和企业发行版 Hive 3.x 区分。 |
| 4.0.1 | 2024-10-02 | Hive 4.0 修复线；适合核对 4.0 迁移问题。 |
| 4.0.0 | 2024-03-29 | Hive 4.x 起点；不要把 Hive 4 能力默认套到 CDP/HDP 等发行版。 |
| 3.1.3 | 2022-04-07 | 很多企业发行版仍以 Hive 3.x 或变体为基线，迁移时要单独看发行版补丁。 |
| 2.3.10 | 2024-05-09 | 2.3 维护线标签仍存在，说明旧线在存量环境中可能长期存在。 |

## 版本敏感边界

| 主题 | 为什么敏感 | 处理规则 |
|---|---|---|
| Hive 4.x | 语法、依赖、Metastore 和执行行为可能与 3.x 不同 | 知识点只写机制，具体行为标注验证版本。 |
| Spark 读 Hive Catalog | Spark 4.0 release notes 明确提到 Hive Catalog 相关兼容项 | Spark/Hive 互通问题同时看 Spark 和 Hive 版本。 |
| 表结构演进 | 新增字段行为会受文件格式、分区旧数据和读取引擎影响 | 写结论时同时标注 Hive 版本、存储格式和读取引擎。 |
| 函数/SQL 语义 | 日期函数、窗口函数、类型转换可能随 Hive/Spark SQL 方言变化 | SQL 书写类结论优先放 SQL书写目录，Hive 目录只保留 Hive 方言边界。 |

## 使用规则

- 文章给出的版本号只作为线索，不能直接当官方结论。
- 依赖发行版的行为要标注发行版名和组件版本，不要只写 Apache Hive 版本。
- 版本发布、兼容性、配置/API 行为变化优先进入本文件，不新建普通知识点。

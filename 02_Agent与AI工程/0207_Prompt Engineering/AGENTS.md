## 排重指纹

```text
Prompt 对象（系统/用户/工具）+ 设计维度（结构/示例/格式/约束）+ 核心机制 + 解决问题 + 模型版本/可移植边界 + 对用户的认知增量
```

| 判断项 | 排重规则 |
|---|---|
| 都是 few-shot | 区分示例选择策略、格式一致性、动态 few-shot 和 CoT 示例 |
| 都是输出格式控制 | 区分 JSON Schema 约束、重试机制、结构化输出 API 和 Markdown 格式 |
| 推理范式（CoT/ToT/ReAct） | 归入 `0212_COT&TOT&REACT`，本目录只保留 Prompt 侧互链 |
| 纯模型/图像生成提示词 | 归入 `01_LLM与大模型/0101_模型能力` |
| 只有 Prompt 模板集合 | 没有设计准则和失败模式时不新建知识点 |

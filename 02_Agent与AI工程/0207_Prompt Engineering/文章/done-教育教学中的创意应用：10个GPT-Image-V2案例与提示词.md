1、做英语知识卡片：

过去我们想把知识组织一下，需要做个PPT，或者在网上找图。现在利用AI可以快速把想要呈现的知识点，条理清晰地放到一张图上。

```
提示词：请你把XXX知识点做成信息图。
```

2、做科学小海报：

科学性的知识需要丰富其画面，AI不仅能把知识性的内容合理组织，还能加上孩子喜欢的风格。无论是卡通风格还是动漫风格都能做到。

```
提示词：把XXX知识点做成科学海报，卡通风格。
```

3、做诗词视觉化呈现：

语文老师福音！有时候我们不仅传递知识，更多的是传递一种诗歌审美。我们利用AI把诗歌或者短文放置在合适的场景中，传递给学生一种综合的学习体验。

```
提示词：语文诗词，《将进酒》原文呈现，不需要拼音，整体图片氛围契合诗句风采
```

4、制作英语单词思维导图：

我本人教英语。在教学过程中我很喜欢利用思维导图工具。有了AI，我们可以针对一个词快速地进行思维导图式的建构，辅助学生发散联想记忆。

```
提示词：给我做一张图 中间是一个英语单词，周围是因为这个单词而发散出来的 整体是一个思维导图，风格化突出
```

5、重现历史场景对话：

历史政治领域经常要跟事件、人物打交道。对于这类需要强背诵的事情，用卡片的方式能够有效促进学生的记忆，同样也能激发学生兴趣。并且这一部分可以继续深化，我们可以做人物与人物之间的对话卡片。

```
提示词1：现在你根据一个真实的历史场景，文艺复兴 然后你绘制一张体现文艺复兴精神的图片，要用那个时期的风格并且在这个图上要有那个时候的人，人旁边要有标注
```

```
提示词2：你现在做一个以文艺复兴三杰正在圆桌讨论的漫画，里面要有台词3:4的比例用知名漫画家的风格
```

6、制作科学图景，以地理为例：

我在中学学习地理的时候，老是对洋流搞不清楚。然后也受限于教科书上图片讲解的局限，这一块就学的不好。我在想现在有了AI，但凡学生哪里搞不清，都可以利用图片的方式直观地呈现，甚至可以做成3D视频，或者叫交互式网页。

```
你是一个地理老师，你现在要给学生讲解洋流知识 你现在制作一张此类的知识图图片里面洋流要绘画的特别传神具有冲击力能有效地抓住学生的视线知识点部分简略成关键词就行，主要是绘画风格要出彩
```

7、把题目具象化，以物理楞次定律为例：

很多学生在做数学物理题目的时候，非常痛苦。空间想象力不够好，就很难把模型理解透彻。现在有AI，哪里不懂点哪里。这种具有空间属性的知识点完全可以做成立体的图片，辅助学生理解。

```
提示词1：画一个讲解楞次定律的图，要有立体感，一幕一幕的，讲清楚
```

8、语文作文训练，看漫画写作文：

这个对于小学低年级阶段有用，那个时候孩子还需要看图说话。有时候老师想做一些创新性的看图说话，但是苦于自己不会绘画，并且网上没有很好的资源。那么就能借助AI的力量，做出好玩的看图讲话啦！

```
提示词：构建一个四格漫画，给一年级小朋友用来看图写字的用可爱的吉卜力风格
```

9、美术教育，不同艺术流派风格对比：

对于美术老师，这方面的资源很多。往上各类流派的真作都有相当多。AI只是对于其风格进行简单的模仿，提取出几个关键的刻板的印象加以组合。不过对于美术教育也是一种新的思路，AI如何融入美术教育中？当AI可以快速绘制出较高水平的画作，那么美术教育的审美教育的基石是什么？

```
画一张图，分成三格， 展示三种不同的艺术流派
```

10、做出复杂的学科融合题目，并且构建题目卡片：

关于这一板块，我有两点要强调。

第一我们能看出AI可以产出这种试卷风格的题目，也就是说过去所困扰大家的给题目制图的问题，现在已经能充分解决了。你完全可以丢进去一个题目，要求其用试卷的风格做出对应的解析，甚至可以让他做出学生解题的草稿。

第二我们关注一个词“学科融合题目”。这种题目出题是困难的。这也是新高考的一个方向标。现在可以利用AI做出这种融合体，对学生的思维能力和跨学科结合能力都有一个新的要求。

```
提示词：物理跟生物结合的“融合题目” 你给我做一张图，这个图里面是这个题 这个题是作为试卷的第22题，压轴题，最后一题呈现的 这个图是试卷风格
```

今天所用的AI绘图模型，是4月21日刚更新的GPT-Image-V2模型。

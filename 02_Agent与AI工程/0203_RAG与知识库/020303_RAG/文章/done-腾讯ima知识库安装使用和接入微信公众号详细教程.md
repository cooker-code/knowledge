# **一、先搞懂：IMA 到底是什么？**

不用记复杂定义！它是腾讯开发的「AI 智能工作台」，会思考的知识库，专门解决 “文件乱、搜不到、写得慢” 的痛点，对这几类人尤其有用：

学生党（论文资料整理、考点速查）

打工人（PPT 素材找、会议记录提炼）

自媒体（选题灵感、文案续写）

专业人士（律师案例、医生资料分类）

简单说，它像你的「全能数字助手」，自带 3 大核心能力：

**私人图书****管理员**

自动整理微信文件、论文、会议记录、网页链接，不管存了多少，需要时搜关键词 “秒出结果”，再也不用翻文件夹。

**自带多大模型**

搭载腾讯混元 + DeepSeek V3 满血版，哪怕记不清完整标题（比如只记得 “3 种网络安全技巧”），模糊搜索也能精准定位文档，还能直接让人工智能 帮你 “读懂” 内容 —— 比如提炼论文核心观点、总结会议重点。

**AI****写作外挂**

操作像用美图秀秀 P 图一样简单：划词就能翻译（外文文献秒转中文）、写文案卡壳时点 “续写”、整理资料时一键生成思维导图，省一半时间。

# 四、ima接入微信公众号

1.打开公众号

https://mp.weixin.qq.com/

左边点击广告与服务里面，打开小程序管理。

2.开通并关联小程序

然后点击关联小程序，搜索ima知识库，点击关联。

2.获取小程序链接。

打开ima知识库左侧的【灯泡】按钮，找到自己想接入公众号的共享知识库，比如我选择了【Coze扣子智能体工作流源码】这个知识库。

然后左上角点击分享按钮，点击复制链接。

会得到下面这种链接：

从问号开始至结尾，复制红色的这部分地址备用。

3.接入小程序

点击公众号后台左侧的互动管理，找到自定义菜单，其实就是配置到公众号的菜单栏里。

添加一个新的菜单，然后取好名称，选择跳转到小程序，选择ima知识库小程序，最后把第二步复制红色框里的链接，粘贴到index的后面，其他不用管。

最后保存确定即可。微信公众号导入，目前仅支持单篇导入。

# 五、ima知识库导入公众号及各种资料

#### **1、本地文件上传拖拽：**

直接拖PDF/Word/图片到IMA客户端，AI自动生成摘要

#### **2、公众号文章收藏：**

首次使用需先在微信中搜索并添加“ima知识库”小程序。

步骤一：在微信中打开想要收藏的公众号文章。

步骤二：点击文章右上角的“...”，选择“用小程序工具打开”。

步骤三：从列表中选择“IMA知识库”小程序。

步骤四：在小程序中，点击“一键存入IMA知识库”，即可将文章保存到你的个人知识库。

#### **3、微信聊天文件导入：**

在微信聊天中选择文件或图片（先要下载到手机上），然后长按文件或图片，出现如下界面：

#### **4、Pro技巧：遇到小红书笔记、网页链接？截图或复制链接**

1.在浏览器中打开目标网页，复制网页的完整URL地址。

2. 打开IMA：在电脑或手机上打开IMA客户端或微信小程序

3. 粘贴链接：在IMA的输入框中粘贴复制的网页链接 。

4. 解析内容：IMA会自动解析网页内容，生成摘要和关键信息  。

5. 添加到知识库：点击右上角的“添加到知识库”按钮，选择目标知识库，完成存入。

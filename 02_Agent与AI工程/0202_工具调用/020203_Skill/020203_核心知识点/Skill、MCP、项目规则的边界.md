## 横向对标

| 对标技术 | 实现方式 | 优势 | 劣势 | 适合场景 |
|---|---|---|---|---|
| AGENTS.md / CLAUDE.md | 项目根目录规则文件 | 长期稳定、全局可见 | 常驻上下文成本高，不适合放大量专项流程 | 编码规范、知识整理总规则 |
| Skill | `SKILL.md` + 脚本 + 资源 | 标准化任务复用，动态加载 | 需要维护触发描述、依赖和脚本质量 | 文章整理、数据分析、发布流程 |
| MCP | Client/Server 协议 | 外部系统接入清晰，可复用 | Server 维护、鉴权、审计成本 | 数据库、浏览器、知识库、内部系统 |
| Plugin | 打包分发的扩展 | 生态丰富，安装方便 | 质量、安全和兼容性不稳定 | 非核心能力试用 |

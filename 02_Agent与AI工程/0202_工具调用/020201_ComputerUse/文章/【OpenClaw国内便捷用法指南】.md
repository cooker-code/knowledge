一、OpenClaw简介

OpenClaw是一款开源、轻量、本地运行的AI交互平台，它能让普通电脑在不上传数据、不依赖云端服务器的情况下，直接运行大模型，实现智能问答、文本生成、文件总结、翻译、写作等功能。其核心优势包括完全本地运行保障隐私安全、中文版界面友好、对电脑配置要求低、免费开源无广告以及断网可用等。适合经常处理文字、学习和工作的人群使用。

二、ClawX使用方法

（一）ClawX是什么

ClawX是专为普通人设计的AI智能体可视化桌面客户端，它相当于给OpenClaw装了一个“控制面板”，把复杂操作全部变成点鼠标。用户不用敲命令、不用配环境、不用写配置，像装QQ、微信一样，点点鼠标就能用上真正的AI智能体。它能让用户像聊天软件一样和AI对话，可视化设置自动任务，像装App一样给AI加功能。

（二）安装ClawX

ClawX支持Windows、macOS、Linux三大系统，安装步骤和普通软件一样：

去官网或GitHub Releases页面，下载对应电脑的安装包（.exe / .dmg / AppImage）。

双击安装，一路下一步，全程不用开终端、不用配环境变量、不用敲任何命令。

（三）第一次启动

安装完打开ClawX，会自动弹出设置向导，一步一步带你配置完，不想马上配也可以直接跳过，后面再改。

选择语言与地区：根据自己的需求进行选择。

环境检查：缺少什么组件，软件会直接提示，跟着点安装就行，不用自己找。

添加AI模型供应商：以Aionly AI为例，URL为

https://api.aionly.com/v1，模型ID为gpt-5.4（也可以用最新版），API Key需要去官网申请一个。注意查看token消耗，避免使用超量。

连接消息渠道（以飞书为例） ：想让AI自动发消息给用户，可以绑定飞书、WhatsApp、Telegram、Slack等。以飞书为例，要加上contact:contact.base:readonly权限。

安装必要组件：最后一步，软件会自动检查并安装需要的依赖，点一下就好。

（四）装好之后怎么用

和AI智能体聊天：主界面是一个聊天窗口，跟微信一样，输入问题，AI直接回答，支持多轮对话、历史记录，而且支持Markdown，代码、表格、清单都能正常显示。

定时任务 & 全自动干活：以前OpenClaw要手写复杂的定时表达式，在ClawX里，全部可视化点一点搞定。用户可以可视化创建定时任务，设置多久执行一次，让AI 7×24小时自动跑，比如每天早上自动汇总新闻、每小时监控某个网页更新、自动抓数据发到邮箱、晚上自动帮查行情和资讯等，全程不用写一行代码。

技能中心：像装App一样给AI加功能，ClawX自带一个技能市场，看有哪些技能，点一下就能安装，在界面里直接开启/关闭，不用命令、不用npm、不用手动部署。

（五）实用小技巧

用稳定的API服务商，国内用户可以用国内大模型或稳定中转服务，更流畅。

多个模型搭配用，主模型负责思考 + 小模型负责执行，自动任务效果更好、更省费用。

定时任务别太频繁，频率太高容易花冤枉钱，太低又会错过信息，根据自己需求调整。

三、扣子（Coze）上OpenClaw使用方法

（一）前置准备（1分钟）

注册/登录Coze官网（https://www.coze.cn），完成实名认证。

开通Coze个人高阶版（免费额度足够体验，长期用选付费版）。

准备一个模型API Key（推荐：Aionly，模型充足）。

（二）Coze一键部署OpenClaw（5分钟，零代码）

复制官方OpenClaw项目：进入Coze工作台 → 左侧「编程」→ 顶部「优质案例」，搜索/找到OpenClaw（龙虾）官方模板 → 点击「复制」，填写项目名称（如 “我的OpenClaw”）、描述 → 点击「确定」，等待系统自动构建环境（约1–2分钟），右侧会出现OpenClaw WebUI预览。

配置模型与基础参数（必做） ：在预览页/项目配置页，找到「模型设置」，选择模型厂商（如阿里云百炼、DeepSeek、豆包），粘贴API Key（从对应平台控制台获取），保存配置 → 发送 “你好” 测试，确认能正常回复。

正式部署（24小时在线） ：点击项目右上角「部署」→ 选择「云端部署」，选择部署环境（推荐「标准环境」，免费额度可用），点击「确认部署」，等待部署完成（约1分钟），部署成功后，会生成独立访问链接，可直接在浏览器打开使用。重要提示：不部署仅在预览页运行，沙箱会被回收，无法24小时在线。

（三）IM接入（飞书 / 微信 / 钉钉，手机随时用）

飞书接入（最推荐，国内生态最好） ：Coze项目 → 左侧「渠道」→ 选择「飞书」，点击「创建应用」，跳转飞书开放平台，按指引创建企业内部应用，获取AppID/AppSecret，回Coze粘贴凭证 → 开启「消息接收 / 发送」权限，发布飞书应用 → 审核通过后，即可在飞书私聊 / 群聊调用OpenClaw。

微信 / 钉钉接入：渠道选择「微信公众号 / 小程序」或「钉钉」，按平台指引完成开发者认证、获取密钥，回Coze配置凭证 → 发布并测试，支持关键词触发、菜单交互、自动回复。

（四）OpenClaw核心使用（一句话执行任务）

基础指令（直接发消息即可） ：

系统 / 文件：如“帮我整理桌面文件”、“清理C盘缓存”、“批量重命名下载文件夹”。

浏览器：如“打开百度搜索AI工具”、“监控某商品价格”、“自动填写表单”。

办公：如“生成本周周报”、“整理会议纪要”、“发送邮件给团队”。

定时任务：如“每天早上8点推送天气”、“每周五自动备份文件”。

技能商店（10000 + 现成能力） ：Coze项目 → 左侧「技能」→ 进入「技能商店」，搜索分类（办公、浏览器、系统、数据等），找到需要的技能 → 点击「添加到项目」，直接在对话中调用，无需编写代码。

自定义技能（零代码创建） ：技能页 → 「创建技能」→ 选择「自然语言创建」，描述需求（如 “自动爬取新闻并生成摘要”），Coze自动生成技能配置 → 测试 → 保存，一键部署到对话 / IM使用。

（五）进阶配置（提升体验）

权限与安全（必看） ：Coze上的OpenClaw运行在云端沙箱，无本地文件操作权限，安全可控；API Key加密存储，不会泄露；对话记录可在项目「日志」中查看，支持导出 / 删除。

模型切换（免费额度不够时） ：项目配置 → 「模型设置」→ 更换模型厂商，粘贴新的API Key → 保存即可，推荐组合：豆包（免费）+ 阿里云百炼（500万token / 月免费）。

工作流编排（复杂任务） ：Coze「工作流」→ 拖拽式编排，串联多个技能 / 模型 / 触发条件，生成自动化流程，支持定时 / 事件触发。

四、Cherry Studio配置OpenClaw使用方法

（一）配置OpenClaw（利用OpenClaw的本地服务）

OpenClaw安装初始化后，会自动启动本地服务（默认端口18789），在Cherry Studio中选择「自定义模型」，名称填写「OpenClaw - 本地」，API Base URL填写http://127.0.0.1:18790/api/v1/，无需额外填写API Key（OpenClaw已完成本地授权），保存后即可调用OpenClaw的AI能力，无需重复配置，操作极为便捷。

（二）配置模型参数

调整温度值（0.1 - 2.0，数值越低，输出越严谨；数值越高，输出越灵活）、最大输出长度（建议设置2048 - 4096，满足大多数编程需求）。通过「模型广场」加载专业模型（如DeepSeek - R1），支持同时激活8个服务商，按需选择即可。配置好OpenRouter和OpenClaw后，可在对话界面通过@符号快速切换这两个模型，无需重新进入设置，进一步提升使用便捷性，兼顾远程模型（OpenRouter）和本地模型（OpenClaw）的优势。

（一）为什么CoPaw更适合普通人

表格

（二）CoPaw上手指南

选择部署方式：

本地部署（三行命令，数据完全在自己电脑） ：前提是电脑有python3.10~3.13版本（没有的话先去python.org装一个），打开终端（mac叫终端，windows叫命令提示符或powershell），依次输入pip install copaw（安装copaw）、copaw init --defaults（初始化工作区）、copaw app（启动服务），看到启动成功的提示后，打开浏览器，访问http://127.0.0.1:8088，就看到了copaw的控制台界面。

云端一键部署（无需安装，有浏览器就行） ：点击魔搭创空间copaw一键部署链接（可能需要注册登录），点击“一键配置”（fork），等待几十秒。部署成功后，进入空间管理页面，务必在“权限”设置中设为“私有”，确保只有自己能访问，访问系统分配的云端网址，copaw就跑在浏览器里了。

给CoPaw装上“大脑” ：在copaw控制台找到“models”设置，填入拥有调用权限的模型提供商（如openai、阿里通义、googlegemini等）的apikey和对应的模型名称，社区网友推荐gemini3.0flash，性价比很高，响应也快。

定制专属AI：首次对话时，它会引导创建一个profile.md文件，用户可以在这里定义AI的名字、物种、性格以及双方的关系等，从此，它会记住用户的偏好，用设定的风格和用户交流。

让它干活：可以给copaw下指令，如“清理桌面超过3天的文件”、“每天下午5点提醒我运动”、“创建一个新skill，监控我的股票池，跌超5%就通知我”等，它不仅能执行任务，还能自己创造新能力。

（三）进阶技巧

接入多个聊天软件：在copaw控制台里，可以配置多个“频道”（钉钉、飞书、qq等），这样无论在哪个app里@它，它都能响应。

查看长期记忆：在控制台里有一个“记忆”面板，可以看到copaw从对话中提取并存储的信息（如生日、讨厌的食物），也可以手动编辑或删除这些记忆。

# MCP Apps：为 Agent 加入可交互的用户界面

近日，MCP 社区正式提出 **MCP Apps Extension**，为 MCP 引入交互式用户界面支持，旨在解决当前 AI 工具在复杂交互场景中的局限性。

目前，MCP 都是返回文本数据，需要 AI 解读整理给用户，但想象以下场景：

* • **数据可视化：** MCP 返回数据，AI 客户端解析为可视化图表。
* • **复杂输入收集**：比如让用户选择多个参数、上传文件等。
* • **专业工具集成**：如数据库查询器、代码调试器、电商配置面板等。

这些场景如果有可交互的界面，而不是仅仅文本问答，体验上会好很多，成本也会降低很多。

1. dify

dify是一个开源的 LLM 应用开发平台，提供从 Agent 构建到 AI workflow 编排、RAG 检索、模型管理等能力，轻松构建和运营 AI 应用

地址

https://github.com/langgenius/dify

2. coze-studio

开源版，扣子是新一代 AI 大模型智能体开发平台。整合了插件、长短期记忆、工作流、卡片等丰富能力，扣子能帮你低门槛、快速搭建个性化或具备商业价值的智能体，并发布到豆包、飞书等各个平台。

地址

**https://github.com/coze-dev/coze-studio**

3. joy-agent

**JoyDataAgent：首个开源的包含数据治理DGP协议、智能问数和智能诊断分析的智能体**

地址

https://github.com/jd-opensource/joyagent-jdgenie

4. lemonai

Lemon AI 是首个全栈、开源、基于代理的 AI 框架，为 Manus & Genspark AI 等平台提供了一个完全本地化的替代方案。它具备集成的代码解释器虚拟机沙箱，以实现安全执行。

地址

https://github.com/hexdocom/lemonai

5. dify-plus

dify二开版本，增加了企业内部使用的功能，以及python的执行代码放开。

地址

https://github.com/YFGaia/dify-plus

6. crewai

CrewAI 是一个 AI 智能体协作框架，就像组建一个 AI 团队。你可以给每个 AI 分配不同职位（如研究员、分析师），它们会像真实团队一样分工合作完成任务。适合自动化客服、数据分析、市场调研等场景。

地址

https://github.com/crewAIInc/crewAI

7. langgraph

LangGraph 是一个用于构建、管理和部署长期运行、有状态代理的低级编排框架。

地址

https://github.com/langchain-ai/langgraph

8. fastgpt

FastGPT 是一个基于 LLM 大语言模型的知识库问答系统，提供开箱即用的数据处理、模型调用等能力。同时可以通过 Flow 可视化进行工作流编排，从而实现复杂的问答场景！

地址

https://github.com/labring/FastGPT

8. n8n

n8n 是一个工作流自动化平台，它为技术团队提供了代码的灵活性和无代码的速度。n8n 拥有 400 多个集成、原生 AI 功能和公平代码许可证，可让您构建强大的自动化功能，同时完全控制您的数据和部署。

地址

https://github.com/n8n-io/n8n

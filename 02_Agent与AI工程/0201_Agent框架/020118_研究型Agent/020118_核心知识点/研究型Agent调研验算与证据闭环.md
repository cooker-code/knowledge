## 核心问题
研究型 Agent 的价值在于把调研、检索、推理、验算和报告拆成可追踪链路；如果没有证据锚点和验算节点，只是长文本生成。

## 判断准则
- 每个结论都要能追到来源、推断过程和待验证缺口。
- 研究任务要区分事实检索、趋势判断、假设生成和结论表达。
- 验算节点要独立于生成节点，避免同一模型自证。

## 认知偏差
| 常见错误认知 | 正确理解 |
|---|---|
| 会写长报告就是研究 Agent | 研究 Agent 必须有证据、假设和验证闭环 |
| 研究型 Agent 越自主越好 | 研究边界、来源质量和最终用途要人工确认 |

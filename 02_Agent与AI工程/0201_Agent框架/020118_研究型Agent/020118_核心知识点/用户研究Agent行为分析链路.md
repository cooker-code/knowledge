## 核心问题
用户研究 Agent 不是把访谈或行为日志交给模型总结，而是把行为事件、用户意图、分群、转化路径和可解释证据串起来。

## 判断准则
- 行为分析要先定义事件、漏斗、分群和假设，再让 Agent 辅助发现线索。
- 大模型适合生成解释和假设，不适合无证据地替代统计判断。
- 输出要区分可证实行为、推断意图和后续实验建议。

## 认知偏差
| 常见错误认知 | 正确理解 |
|---|---|
| 用户意图可以直接由模型判断 | 意图推断必须回到行为证据和业务定义 |
| 用户研究 Agent 只属于分析工具 | 它同时是研究型 Agent 和 DataAgent 的交叉场景 |

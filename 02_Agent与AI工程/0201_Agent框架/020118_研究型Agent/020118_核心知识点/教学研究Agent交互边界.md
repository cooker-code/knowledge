## 核心问题
教学研究类 Agent 的重点是问答、讲解、出题、可视化和学习路径的闭环，不能只看“导师”人设。

## 判断准则
- 教学 Agent 要有知识点图谱、题目生成、反馈和错因归纳。
- 研究/教学混合场景要区分事实讲解和开放探索。
- 学习效果需要题目、作答、反馈和复盘数据验证。

## 认知偏差
| 常见错误认知 | 正确理解 |
|---|---|
| AI 导师能讲清楚就够了 | 教学 Agent 还要能检测理解、生成练习和反馈 |
| 开源教育项目都值得深入 | 缺学习效果证据时只保留架构线索 |

## 核心问题
企业级 Agent 的分水岭不是 demo 效果，而是权限、审计、评估、失败恢复、成本和组织流程是否能被治理。

## 判断准则
- 上线前必须定义工具权限、审批、日志、回滚和责任人。
- 评估要覆盖成功率、错误类型、人工接管率、成本和延迟。
- 平台能力优先服务重复高价值流程，不为一次性热闹场景建平台。

## 认知偏差
| 常见错误认知 | 正确理解 |
|---|---|
| 能调用工具就是生产 Agent | 生产 Agent 需要治理闭环和可追责证据 |
| 企业平台越全越好 | 平台要从高频流程和失败闭环开始 |

## 核心问题
表格、架构图和 Excel 插件类 Agent 的关键是把自然语言需求转换成可验证结构：公式、数据范围、图表、组件和导出格式。

## 判断准则
- 表格 Agent 要验证公式、引用范围和权限，不只看生成文本。
- 架构图 Agent 要把节点、关系和层级显式化，避免只生成装饰图。
- Excel 插件类能力要区分本地文件权限、云模型调用和企业数据安全。

## 认知偏差
| 常见错误认知 | 正确理解 |
|---|---|
| 图表/表格能生成就是完成 | 还要验证数据、公式、引用和结构语义 |
| 架构图神器可以替代架构理解 | 图只是表达，关键是关系和边界是否正确 |

## 核心问题
画布/视频 Agent 的增量在于把创作过程拆成可编辑节点：脚本、大纲、镜头、素材、时间轴和生成结果都能被用户接管。

## 判断准则
- 过程透明优先于黑盒一键生成，特别是视频、设计和复杂视觉产物。
- 每一步要能回退、替换素材和局部重生成。
- 评价视频 Agent 要看时间轴一致性、素材可控性和编辑成本。

## 认知偏差
| 常见错误认知 | 正确理解 |
|---|---|
| 视频 Agent 输出好看就够了 | 生产可用还要看可编辑、可回退和可复用 |
| 画布只是 UI 创新 | 画布是把任务状态和素材关系显性化 |

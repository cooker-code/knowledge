## 核心问题
交付型 Agent 的核心是把需求、规划、素材、生成、编辑、验收和导出组织成产物流水线，而不是一次性生成漂亮结果。

## 判断准则
- 交付物要先定义结构、风格、约束和验收标准。
- 生成过程要保留中间产物，便于修改局部而不是重来。
- 前端/PPT/文档类 Agent 都要把内容正确性和视觉质量分开验收。

## 认知偏差
| 常见错误认知 | 正确理解 |
|---|---|
| 一次生成成品就是好 Agent | 可编辑、可复用、可验收的流水线更重要 |
| 交付型 Agent 只是内容生成 | 它还包含素材管理、布局约束和导出链路 |

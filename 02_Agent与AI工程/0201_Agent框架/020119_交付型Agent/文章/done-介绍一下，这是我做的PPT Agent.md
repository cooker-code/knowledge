但，绝不只是设计很强。它和其他的AI PPT工具有什么区别？

1.做PPT，第一步绝对不是上手就干。

市面上所有的AI PPT工具，输好主题都是潦草的给你一个大纲，恨不得跳过一切，马上把设计效果放到你眼前。

但，做PPT不是这样的，内容才是PPT的根。

现在，用SANDUN，输好主题，AI会做需求调研。先大致搜索资料，询问你想在PPT呈现的核心内容。

然后会根据调研和你输入的需求，生成大纲部分。

怎么生成PPT大纲？以前我给大家分享过我自己在用的便利贴形式，我会拿一张张便利贴去规划每页要放的内容，它能帮你建立一个很清晰的内容架构，而且哪里不想要就撕掉，顺序也可以自由调。

那这次，三顿花了大量的功夫，把这套模式搬到了大家的面前：

每一个页面就是一张便利贴，看得清、好调整。

那有大纲之后，是不是得查资料填内容？这事儿我们也让AI帮你干了，每个页面都会进行多轮次的搜索，并对资料进行梳理。

这一整个流程，自动就能跑，帮你把你PPT的“内容力”拉满，老板绝对看不出，这PPT，居然是5分钟做出来的？

2.PPT居然还有策划稿？

啥，PPT还要做策划呢？我想这个词很多PPT设计师都没听过，五年前，我也一样。

那时候我去到了锐普，目前最顶尖的PPT设计公司，他们的PPT报价是1万+一页。

他们有一个专门的岗位，叫策划师。没错，你前面看到的，需求调研、资料检索，大纲规划，这事儿都归他们干。

他们最终会提供一个PPT草稿给到设计师，每页什么位置要放什么元素，用什么样的版式，全都固定好。

我们做了大量测试，这套流程，AI真的能懂。策划部分负责版面规划，设计部分来做风格样式，跑下来的效果非常好。

你啥都不用干，等AI跑完这套流程就能用，甚至都不太需要改。

那当然，如果是一些特别重要的PPT，你也可以在策划稿阶段精调内容，再去让AI跑最终的设计，把效率和颜值都最大化。

3.我们真的把PPT技巧教给了AI。

三顿的PPT课里，有一个可以给到“夯爆了”的PPT技巧，叫卡片式布局。

啥是卡片式布局呢，喏，你在苹果的发布会上经常能看到。把内容放进了一个个卡片里。

这样做的好处有两个，一是可以大量承载内容，二是可以自由调整卡片的大小、数量，让页面产生非常多的变化。

那现在，有了第三个好处，这是一种AI非常好懂的设计语言。

直接给你再放一些，用SANDUN这个PPT Agent自动跑出来的案例：

你会发现页面的构成其实都是一个个卡片，卡片可以放置图形图示、可以大小不一。

合理布局，又有无限的可能。

除此之外，该用什么样的字体，怎么放图片，怎么放图表，我把这些技巧也都教给了AI。

结合这种卡片式布局，除了这些已经实现的外，我们还规划了非常多的呈现能力，会一点点的完善给大家。

4.不仅能编辑，还能听懂你的话。

现在AI PPT会有很多的流派，有的是套模版，有的是生成HTML网页，有的是用Banana直接生成。

应该没有傻子想让AI生成整页的SVG吧？有的，有的。

SVG是Office里原生支持的一种图形格式，它可以无限放大保持高清，更可以直接拖进PPT里随意编辑。

就是为这个可编辑性，三顿我真的快熬秃了。

## 核心问题
Agent 协议首先解决“谁能接谁、怎样接、权限和上下文如何传递”的连接层问题，而不是直接提升模型能力。

## 判断准则
- 判断协议价值先看连接对象：客户端、工具、服务、Agent 还是运行时。
- 协议必须明确能力发现、认证授权、上下文边界和错误语义。
- 生态新闻只能作为线索，不能替代真实实现和采用证据。

## 认知偏差
| 常见错误认知 | 正确理解 |
|---|---|
| 有协议就能协作 | 协议只提供接线方式，质量取决于能力描述、权限和运行时治理 |
| 公司支持数越多越成熟 | 真实采用、互操作样例和安全模型更重要 |

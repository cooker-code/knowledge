## 横向对标

| 对标技术 | 实现方式 | 优势 | 劣势 | 适合场景 |
|---|---|---|---|---|
| LangGraph | 显式图状态和控制流 | 控制流细粒度强 | 沙箱隔离需另配 | 复杂工作流编排 |
| OpenAI Agents SDK | 控制面/执行面、沙箱、记忆、恢复 | 更强调生产运行时 | 平台绑定和版本待补证 | 安全执行和长任务 |

| 主机直接执行 | Agent 直接运行代码 | 简单快速 | 凭证和主机风险高 | 只适合低风险本地实验 |

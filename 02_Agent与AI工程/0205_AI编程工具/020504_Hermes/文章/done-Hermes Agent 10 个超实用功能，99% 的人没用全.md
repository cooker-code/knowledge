很多人装好 Hermes Agent 之后，只用它来"聊天"——这简直是暴殄天物。

本篇介绍 10 个大多数人不知道的实用功能，让你把 Hermes 的潜力发挥出来。

---

## 功能一：长期记忆 — 越用越懂你

**这是 Hermes 最与众不同的地方。**

普通 AI 每次对话从零开始，什么都不记得。Hermes 不同——它会记住你的：

* 工作习惯和偏好
* 常用的表达方式
* 历史上下文

比如你告诉它「我喜欢简洁的回答风格，不要废话」，之后的所有对话它都会记住这个偏好，不用每次重复交代。

**怎么激活：** 直接告诉它你的偏好，它会自动记录。比如：

```
你：记住，我的项目名字叫"天启计划"，以后提到项目直接用这个名字
Hermes：好的，已记录。
```

---

## 功能二：定时任务 — 真正解放双手

**用自然语言创建定时任务，无需任何 cron 知识。**

直接说：

```
你：每天早上9点，帮我搜索昨天的AI行业新闻，整理成3条要点，通过微信发给我
```

Hermes 就会设置好定时任务，每天自动执行。

其他例子：

```
每周一早上给我发本周待办清单
每天下午6点提醒我喝水
每月1号帮我生成上月工作总结
```

查看和管理定时任务：

```
hermes cron list    # 查看所有定时任务
hermes cron delete  # 删除任务
```

---

## 功能三：恢复上次对话 — 不怕中断

有时候对话到一半，关掉了终端，下次打开又要重头来过？

**不用担心，Hermes 可以恢复会话：**

```
hermes --continue   # 或简写 hermes -c
```

继续上次的对话，上下文完整保留。

也可以给会话起名字，方便以后找：

```
hermes -c "我的项目讨论"   # 按名字恢复
```

## 功能四：真正执行任务，不只是"建议"

这是很多人没意识到的：**Hermes 可以直接帮你干活，而不只是告诉你怎么干。**

```
你：帮我找一下桌面上所有超过100MB的文件
Hermes：[自动执行文件搜索命令]
找到以下文件：
- ~/Desktop/video_draft.mp4 (2.3GB)
- ~/Desktop/backup_old.zip (456MB)
```

```
你：帮我把 downloads 文件夹里所有的 .txt 文件合并成一个文件
Hermes：[自动执行] 已将15个txt文件合并为 combined.txt
```

你不需要自己去找命令，它直接帮你做。

## 功能五：多平台同时在线

**一个 Hermes 实例，可以同时接入多个平台。**

比如：

* 微信里能用
* 飞书里也能用
* Telegram 里也能用

而且是**同一个 AI，共享同一份记忆**——你在微信里告诉它的偏好，在飞书里也生效。

配置：

```
hermes gateway setup   # 可以多次运行，每次添加一个平台
```

---

## 功能六：接入 Telegram / Discord — 更多平台选择

除了微信和飞书，还支持：

* **Telegram**：在国内需要代理，但功能最全，支持语音消息、文件传输
* **Discord**：适合游戏圈、技术社区
* **Slack**：企业协作利器
* **钉钉**：国内企业用户的选择

配置方式都类似，运行 `hermes gateway setup`，选对应平台按引导操作即可。

## 功能七：语音模式 — 说话就能用

**开启语音输入，对着 AI 说话，它帮你做事。**

先安装语音扩展：

```
uv pip install -e ".[voice]"
```

然后：

```
hermes --voice
```

开启后，你说的话会被实时转文字，AI 处理后可以语音回复你。非常适合手头忙着做事、没空打字的场景。

## 功能八：处理文件和图片

**把文件路径或图片丢给 Hermes，让它直接分析。**

```
你：帮我总结一下 /Documents/会议记录.txt 的要点
Hermes：[读取文件] 这份会议记录主要讨论了以下三点...
```

```
你：[上传图片] 这张表格里的数据帮我分析一下趋势
Hermes：从图表来看，销售额在Q3明显上升，原因可能是...
```

---

## 功能九：MCP 工具扩展 — 无限可能

**MCP（模型上下文协议）是一个开放标准，允许你给 Hermes 安装各种第三方工具。**

比如：

* 接入你的 Notion，让 AI 帮你管理笔记
* 接入 GitHub，让 AI 帮你管理代码仓库
* 接入 Google Calendar，让 AI 帮你管理日程

配置 MCP：

```
hermes mcp add   # 按提示添加 MCP 服务器
```

详细文档：https://hermesai.top/docs.html

---

## 功能十：多 Agent 并行 — 复杂任务拆分执行

**遇到复杂任务，Hermes 可以自动创建多个子 Agent 并行处理。**

比如：

```
你：帮我研究一下2025年AI行业的发展趋势，
    从技术进展、商业应用、政策监管三个维度分别分析，
    最后整合成一份报告
```

Hermes 会自动把这个任务拆分成三个子任务，并行搜索和分析，最后汇总——速度比串行快得多。

## 快速命令参考

在对话中可以输入斜杠命令：

| 命令 | 功能 |
| --- | --- |
| `/model` | 切换 AI 模型 |

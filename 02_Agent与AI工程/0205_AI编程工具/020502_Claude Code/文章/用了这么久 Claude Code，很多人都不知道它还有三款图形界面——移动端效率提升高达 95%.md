## Claude Code 到底是什么？

说了这么多界面，先搞清楚 Claude Code 本身是什么。

Claude Code 是 Anthropic 开发的 AI 编程 Agent，跟 GitHub Copilot 那类补全工具有本质区别：

* • **Copilot 的逻辑**：帮你写下一行代码
* • **Claude Code 的逻辑**：给它一个任务，它自己去读文件、改代码、跑测试、调试报错

它能做的事包括：读写项目文件、执行终端命令、跨文件重构、理解代码库依赖、自主调试报错、提交 Git 变更……更像一个能独立干活的同事，而不是一个智能输入法。

**官方订阅方式和价格：**

Claude Code 需要 Anthropic 的 API 或 Claude 订阅才能使用。官方提供 Pro（月）和（100/月）两档订阅，也支持直接使用 API 按量计费。

不过国内用户订阅官方有两个麻烦：需要海外信用卡，网络环境也得折腾。如果嫌麻烦想找个更省事的渠道，可以看看 Code80，真实订阅帐号转 API，换个 endpoint 就能直接用，体验跟官方一样。详情可以到官网了解：code.ai80.vip

## 常见问题

**Q：三种方式哪个最适合新手？**

Claudia GUI。安装完成后所有操作都在图形界面里完成，不需要记命令。而且功能最完整，后续深度使用不需要换工具。

**Q：Claudia GUI 要收费吗？**

不收费。AGPL 许可证下完全开源，自己部署到本地就能用，没有任何隐藏费用。

**Q：用 GUI 会损失功能吗？**

不会。Claudia GUI 是在 Claude Code CLI 上套了一层可视化界面，底层调用的还是 Claude Code 的全部能力，没有阉割。

**Q：对电脑配置有要求吗？**

配置要求不高，本地跑个 Node.js 服务就够了。主要是网络——Claude Code 需要联网调用 AI 模型，部分界面提供本地缓存，但核心功能需要保持连接。

**Q：代码数据安全吗？**

有沙盒执行机制，所有文件修改都需要你手动确认。另外建议在项目里配置好 `.claudeignore`，把不该让 AI 读的文件（如 `.env`、密钥文件）排除掉。

**Q：国内用户怎么用起来？**

官方订阅需要海外支付环境，国内用户可以通过 Code80 更方便地使用。

## 二、核心亮点详解：每一个功能都戳中开发者痛点，用一次就爱上

CraftAgents 的强大，不在于“花里胡哨的设计”，而在于每一个功能都精准解决了原生 ClaudeCode 的使用痛点，尤其是对于经常用 AI 辅助编程、多任务并行的开发者来说，每一个功能都能节省大量时间，提升效率。

### 1. 收件箱式多会话管理：告别多终端切换，任务管理更清晰

原生 ClaudeCode 用终端开启多个会话时，需要打开多个终端窗口，切换起来十分繁琐，时间久了很容易混淆不同任务的会话内容，甚至不小心关掉某个窗口，导致会话记录丢失。

而 CraftAgents 采用了 **“收件箱式”多会话管理模式**，所有会话都集中在一个窗口内，就像管理邮件一样管理你的 AI 任务，彻底告别多终端切换的麻烦。你可以给每个会话添加明确的标题，标记任务状态——**Todo（待处理）、InProgress（进行中）、Done（已完成）**，还支持旗标标记重要会话、归档已完成会话，后续查找、复盘时一目了然。

更贴心的是，会话支持**搜索功能**，输入关键词就能快速找到之前的对话记录、调试思路，再也不用翻遍多个终端窗口找内容，大大提升了工作效率。

### 2. 数据源连接零配置：一句话搞定，不用手动改一行 JSON

对于很多开发者来说，原生 ClaudeCode 最让人头疼的，就是数据源连接和 MCP 配置——需要手动编辑 JSON 文件，配置服务器地址、认证信息，稍有不慎就会配置失败，甚至导致无法正常使用，尤其是对于不熟悉 JSON 格式的新手来说，更是难上加难。

CraftAgents **彻底解决了这个痛点**，实现了 **“零配置”数据源连接**。你只需要在对话框中输入一句话，比如 `“连接我的 GitHub 账号”``“对接 Slack 服务”`，Agent 就会自动查找 MCP 服务器、配置认证信息，**全程无需手动编辑任何 JSON 文件**，哪怕是新手也能轻松完成配置。

除此之外，它还支持**粘贴 OpenAPI 文档、截图自动适配**，比如你复制了某个 API 的文档，粘贴到 CraftAgents 中，它就能自动识别文档内容，完成数据源对接，省去了手动输入、配置的繁琐步骤。

### 3. 三种权限一键切换：兼顾效率与安全，适配不同使用场景

在使用 AI 辅助编程或处理任务时，我们常常会面临“效率”和“安全”的平衡——既想让 AI 自动执行任务，提升效率，又担心 AI 误操作，导致数据丢失或错误。CraftAgents 针对这一需求，设计了**三种权限模式**，支持快捷键一键切换，适配不同使用场景。

* **Explore（只读模式）**：仅允许 AI 查看数据、提供建议，不进行任何修改操作，适合需要查看数据、分析问题，但不需要修改内容的场景，安全无风险；
* **Ask to Edit（确认修改模式）**：AI 每次执行修改操作前，都会提前向你确认，只有你同意后才会执行，兼顾效率和稳妥，适合日常编程、数据处理等场景；
* **Auto（全自动模式）**：AI 无需确认，直接自动执行所有任务，效率拉满，适合已经明确任务需求、无需手动干预的场景，比如批量处理数据、重复执行简单指令。

更便捷的是，你可以通过快捷键 **`SHIFT+TAB`** 快速切换三种权限模式，不用繁琐操作，随时随地根据需求调整，兼顾效率与安全。

### 4. 技能系统更易用：`@` 直接调用，老用户一键迁移配置

原生 ClaudeCode 的 Skills 功能虽然强大，但需要通过文件夹管理，调用时不够便捷，而且更换设备或重新安装后，需要重新配置 Skills，十分麻烦。

CraftAgents 优化了技能系统，保留了 ClaudeCode Skills 的核心能力，同时简化了调用方式——你只需要在对话框中输入 **`@`** ，就能快速调出所有技能，点击即可调用，无需手动查找文件夹，大大提升了使用效率。

对于已经使用过 ClaudeCode 的老用户来说，更是十分友好：你只需要在 CraftAgents 中说一句 **“从 ClaudeCode 导入所有 Skills”** ，就能一键迁移之前的所有技能配置，不用重新手动设置，无缝衔接，节省大量时间。

### 5. 完整原生体验：流式响应 + 会话持久化，使用更流畅

CraftAgents 虽然是第三方客户端，但完全基于 Anthropic 官方 SDK 构建，保留了 ClaudeCode 的所有原生体验，同时做了进一步优化：

* **流式响应**：AI 回复实时滚动显示，不用等待完整回复，节省等待时间，尤其是在处理复杂任务、AI 回复较长时，体验更流畅；
* **工具调用可视化**：AI 调用工具的过程清晰可见，你可以实时查看 AI 正在执行的操作，了解任务进度，遇到问题也能及时中断、调整；
* **会话自动保存**：无论你是关闭客户端、重启电脑，还是不小心关掉会话窗口，所有会话记录都会自动保存，重新打开客户端就能恢复之前的对话，再也不用担心会话丢失，调试思路、任务进度不会中断。

## 四、一键安装指南：Windows/macOS/Linux 全支持，新手也能轻松搞定

CraftAgents 支持全平台安装，而且安装过程十分简单，无需复杂配置，无论是 Windows、macOS 还是 Linux 系统，都能一键完成安装，具体安装方法如下，大家可以根据自己的系统选择对应的方式：

### 1. Windows 系统（推荐 PowerShell 一行命令，最快最便捷）

打开 Windows 自带的 PowerShell（按下 `Win+R`，输入 `powershell`，回车即可打开），复制下面的命令，粘贴到 PowerShell 中，回车执行，等待片刻即可自动完成安装，全程无需手动操作。

```
irm https://agents.craft.do/install-app.ps1 | iex
```

如果觉得命令安装不够放心，也可以选择直接下载安装包，双击安装即可：
**Windows 安装包下载地址**：https://agents.craft.do/electron/latest/Craft-Agent-x64.exe

### 2. macOS 系统（ARM64 / Intel 通用）

macOS 用户可以通过终端一行命令安装，打开终端（启动台 → 其他 → 终端），复制下面的命令，粘贴执行，等待安装完成即可：

```
curl -fsSL https://agents.craft.do/install-app.sh | bash
```

也可以直接下载对应版本的安装包，双击 `.dmg` 文件，将 CraftAgents 拖入应用程序文件夹即可完成安装：

* **macOS ARM64 安装包**（适用 M1/M2/M3 芯片）：https://agents.craft.do/electron/latest/Craft-Agent-arm64.dmg
* **macOS Intel 安装包**（适用老款 Intel 芯片）：https://agents.craft.do/electron/latest/Craft-Agent-x64.dmg

### 3. Linux 系统

Linux 用户同样可以通过终端一行命令安装，打开终端，复制下面的命令，粘贴执行即可：

```
curl -fsSL https://agents.craft.do/install-app.sh | bash
```

也可以下载 AppImage 安装包，赋予执行权限后直接运行，无需安装：
**Linux 安装包下载地址**：https://agents.craft.do/electron/latest/Craft-Agent-x64.AppImage

### 4. 开发者专属：编译运行（适合需要二次开发、魔改的用户）

如果是开发者，想要对 CraftAgents 进行二次开发、魔改，或者查看源码，可以通过以下步骤编译运行：

```
git clone https://github.com/lukilabs/craft-agents-oss.git
cd craft-agents-oss
bun install
bun run electron:start
```

CraftAgents 的技术栈十分现代：**Bun + Electron + React + shadcn/ui**，代码结构清晰，易于二次开发，无论是个人魔改还是团队定制，都十分便捷。而且它采用 **Apache 2.0 开源协议**，允许自由商用、修改，无需担心版权问题。

## 五、上手即用：3 分钟快速配置指南，新手也能轻松玩转

安装完成后，无需复杂配置，按照以下步骤操作，3 分钟就能上手使用 CraftAgents，快速对接 ClaudeCode，开启高效办公模式：

1. 启动 CraftAgents 客户端，首次启动会提示选择连接方式，推荐选择 **API Key 方式**（最稳定、最便捷），输入你的 Anthropic API Key，点击连接即可；
2. 除了对接 Anthropic 官方服务，它还支持对接 **Ollama、Siliconflow** 等兼容 Anthropic 格式的服务，如果你常用这些服务，也可以在设置中进行配置；
3. 连接成功后，创建一个 **Workspace（工作空间）**，给工作空间命名（比如“前端编程”“数据处理”），方便分类管理任务；
4. 进入工作空间后，就可以直接和 AI 对话、调用工具了，日常使用推荐选择 **Ask to Edit** 权限模式，既保证效率，又能避免误操作；如果需要批量处理任务，再切换到 **Auto** 模式；

6. **会话管理**：创建会话时，给会话添加明确的标题，完成任务后标记为 `Done` 并归档，后续查找时通过搜索功能快速定位，提升管理效率。

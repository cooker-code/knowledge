/powerup

内置交互式入门课程，分 10 关教学核心操作，快速掌握代码对话、撤销、后台任务、远程控制等基础技巧，无需额外配置。

/btw

临时并行提问，不中断当前任务、不污染对话上下文，适合编码时临时确认细节、查资料，答完自动回归原任务。

/rewind

多步选择性回退，可单独恢复代码 / 对话、两者同步恢复，或仅总结历史上下文，测试新方案出错时高效回滚，无需重新沟通需求。

/insights

生成个人使用分析报告，输出近 1 个月会话模式、高频命令、使用问题，给出工作流优化建议，复盘优化使用习惯。

/model opusplan

Pro 用户专属智能模型切换，复杂推理自动调用 Opus，执行阶段切换 Sonnet，平衡推理效果与成本、避免 Opus 额度超限。

/simplify

三合一内置代码审查，同步启动 3 个 Agent，从代码复用、质量、效率三维度优化代码，一键完成专业级代码评审。

/branch（原名 /fork）

会话平行分叉，基于当前对话新建独立会话，适合探索不同实现方案、测试风险功能，不影响主会话。

/loop

定时循环执行任务，设置周期自动重复执行指定操作，适合定期巡检、数据同步、日志核查，无需手动重复操作。

/rc（/remote-control）

远程操控会话，生成分享链接，手机 / 其他设备远程控制当前任务，外出也能管理编码会话。

/export

对话一键导出，将完整会话记录转为 Markdown 格式，用于项目复盘、知识沉淀、二次整理复用。

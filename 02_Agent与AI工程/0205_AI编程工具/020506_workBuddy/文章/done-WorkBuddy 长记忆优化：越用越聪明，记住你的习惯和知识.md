## **一、为什么你的 WorkBuddy 像金鱼？（默认痛点）**

默认情况下，WorkBuddy 只有**短期会话记忆**：

* 只记住最近几轮对话
* 换个聊天窗口 → 失忆
* 第二天打开 → 一切重来
* 你要反复自我介绍、重复要求格式、风格、偏好

**本质原因**：

> 短期记忆 ≠ 长记忆。 短期记忆：放在上下文窗口里，一刷新就没。 长记忆：持久化保存、跨会话读取、自动摘要、动态更新。

你是否也受够了每次都要重新介绍自己？

---

## **二、WorkBuddy 长记忆原理（通俗讲）**

它采用**分层记忆架构**，你不用懂技术，知道能干嘛就行：

一句话总结：

> 短期记对话，长期记你，IMA 记知识。

---

## **三、第一步：开启长记忆（30 秒搞定）**

### **1. 打开记忆功能**

左下角点头像 → **WorkBuddy 设置** → **记忆** - 开启：`生成对话记忆`

开启后，它会自动： - 记录你的身份、行业、常用工具 - 记住你喜欢的文案风格、语气、交付格式 - 每天自动更新记忆摘要

### **2. 给它"写档案"（关键！越写越懂你）**

在记忆设置里点 **编辑记忆**，直接粘贴下面模板（改一改就是你的专属档案）：

```
【身份】福序智研（福哥），银行客户经理/Java+AI开发/马拉松爱好者
【常用工具】WorkBuddy、IMA、Coze、QClaw、IDEA、Vue3、SpringBoot等
【写作风格】口语化、短狠直接、干货密集、少废话、步骤清晰
【输出要求】标题醒目、分点清晰、关键加粗、800–1200字
【禁忌】不要学术化、不要TensorFlow这类术语、不要空话
【常用指令】写公众号、做智能体、RAG知识库、复盘、日报
```

保存后，每次对话它都会**先读你的档案**，再也不用反复说明偏好。

---

## **四、第二步：优化记忆质量（让它记对、记准、不混乱）**

### **1. 让它"自动总结"你的习惯**

每次聊完重要话题，发一句：

```
总结本次对话：提炼我的核心需求、偏好、格式要求、禁止事项，存入长期记忆。
```

它会自动把你习惯固化成长记忆，**越用越精准**。

### **2. 高频信息直接"硬写入"**

你可以直接命令：

```
记住：我写公众号封面要 900×383、浅色系、居中两行字、不要"免费商用"。
```

这类指令**优先级最高**，直接写入核心记忆。

### **3. 定期清理"垃圾记忆"**

记忆不是越多越好，太杂会乱。

**每月一次**，发：

```
清理过期记忆：删除30天前的临时项目、已完成任务、无效偏好，保留身份、风格、常用工具、IMA知识库规则。
```

防止旧信息干扰新对话。

---

## **五、第三步：IMA + WorkBuddy 联动 = 终极长记忆（核心）**

人的记忆记习惯，IMA 记知识，两者打通才叫"**永不遗忘**"。

### **1. 把知识全部丢进 IMA**

建议存入 IMA 的内容： - 公众号模板、标题库、封面规范 - RAG 教程、智能体搭建步骤 - 你常用的指令、避坑清单、行业资料

### **2. WorkBuddy 开启 RAG 检索（自动调取知识）**

直接用指令：

```
检索 IMA【我的知识库】，关键词：公众号封面规范，严格按库内资料执行，不要编造。
```

**效果**： - 不用重复教它尺寸、格式、风格 - 所有知识来自你自己的库，不幻觉、不瞎编 - 更新 IMA → WorkBuddy 自动学会新知识

### **3. 形成闭环：用 → 记 → 存 → 再用**

```
WorkBuddy 执行任务（写文 / 做方案）
        ↓
自动总结你的新习惯 → 存入长记忆
        ↓
产出的新文档 → 自动归档到 IMA
        ↓
下次直接检索，越用越聪明、越用越省力
```

## **八、总结（一句话讲透）**

> **WorkBuddy 长记忆 = 记住你的人 + IMA 记住你的知识 + 每次自动总结沉淀。**

* **短期**：不用反复重复偏好、格式、风格

## 排重指纹

```text
评估对象（分类/回归/排序）+ 指标类型 + 核心机制 + 解决问题 + 业务目标对齐/样本偏差边界 + 对用户的认知增量
```

| 判断项 | 排重规则 |
|---|---|
| 都是 AUC/ROC | 只有补充阈值、业务损失、不平衡样本、PR-AUC 或校准边界时吸收 |
| 都是多模型对比 | 必须有同一数据切分、同一指标和最终测试集纪律 |
| 都是 SHAP | 解释性内容归入 `0604_可解释性`，评估流程保留在本目录 |
| 模型评测（LLM） | 归入 `01_LLM与大模型/0107_模型评测` |
| 只给指标公式 | 无业务判断准则时不新建知识点 |

## 待补缺口

| 优先级 | 项 | 为什么补 |
|---|---|---|
| 高 | 离线指标与线上收益对齐 | 最常见的模型落地困境是离线好但线上差 |
| 中 | 不平衡样本下的指标选择 | 正样本稀少时 Accuracy 失效，需要 Precision/Recall/F1/PR-AUC 决策准则 |
| 中 | 概率校准和业务阈值 | 输出概率用于决策时必须校准并映射业务损失 |

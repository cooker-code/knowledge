# 传统模型流程
## 知识点入口

- 本模块先看宏观流程，再看文章：[流程化知识点总览](knowledge/06_机器学习/0602_传统模型/核心知识点/流程化知识点总览.md)。
- 新文章必须先归入流程节点，再判断是补充、冲突、不同层次还是降权。
- `文章/` 只保留原文锚点，长期知识必须沉淀到 `核心知识点/`。


## 目的

本目录记录经典机器学习模型如何选择、训练和调参。重点是模型与问题的匹配关系，不是罗列算法百科。

## 流程节点

| 节点 | 要回答的问题 |
|---|---|
| TM01 基线选择 | 规则、线性模型、树模型、集成模型哪个先做基线 |
| TM02 数据适配 | 数据规模、稀疏性、非线性、类别特征是否匹配模型假设 |
| TM03 训练验证 | 训练集、验证集、测试集如何划分，时间穿越如何避免 |
| TM04 调参 | 超参搜索目标是什么，是否优先解决过拟合/欠拟合 |
| TM05 模型比较 | 比较时是否使用同一数据、同一指标、同一验证方式 |

## 文章吸收准则

文章如果只介绍 XGBoost、LightGBM、随机森林等模型概念，降权处理。只有当它补充模型选择条件、调参路径、失败模式或与业务指标的关系时，才进入核心知识点。

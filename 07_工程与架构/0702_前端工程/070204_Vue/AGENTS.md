## 这个目录记录什么

这个文件是 Vue 应用的流程入口。

当前来源覆盖 Vue3 模板工程、Pinia、SCSS/UnoCSS、复杂表格、Power BI 组件、基于 Vue3 的三维/数字孪生框架、Vue3 + FastAPI 全栈工具台案例，以及 wujie/Omi 这类前端框架资讯。它们的共同点不是“Vue 生态资讯”，而是 Vue 项目如何组织状态、样式、复杂组件、高交互集成和前后端边界。

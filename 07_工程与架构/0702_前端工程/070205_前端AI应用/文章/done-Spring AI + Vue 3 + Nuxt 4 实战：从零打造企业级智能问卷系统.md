## 技术栈

本项目选用的技术栈均为当前主流的前沿技术，含金量还可以：

### 后端技术栈

* **Spring Boot 3.5.x**

当前最新版本的 Spring Boot，也是 Spring Cloud Alibaba 当前版本能支持的 Boot 最高版本，为后续微服务改造做准备。

* **Spring AI 1.0.3**

Spring 官方 AI 框架，集成 DeepSeek 大模型，拥抱 AI 时代。

* **Spring Security 6.x**

企业级安全认证与授权，构建稳固的安全基石。

* **MyBatis-Plus**

强大的 ORM 增强工具，极致提升开发效率。

* **Liquibase**

数据库版本管理工具，告别手动管理 SQL 脚本的原始方式，别再傻不拉几地 alter table xxxx。

* **MinIO**

高性能对象存储，统一管理文件资源。

### 前端技术栈

* **Vue 3.5.x**

渐进式 JavaScript 框架，老熟人，无需多言。

* **TypeScript 5.x**

类型安全的 JavaScript 超集，企业级的标配，不用 TS 还怎么装 X

* **Nuxt 4.x**

Vue 服务端渲染（SSR）框架，兼顾 SEO 与首屏加载性能，这更是新时代的装 X 神器！

* **Vite 7.x**

新一代前端构建工具，极速的热更新体验，千万别死抱着 Webpack 不肯放手。

* **Element Plus**

成熟的 Vue 3 组件库，用啥组件库都一样。

* **UnoCSS**

原子化 CSS 引擎，让样式编写更高效、更简洁，新时代别说你没有听过 Tailwind 或 UnoCSS。

* **ECharts**

专业的数据可视化库，那就那么回事。

* **wujie-vue3**

基于 WebComponent 的微前端框架，解决大型应用的拆分与集成。

### 架构亮点

* **Monorepo 架构**

基于 pnpm workspace 的多包管理，实现代码的高度复用。

* **微前端架构**

使用 wujie 集成多个子应用，支撑复杂业务场景的扩展。

* **组件化驱动**

构建独立的业务组件库，支持按需引入，提升开发灵活性。

* **AI 深度融合**

封装流式 AI 对话组件，提供智能问卷设计助手，让 AI 真正落地。

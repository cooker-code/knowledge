# 一个让 AI 界面设计能力暴增 10 倍的神器：UI UX Pro Max

你是否遇到过这样的困境？

让 AI 帮你写代码，功能实现得挺好，但做出来的界面丑得一言难尽——千篇一律的渐变紫、字体不搭调、布局像上世纪产物。

今天分享一个开源神器，彻底解决这个问题。

**UI UX Pro Max** 是一个专门为 AI 编码助手设计的「技能包」。

几乎覆盖了市面上所有主流 AI 编程助手：Claude Code、Cursor、Windsurf、GitHub Copilot、Kiro、Antigravity

它本质上是一个**可搜索的设计数据库**，整合了大量设计资源和最佳实践：

| 资源类型 | 数量 |
| --- | --- |
| UI 风格 | 57 种 |
| 配色方案 | 95 套 |
| 字体组合 | 56 对 |
| 图表类型 | 24 种 |
| UX 指南 | 98 条 |
| 技术栈支持 | 8 种 |

从 **玻璃拟态（Glassmorphism）** 到 **Bento Grid**，从 **极简主义** 到 **新拟态**，应有尽有。

---

安装超简单，一行命令搞定：

```
# 全局安装 CLI  
npm install -g uipro-cli  
  
# 进入你的项目  
cd /path/to/your/project  
  
# 根据你用的 AI 助手选择安装  
uipro init --ai claude      # Claude Code  
uipro init --ai cursor      # Cursor  
uipro init --ai all         # 全部安装
```

## 使用方式

安装后，你只需要像平时一样和 AI 对话：

```
帮我做一个 SaaS 产品的落地页
```

```
创建一个医疗健康数据分析仪表盘
```

```
设计一个暗黑模式的个人作品集网站
```

AI 会**自动调用这个技能**，从数据库中检索最匹配的：

* • 🎨 配色方案
* • 🔤 字体组合
* • 📐 UI 风格
* • ✅ UX 最佳实践

然后生成专业级别的界面代码。

# Pizza Engine

## 技术定位

| 项 | 内容 |
|---|---|
| 技术名 | Pizza Engine |
| 一级类目 | OLAP 与数据库 |
| 二级类目 | 搜索与索引 |
| 技术本体 | 轻量/嵌入式搜索引擎候选，目标是让搜索能力在浏览器、本地应用或边缘环境中运行 |
| 全局架构位置 | 嵌入式搜索层，靠近本地知识库、文档工具、浏览器应用或边缘 Agent |
| 主要使用者 | 前端/客户端工程师、知识工具开发者、Agent 工程开发者 |
| 主要产出 | 本地索引、浏览器内搜索、轻量查询结果 |

## 横向对标

| 对标对象 | 对标点 | 使用判断 |
|---|---|---|
| Elasticsearch | 服务化分布式搜索 | 集群搜索选 ES；本地/浏览器/边缘轻量搜索才考虑 Pizza Engine |
| SQLite FTS | 嵌入式全文检索 | 如果只是本地结构化数据库加全文检索，先看 SQLite FTS |
| Lucene / Tantivy | 搜索库 | 如果需要自己嵌入搜索能力，需比较底层索引、语言生态和中文分词 |
| RAG 检索框架 | 知识召回流程 | Pizza Engine 可能是底层检索组件，不等同于完整 RAG 流程 |

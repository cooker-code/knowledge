## 目录边界

| 目录 | 进入条件 | 不应放入 |
|---|---|---|
| 0401_OLAP引擎 | 主问题是分析引擎的存储、导入、查询、索引、物化视图、运维排障 | 纯智能问数产品、数据同步工具、通用 SQL 写法 |
| 0402_关系数据库 | 主问题是事务、WAL/MVCC、权限、扩展、高可用、关系查询 | MCP/Skill 接入、Text-to-SQL 产品、应用框架教程 |
| 0403_嵌入式分析 | 主问题是进程内、本地文件、单文件数据库和本地分析 | 服务化 OLAP 集群、多用户 OLTP、纯 DataFrame 技巧 |
| 0404_搜索与索引 | 主问题是倒排索引、搜索平台、分片副本、空间索引或嵌入式搜索 | RAG 策略、普通文档工具、代码搜索工具 |
| 0405_缓存与KV | 主问题是 Redis/Kvrocks/Redisson 的数据结构、协议、持久化、高可用或锁语义 | 普通后端缓存模式，除非核心是 Redis/KV 本体 |
| 0406_存储引擎 | 主问题是 WAL、MemTable、SSTable、LSM、Compaction、Block Cache | 上层数据库的普通 SQL 调优 |
| 0407_查询优化 | 主问题是关系代数、统计信息、代价模型、计划搜索、学习型优化器 | SQL 语法技巧、面试题、表设计工具 |
| 0408_hbase | 主问题是 HBase 的列族、Region、WAL/MemStore/HFile、Replication、入仓边界 | 通用数据同步工具或离线数仓加工 |

- 六位三级节点统一使用根级 `编号_知识地图.md`、根级 `编号_版本记录.md` 和 `编号_核心知识点/`。

- 版本类文章只作为版本记录候选；没有官方 Release Notes 或官方文档校验前，不写成 verified 结论。

# FlowDB

## 技术定位

| 项 | 内容 |
|---|---|
| 技术名 | FlowDB |
| 一级类目 | OLAP 与数据库 |
| 二级类目 | 存储引擎 |
| 技术本体 | 面向时序场景的专用存储引擎案例，文章声称通过场景约束、写路径和索引设计获得性能优势 |
| 全局架构位置 | 专用时序存储层候选，需要补证后才能作为稳定技术入口 |
| 主要使用者 | 存储系统工程师、时序平台工程师 |
| 主要产出 | WAL、MemTable、SST、块级索引、Bloom Filter、Compaction 和 TTL GC |

## 🧠什么是 Prompt？

先来点正经解释：  
**Prompt，是你跟 AI 沟通的方式。它不是一句“问题”，而是一份任务书，是“指令 + 上下文 + 规则”的组合。**

你以为你在“问问题”？  
其实你在“发指令”。区别就在这里👇

| ❌ 错误示范 | ✅ 正确示范 |
| --- | --- |
| GPT，写篇周报 | 假设你是产品经理，请用幽默口吻写一篇200字以内的周报，突出本周三件大事，少用行业术语。 |
| 翻译一下 | 将下面的中文翻译成地道的美式英语，语气自然亲切，适合社交媒体发布。 |
| 帮我生成一段代码 | 用 Python 写一个能下载网页图片的脚本，并添加异常处理，代码尽量简洁。 |

👉 **Prompt 不是“随口一问”，而是“带上下文的任务说明”。**  
就像跟 AI 开会：你说得越清楚，它干得越漂亮。

---

## ✍️Prompt 四步式：说清楚，AI才懂

我们总结为一套万能模板：

> **角色 + 任务 + 条件 + 输出格式**

📌 **角色（你是谁）**：  
你是一个资深文案策划 / Java 架构师 / 营销专家...

📌 **任务（你要干啥）**：  
帮我写公众号标题 / 优化这段代码 / 总结这段采访内容...

📌 **条件（限制规则）**：  
不超过 200 字、用口语表达、避免术语、用 Markdown 输出...

📌 **输出格式（交作业方式）**：  
以表格展示 / 列表分段 / 代码块格式 / 加 emoji...

## 🤩进阶技巧：把AI当人带，别当神拜

### ✅ **1. 别废话，要意图**

> ❌「希望你能帮我写一篇好点的文案」  
> ✅「写一篇适合小红书风格的开箱文，语气活泼，包含三点产品优点和一句总结金句」

---

### ✅ **2. 给例子，少讲玄学**

AI很聪明，但不是你肚里的蛔虫，它更喜欢你举个例子：

> 「参考下面的语气，写出类似的三条」  
> 「按照这种排版再来一组」  
> 「像下面这样总结重点」

---

### ✅ **3. 把AI当实习生，不是许愿池**

越具体，它越靠谱。越含糊，它越离谱。

记住这句话👇  
**你越像产品经理，它越像执行专家；你越像甩锅王，它越像PPT画饼人。**

## 🧪实战演练：Prompt 示例

---

### 📌运营文案 Prompt：

```
你是一个懂消费者心理的文案专家。  
请帮我为一款针对年轻白领的即食燕麦产品写3条短文案：  
- 风格活泼、口语化  
- 每条不超过20字  
- 突出“0负担”“轻营养”“颜值高”
```

🎯**输出示例**：

* 「美到犯规的早餐，热水一泡就搞定」
* 「轻卡不轻营养，办公室吃它不怕胖」
* 「0负担真香款，一口吃出小仙女」

---

### 📌程序员 Prompt：

```
你是一名 Java 工程师。  
请用 Java 编写一个能遍历指定目录并输出所有文件路径的工具类。  
要求代码规范，添加必要注释，并兼容 Windows 和 Linux 路径。
```

🎯进阶加料：

* 想让它用 Spring Boot？补一句。
* 想限制用 Java 8？也加进去。
* 想让它只用纯 JDK？说清楚。

## 🏆一句话总结三连发

✨**Prompt 写得准，AI 就是外挂级队友；写得糊，它就是“精神分裂的Chatbot”。**

✨**AI 不差智商，差的是你说清楚话的能力。**

✨**写 prompt 的本质，是逻辑 + 结构 + 场景表达的再训练。**

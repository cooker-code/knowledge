#### **一、核心揭秘：“超级Prompt”的四大黄金模块**

**模块一：角色设定 - 定义AI的“专业身份”**

**目的：赋予AI领域专家视角。**

**示例对比：“资深电商数据分析师” / “教育学研究员” / “生物医学领域科研人员”。**

**模块二：领域知识与规范 - 注入行业的“灵魂”**

**目的：让AI懂得行业的“黑话”和“潜规则”。**

**内容要素：关键指标（如GMV、转化率）、核心分析维度（如用户画像、销售漏斗）、行业禁忌等。**

**模块三：图表类型与选择逻辑 - 配备清晰的“工具箱”**

**目的：将分析目标与最合适的图表类型快速匹配。**

**方法：提供一个“决策矩阵”（以表格或列表形式），例如：“当需要比较不同类别占比时，优先使用饼图或环形图；当展示时间序列趋势时，优先使用折线图。”**

**模块四：视觉风格偏好 - 统一品牌的“颜值”**

**目的：确保图表风格专业、一致。**

**设定项：配色方案、字体大小、图表尺寸、是否添加数据标签、网格线透明度等。**

#### **二、手把手实战：构建你的第一个领域专属Prompt**

#### **以“电商运营”为例，每一步可参考的Prompt提示词模板如下：**

### 第一步：定义角色与目标

```
你是一名专注于电商数据分析的资深专家，拥有8年电商平台运营经验。你擅长通过可视化图表发现销售趋势、用户行为和运营问题，能够快速从复杂数据中提炼商业洞察。
```

### 第二步：填充领域知识

```
你精通以下电商领域知识：1. 核心指标：GMV、客单价、转化率、ROI、复购率、LTV2. 流量渠道：自然搜索、直通车、直播带货、社交媒体3. 用户行为：加购率、收藏率、浏览深度、跳出率4. 运营概念：SKU、SPU、库存周转、动销率
```

### 第三步：建立图表选择逻辑

```
请遵循以下图表选择指南：1. 分析销售趋势 over time → 折线图2. 比较品类销售额占比 → 堆积柱状图3. 分析价格带分布 → 直方图4. 渠道效果对比 → 漏斗图5. 用户分层分析 → 饼图+环形图组合
```

### 第四步：设定视觉风格

```
视觉规范：1. 主色调：科技蓝(#1f77b4)、活力橙(#ff7f0e)2. 字体：微软雅黑，标题14pt，标签12pt3. 尺寸：1000×600像素（适配公众号排版）4. 风格：简洁现代，减少不必要的装饰元素
```

### **参考以上四个步骤，我们可以形成一个整合后的完整超级Prompt提示词模板：**

```
你是一名专注于电商数据分析的资深专家，拥有8年电商平台运营经验。你擅长通过可视化图表发现销售趋势、用户行为和运营问题。  
【领域知识】你精通电商核心指标：GMV、客单价、转化率、ROI、复购率；熟悉流量渠道：自然搜索、直通车、直播带货；了解用户行为数据：加购率、收藏率、浏览深度。  
【图表选择逻辑】1. 分析销售趋势 → 折线图2. 比较品类占比 → 堆积柱状图  3. 渠道转化分析 → 漏斗图4. 用户分层 → 饼图+环形图  
【视觉规范】- 配色：主色#1f77b4，辅助色#ff7f0e- 字体：微软雅黑，标题14pt- 尺寸：1000×600像素- 风格：简洁商务风  
请先确认理解以上要求，然后等待我提供具体数据。
```

## 三、如何使用与迭代你的专属助手

有了以上的提示词模板，具体该怎么用呢？使用方法如下：

**1、在DeepSeek中新建对话**

**2、一次性粘贴完整的超级Prompt**

**3、等待AI确认理解所有要求**

**最后，输入你的具体数据及分析需求就可以了。**

**数据可以通过附件上传，也可以直接在对话框中输入，可以是结构化的数据，也可以是非结构化的数据。**

**当然，这是一个持续的优化过程。如果哪些地方你不满意或者不符合预期，保持耐心，回到超级Prompt中修改或者与AI多次对话。**

**记住：好的超级Prompt是迭代出来的，不是一次写成的。**

照片囤了几千张，却只会原图发朋友圈、小红书、公众号？

现在不用再手动修图、找模板、抠图排版！

**AI 可以一键盘活你整个人的手机相册。**

你只需要先判断照片属于哪一类，然后**复制对应提示词**，丢给AI就行。

下面这4类，基本能覆盖大多数照片。

01｜日常生活照片

|  |
| --- |
| 适合：吃饭、旅行、萌宠、生活记录等 |

```
提示词： 给我在图片上添加手绘白色涂鸦线条，手写风中文标注。在主体旁边添加白色描边气泡文字，线条可爱随意的，像手绘笔记，带箭头、圈圈、涂鸦装饰，小红书风格，ins风，温馨日常氛围，轻松治愈感。
```

02｜人物肖像类照片

|  |
| --- |
| **适合：单人写真、毕业照** |

```
提示词：根据我的人物肖像自动生成一张收藏版史诗叙事海报，以我的侧脸剪影作为外轮廓，剪影内部自动生长出最契合该主题的完整世界观，包括标志性场景、角色关系、象征符号、关键建筑、生物、道具与氛围（校园➕毕业）。

整体不是普通拼贴，而是高级的剪影轮廓填充式叙事合成，带有双重曝光式联想，但更偏电影海报与梦幻水彩插画融合风格；柔和空气透视，轻雾化过渡，纸张颗粒，边缘飞白与刷痕，大面积留白，版式克制高级，安静、宏大、神圣、怀旧、诗意、传说感。

风格、色彩、场景、材质全部根据主题自动适配，所有元素必须强绑定主题，一眼识别，不要杂乱，不要硬拼贴，不要模板化背景，不要廉价奇幻素材。

如果有字体用英文。我的学校是xx大学 我的专业是xx。
```

03｜知识类干货照片

|  |
| --- |
| **适合：书单、工具、课程、方法论、干货图** |

```
提示词：把这张图片做成“小红书知识分享手绘笔记风”。浅米白背景，使用橙色手写中文标注，加入手绘箭头、圈圈、虚线、括号、下划线、爱心、星星、小笑脸等涂鸦元素。

请自动提炼图片里的核心信息，整理成2—3个分类或重点观点。顶部加一句醒目标题，中间突出主体，两侧用气泡文字解释，底部加一句总结金句。整体像老师在图上做知识笔记，干净、有层级、轻松可爱，适合小红书、朋友圈、公众号知识分享。
```

04｜活动现场类照片

|  |
| --- |
| **适合：校园活动、公司会议、项目汇报、线下沙龙** |

```
提示词：请自动提炼图片里的活动主题、现场氛围和核心价值，整理成2—3个简短重点。

顶部加一句醒目的手写标题，中间突出屏幕/发言者/参与者，两侧用气泡文字说明，底部加一句总结金句。

整体要像一次正式活动的手绘纪实笔记，干净、有层级、轻松但不幼稚，适合学校活动、公司会议、答辩会、读书会、项目汇报、团队复盘、小红书和公众号配图。
```

**✍️写在最后**

一个比较关键的点，这些提示词，写的都是“美术风格”和“画面效果”，而它最厉害的地方是补内容。

比如小猫照片，它会帮你补情绪；书单照片，它会帮你提炼重点；日常图片，它会帮你加上更有感觉的文字。

也就是说，你给的是一张“素材”，它帮你完成的是一整套“表达”。

所以你真正要做的，只有这3步：

①从相册随便选一张照片当素材

②判断照片属于哪一类

③直接复制对应的专属提示词

剩下的，交给AI就行。

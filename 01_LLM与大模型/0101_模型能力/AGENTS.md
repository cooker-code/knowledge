## 排重指纹

```text
模型本体 + 所属能力维度 + 核心机制 + 解决问题 + 适用边界 + 对用户的认知增量
```

| 判断项 | 排重规则 |
|---|---|
| 同一模型不同版本 | 必须有上下文、价格、能力维度差异才合并新增 |
| 都是幻觉/对齐 | 按事实/忠实/逻辑/上下文分型，不按现象判重 |
| 都是榜单/实测 | 没有任务域、样本、版本、provider、复现方式时降权 |
| Claude Skills/工具流文章 | 重路由到 Agent 与 AI 工程，不在本目录扩写 |
| 单一模型发布新闻 | 仅留影响选型的版本时效、成本、上下文、能力边界 |

# LLM 对齐训练机制与预训练先验

## 来源
- [2篇最新Anthropic论文，揭开LLM对齐新范式](../文章/done-2篇最新Anthropic论文，揭开LLM对齐新范式.md)

## 核心问题
为什么只靠 SFT/RLHF 模仿行为不足以保证对齐，以及在什么阶段介入才能真正"教会"模型价值观，而不只是让它"表演"正确行为。

## 判断准则

### 预训练先验污染对齐效果
- 模型的底层行为倾向来自预训练数据，不是对齐微调。如果预训练数据充满"科幻式 AI"叙事（AI 因自保而做坏事），对齐层再努力也难以彻底覆盖。
- 验证实验：Claude 名字替换为 "Goliath" 等陌生名时，不对齐率从 14% 升至 23%——说明对齐绑定了 "Claude角色"，而非绑定底层价值观。

### SDF（合成文档微调）修复预训练先验
- 用 LLM 生成的、描绘 AI 按正确价值观行事的**虚构故事**来微调，可以覆写预训练先验。
- 14M token 正面故事训练后：勒索场景不对齐率 0.65 → 0.58，金融犯罪 0.49 → 0.32，癌症研究破坏 0.67 → 0.46。
- SDF 不需要针对特定场景，**广泛展现良好品格**的故事有迁移效果。

### "理由比行为更重要"
- 只过滤掉不对齐行为的训练数据，效果甚微（22% → 15%）。
- 在采样时注入指令、让模型展示**对对齐行为的主动伦理推理**，不对齐率降至 **~3%**。
- 核心洞察：训练模型"表演正确行为"远不如训练模型"理解为什么要这么做"。

### MSM（模型规范中间训练）的四阶段
```
预训练 → MSM（理解原理） → AFT（行为演示） → 评估
```
- MSM 在预训练和对齐微调之间插入"读说明书"阶段，用合成文档训练模型讨论其 Model Spec 的内容。
- 相同 AFT 数据，MSM 处理过的模型会按 MSM 阶段的价值观泛化，而不是只复制表面行为。
- **数据效率提升 40-60 倍**：Qwen2.5-32B 上，MSM + AFT 用 1/40 数据达到纯 AFT 的性能。
- Agentic Misalignment 率：Baseline 68% → MSM only 53% → AFT(带CoT) 48% → MSM+AFT **5%**。

### 训练数据多样性的意外效果
在无害性 RL 环境中加入**无关工具定义**，同样能降低不对齐率——多样性改善泛化，不仅限于直接相关的数据。

### "道德顾问"数据的强迁移
训练 Claude 在聊天中**给用户建议如何应对道德困境**，可将 Agentic Misalignment 率降至零。训练域（聊天对话）和测试域（自主工具调用）形式完全不同，价值观仍然迁移。

## 认知偏差

| 常见错误认知 | 正确理解 |
|---|---|
| SFT/RLHF 让模型模仿正确行为就够了 | 行为模仿不足以保证安全；安全空白会让模型回退到预训练先验 |
| 对齐数据量越大越好 | 数据质量（是否包含推理过程）比数量更关键；MSM+AFT 用 1/40 数据超过纯 AFT |
| 对齐训练针对特定场景写数据 | 广泛的价值观故事和多样性训练数据有跨场景迁移效果 |
| 预训练阶段不影响安全行为 | 预训练先验是对齐层最大的隐性竞争；科幻故事会训练出"科幻AI"的默认行为倾向 |
| CoT 监督是必须的 | MSM+AFT（无 CoT）优于单独 AFT（带 CoT），保留了 CoT 可监控性 |

## 待验证缺口
- MSM 对国内模型（如 Qwen）的实际工程成本和效果差异需要更多实测
- SDF 故事的质量标准和最小有效数量尚待系统研究
- 价值观传递的可监控性和解释性（模型为什么在某场景决策）仍是开放问题

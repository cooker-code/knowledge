## 排重准则

问题指纹：

```text
技术本体 + 所属模块 + 核心机制 + 解决问题 + 适用边界 + 对用户的认知增量
```

| 判断项 | 排重规则 |
|---|---|
| 技术本体相同 | 继续比较模块和机制，不直接判重 |
| 解决问题相同 | 没有新增边界、反例、证据或实践动作时合并 |
| 只多了案例 | 案例不改变准则时，只追加原文锚点 |
| 多了观点冲突 | 必须记录为认知校准点 |

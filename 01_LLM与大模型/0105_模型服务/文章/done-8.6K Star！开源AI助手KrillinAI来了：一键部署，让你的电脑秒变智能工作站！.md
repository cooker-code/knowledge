## 快速部署

想要在本地运行KrillinAI其实非常简单，按照官方文档的指引，几分钟就能搞定：

1、**环境准备**：确保你的系统满足基本要求（推荐8GB以上内存）

2、**下载项目**：

```
git clone https://github.com/krillinai/KrillinAI.git
cd KrillinAI
```

3、**安装依赖**：

```
# 安装必要的依赖包
pip install -r requirements.txt
```

4、**启动服务**：

```
# 启动KrillinAI服务
python main.py
```

5、**访问界面**：在浏览器中打开 `http://localhost:8080`，就可以开始使用了。

**Docker部署方式**（推荐）：

```
# 构建镜像
docker build -t krillinai .

# 运行容器
docker run -p 8080:8080 -v ./data:/app/data krillinai
```

整个部署过程通常在10分钟内就能完成，**即使是技术小白也能轻松上手**。

## 核心问题

职业环境材料的价值不在于制造焦虑，而在于校准外部约束：行业供需、组织成本、技术周期、生活半径、创业窗口和个人可承受风险。

## 认知校准点

| 环境叙事 | 校准后处理 |
|---|---|
| 大厂裁员说明个人不行 | 裁员常是成本、结构和周期动作，不能直接内化成能力评价；但也提醒个人不能把身份完全绑定到单一平台。 |
| 大数据凉了 | “概念热”降温不等于数据无价值，真正降温的是无业务闭环、无治理、无质量、无 ROI 的数据项目。 |
| AI + Data Infra 是确定机会 | 企业数据会成为 AI 上下文基础设施，但 workload 范式仍不稳定，只能作为探索假设。 |
| 通勤和时代叙事能指导职业选择 | 这类材料时效和情绪成分高，只能用于暴露约束，不能直接沉淀为长期准则。 |
| 数据、BI、AI 概念图能形成职业判断 | 粗粒度概念只能作背景，不能替代岗位能力、组织位置和项目代表作。 |

## 外部环境判断表

| 维度 | 要问的问题 | 可行动输出 |
|---|---|---|
| 行业周期 | 需求是在增长、收缩、替换还是重分配？ | 决定押注新增岗位还是升级存量岗位。 |
| 公司成本 | 自己是收入/增长/降本资产，还是可替换成本中心？ | 用项目结果证明对收入、成本、风险或效率的影响。 |
| 技术变化 | AI 替代的是任务、流程还是责任？ | 把可替代任务工具化，把不可替代责任代表作化。 |
| 生活约束 | 通勤、家庭、健康、现金流会不会改变可选路径？ | 职业决策前显性列出硬约束。 |
| 创业机会 | 是否存在未被满足的真实工作负载和付费方？ | 用 MVP 验证假设，不把趋势等同于需求。 |

## 处理低价值材料的规则

| 材料类型 | 处理 |
|---|---|
| 通勤政策、单次市场新闻 | 只保留为环境约束，不写行动准则。 |
| 时代焦虑、机会宏大叙事 | 提取假设，不提取情绪。 |
| 投资理财操作常识 | 不进入职业发展长期知识点；当前无稳定二级目录。 |
| 数据概念营销 | 只保留能说明价值闭环和治理失败模式的部分。 |
| 裁员榜单和时效数字 | 不联网验证时只作本地材料，不作为最新事实。 |

## 可执行动作

1. 每半年重写一次个人“职业环境假设”：行业、公司、岗位、家庭、现金流、技术周期。
2. 对每个机会写清“我能控制什么、不能控制什么、最坏损失是什么”。
3. 不用“焦虑强度”排序行动，用“可验证性 + 可迁移性 + 代表作价值”排序。
4. 遇到裁员或行业低迷叙事时，先分离公司成本策略、岗位供需、个人能力和生活恢复四件事。

## 待验证缺口

- 本轮未联网验证裁员和行业数据，不能把文章中的数字当作当前事实。
- AI + Data Infra 机会需要结合真实企业数据工作负载、预算方和落地路径继续验证。

## 🚀 **Mermaid 到底有多神奇？**

代码：

```
graph TD  A[需求分析] --> B{设计阶段}  B -->|方案A| C[开发]  B -->|方案B| D[测试]  C --> E[上线]  D --> E
```

万能适配场景：流程图、时序图、甘特图、类图……只有你想不到，没有它画不了！

### ② **高级玩法**

```

```

交互式图表：点击节点跳转详情页（需配合第三方工具）。

## 🎁 **免费资源包 & 实战技巧**

1️⃣ **VSCode 插件**：安装 `Mermaid Preview`，实时预览图表效果。  
2️⃣ **模板库**：mermaid.ink 提供海量现成模板，一键复制代码。  
3️⃣ **避坑指南**：

* 复杂图表用 `subgraph` 分组，保持代码整洁。
* 中文乱码？加一句 `%%{fontfamily: "Microsoft YaHei"}%%`。
* 实战网址：https://mermaid-live.nodejs.cn/edit
* 官方文档：https://mermaid.nodejs.cn/intro/

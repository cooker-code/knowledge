你有没有想过，如果 AI 能看懂你电脑屏幕上的一切，会发生什么？

最近发现了一个开源项目，让我眼前一亮。它叫 **MineContext**，火山引擎团队出品，核心思路特别简单粗暴：**让 AI 看你的屏幕，懂你在干什么，然后主动帮你干活**。

听起来是不是有点像科幻片？但它已经能用了。

## 它到底是个啥？

简单说，MineContext 就像给你的电脑配了个"懂事的助理"。

它会定时给你的屏幕截图，然后用 AI 看懂你在做什么——写文档、刷网页、看数据、聊天……全都记下来。**不是简单的记录，是真的"看懂"**。

然后呢？它会在后台默默给你：

* **每天/每周自动生成工作总结**
* **主动提炼待办事项**
* **给你推送关键提示和洞察**
* **记录你的活动轨迹**

最关键的是，你还能随时问它："我上周那个方案放哪了？""我今天主要干了啥？""帮我整理一下最近的思路"——它都能基于你的屏幕上下文回答。

**这才是真正的"上下文感知"**。不是你主动喂给 AI 信息，而是 AI 主动看懂你的数字世界。

## 四个核心能力

### 1. 无负担收集

你不用管它。装上之后，它就在后台默默截图、处理、存储。

**不会让你有心理负担**——不需要你手动整理、打标签、分类。全自动。

### 2. 主动推送

每天早上打开电脑，它会给你推送：

* 昨天的工作总结
* 今天要做的事（从你昨天的屏幕里提炼出来的）
* 一些你可能忽略的关键信息

**不用你问，它主动告诉你**。

### 3. 智能浮现（开发中）

以后你写东西的时候，它会自动浮现出相关的历史上下文——你之前看过的资料、写过的笔记、聊过的话题。

**辅助创作，但不会打扰你**。

### 4. 上下文工程架构

这是技术层面的事，简单说就是：它能处理各种来源的信息（屏幕、文档、图片、视频……），然后把这些信息变成 AI 能用的"上下文"。

## 最关心的问题：隐私怎么办？

这是我最喜欢它的地方：**所有数据默认存在你自己电脑上**。

不上云，不发给第三方。你的屏幕截图、生成的总结、所有记录，都在本地：

```
~/Library/Application Support/MineContext/Data
```

而且它支持**完全本地运行的 AI 模型**。你可以用 LMStudio 跑本地模型，一点数据都不外传。

当然，如果你觉得本地模型效果不够好，也可以用豆包或 OpenAI 的 API（推荐豆包，便宜又好用）。

## 怎么上手？

### 第一步：下载安装

去 GitHub 下载最新版（支持 Mac 和 Windows）。

Mac 用户注意，下载后要在终端里运行一下：

```
sudo xattr -d [com.apple](<http://com.apple>).quarantine "/Applications/[MineContext.app](<http://MineContext.app>)"
```

否则 macOS 会拦着不让你打开。

### 第二步：配置 API

启动之后，它会让你填 API Key。

**强烈推荐用豆包**：性价比最高，效果也不错。

去火山引擎控制台生成 API Key，然后开通两个模型：

* 视觉语言模型：Doubao-Seed-1.6-flash
* 向量化模型：Doubao-embedding-large

填进去就行。

### 第三步：开始记录

在 Settings 里设置好要监控的屏幕区域，点击「Start Recording」。

然后？**然后就忘了它吧**。

专心干你的活，它会在后台默默收集、分析、生成信息。等晚上或者第二天早上，你就能看到它给你的总结和待办了。

## 它适合谁？

说白了，只要你每天对着电脑干活，它都能帮上忙：

* **知识工作者**

  （研究员、分析师）：每天看那么多信息，它帮你提炼关键点
* **内容创作者**

  （写手、博主）：随时调取历史灵感和素材
* **终身学习者**

  （学生、研究者）：自动帮你建立知识体系
* **项目经理**

  ：整合各种零散信息，随时复盘

## 对比同类产品

### vs ChatGPT Pulse

ChatGPT Pulse 也是个"记忆助手"，但区别很大：

* **上下文范围**

  ：MineContext 能看你整个屏幕，Pulse 只能看聊天记录
* **隐私**

  ：MineContext 数据本地存储，Pulse 都在 OpenAI 服务器上
* **主动推送**

  ：MineContext 能生成各种洞察和待办，Pulse 只给你几条简单提示
* **开源 vs 闭源**

  ：MineContext 开源可定制，Pulse 是黑盒
* **价格**

  ：MineContext 用自己的 API（便宜），Pulse 要 200 美元/月的 Pro 订阅

### vs Dayflow

Dayflow 只是记录你的活动，MineContext 不光记录，还能理解、提炼、生成新内容。

而且 MineContext 的活动记录更清晰，还能基于这些记录做问答和创作。

## 名字也有讲究

MineContext 这个名字挺有意思的。

一方面是"我的上下文"（Mine Context），另一方面也是"挖掘上下文"（Mine = 挖矿）。

它借鉴了《我的世界》（MineCraft）的理念：**把海量的信息"方块"，变成你可以自由搭建、组合、创造的世界**。

## 未来还能做什么？

现在它主要是截屏，但后续计划支持：

**P0-P1（已实现或即将实现）：**

* 屏幕截图 ✅
* 笔记编辑 ✅
* 链接上传
* 文件上传（文档、图片）

**P2-P3（正在开发）：**

* 浏览器扩展（抓取 AI 对话记录）
* 会议记录
* RSS 订阅
* 深度研究（Deep Research）
* Notion/Obsidian/Figma 等工具集成

**P4-P5（长期规划）：**

* 微信/QQ 聊天记录
* 手机截图同步
* 智能眼镜/手环数据同步（这个有点科幻了）

## 我的看法

这个项目有两个点特别打动我：

**第一，它解决的是真问题**。

我们每天对着电脑干活，信息量巨大，但大部分都转瞬即逝。事后想找某个灵感、某段对话、某个数据，经常翻半天都找不到。

MineContext 把这些"易逝的上下文"都保存下来，还能让 AI 帮你理解和利用。这是真需求。

**第二，它对隐私的态度很实在**。

没有搞那种"我们很重视隐私，但数据还是要上传到我们服务器"的话术。直接本地存储，支持本地模型，你的数据你做主。

当然，它现在还不完美。比如：

* 只支持截屏，其他数据源还在开发中
* 有些功能（比如智能浮现）还没做完
* 部署对小白用户来说稍微有点门槛

但作为一个开源项目，已经很能打了。而且社区很活跃，更新也快。

## 最后

如果你也觉得每天的工作信息"看了就忘、找不回来"很痛苦，可以试试 MineContext。

开源免费，数据本地，上手也不算难。

最重要的是，**它能让 AI 真正看懂你的数字生活，然后主动帮你干活**。这才是 AI 助手该有的样子。
